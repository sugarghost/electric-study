# 04. 직렬·병렬 연결과 합성 저항

## 핵심 개념

여러 저항을 하나의 등가 저항으로 바꾸는 것 = **합성저항**  
회로 분석의 첫 번째 단계입니다.

---

## 직렬 연결 (Series)

### 구조

저항들이 **한 줄로** 연결된 것

```
+--[R1]--[R2]--[R3]--+
```

### 특징

| 항목 | 특징 |
|------|------|
| 전류 | 모든 저항에 **같은 전류** 흐름 |
| 전압 | 각 저항에 걸리는 전압은 **저항에 비례**해서 나뉨 |

### 합성 저항 공식

```
R_합 = R1 + R2 + R3 + ...
```

> 그냥 다 더하면 됩니다.

### 예제

R1=10Ω, R2=20Ω, R3=30Ω 직렬 연결 시:
```
R_합 = 10 + 20 + 30 = 60 [Ω]
```

---

## 병렬 연결 (Parallel)

### 구조

저항들이 **나란히** 연결된 것

```
     +--[R1]--+
     |        |
+----+--[R2]--+----+
     |        |
     +--[R3]--+
```

### 특징

| 항목 | 특징 |
|------|------|
| 전압 | 모든 저항에 **같은 전압** 걸림 |
| 전류 | 각 저항에 흐르는 전류는 **저항에 반비례**해서 나뉨 |

### 합성 저항 공식

```
1/R_합 = 1/R1 + 1/R2 + 1/R3 + ...
```

### 2개짜리 병렬 (자주 쓰는 간편식)

```
R_합 = (R1 × R2) / (R1 + R2)
```

> 곱 나누기 합 → "곱나합"으로 외우세요.

### 예제

R1=6Ω, R2=3Ω 병렬 연결 시:
```
R_합 = (6 × 3) / (6 + 3) = 18 / 9 = 2 [Ω]
```

> 병렬 합성저항은 항상 **가장 작은 저항보다 작습니다**.

---

## 같은 저항 n개 병렬

```
R_합 = R / n
```

예) 12Ω짜리 3개 병렬 → R_합 = 12/3 = 4Ω

---

## 직렬 vs 병렬 비교

| 구분 | 직렬 | 병렬 |
|------|------|------|
| 전류 | 동일 | 나뉨 |
| 전압 | 나뉨 | 동일 |
| 합성저항 | 커진다 | 작아진다 |
| 공식 | R1+R2+... | 역수의 합의 역수 |

---

## 혼합 회로 풀이 순서

복잡한 회로도 단계별로 분해하면 됩니다:

1. 가장 안쪽(작은 부분)부터 합성
2. 합성된 등가저항으로 바깥쪽 계산
3. 반복

### 예제

```
R1=4Ω 직렬, (R2=6Ω, R3=3Ω 병렬)
```
1단계: R2, R3 병렬 합성  
```
R23 = (6×3)/(6+3) = 2Ω
```
2단계: R1과 R23 직렬 합성  
```
R_합 = 4 + 2 = 6Ω
```

---

## 시험 포인트

- 병렬 2개: **곱나합** (R1×R2)/(R1+R2) 반드시 암기
- 직렬: 전류 같음, 병렬: 전압 같음 → 혼동하지 말 것
- 합성저항 구한 후 옴의 법칙으로 전류, 전압 계산
