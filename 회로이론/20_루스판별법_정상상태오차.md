# 20. 루스 판별법 & 정상상태 오차

## Part 1. 루스 판별법 (Routh-Hurwitz Criterion)

### 왜 필요한가?

제어 시스템이 **안정한지 불안정한지** 판별하는 방법.  
특성방정식의 근을 직접 구하지 않고도 안정성을 알 수 있습니다.

### 안정 조건

> 특성방정식의 모든 근이 s 평면의 **왼쪽(실수부 < 0)** 에 있어야 안정

---

### 루스 배열 만드는 법

특성방정식: **a₀s³ + a₁s² + a₂s + a₃ = 0**

#### 1단계: 배열 초기 설정

```
s³ | a₀   a₂
s² | a₁   a₃
s¹ | b₁   0
s⁰ | c₁
```

#### 2단계: b₁ 계산

```
b₁ = (a₁×a₂ - a₀×a₃) / a₁
```

#### 3단계: c₁ 계산

```
c₁ = (b₁×a₃ - a₁×0) / b₁ = a₃
```

#### 4단계: 안정 판별

**첫 번째 열 (a₀, a₁, b₁, c₁)의 부호 변화 횟수 = 불안정 근의 수**

→ 부호 변화가 0이면 안정!

---

### 예제

특성방정식: s³ + 6s² + 11s + 6 = 0

```
s³ | 1    11
s² | 6    6
s¹ | b₁   0
s⁰ | c₁

b₁ = (6×11 - 1×6) / 6 = (66-6)/6 = 10
c₁ = 6

첫 번째 열: 1, 6, 10, 6  → 모두 양수, 부호 변화 없음
→ 안정!
```

---

### 불안정 판별 예제

특성방정식: s³ + s² - s + 1 = 0

```
s³ | 1    -1
s² | 1     1
s¹ | b₁
   b₁ = (1×(-1) - 1×1) / 1 = -2

첫 번째 열: 1, 1, -2, ...  → 부호 변화 2번
→ 불안정 근 2개 → 불안정!
```

## Part 2. 정상상태 오차 (Steady State Error)

### 핵심 개념

목표값(R)과 실제 출력(Y)의 **차이가 시간이 지나도 남는 것**

```
e_ss = lim(t→∞) e(t) = lim(s→0) s·E(s)
```

최종값 정리 활용: `lim(t→∞) f(t) = lim(s→0) s·F(s)`

---

### 시스템 형(Type)에 따른 오차

단위 피드백 시스템에서 개루프 전달함수 G(s) 기준:

```
G(s)의 분모에 s가 몇 개 있느냐 = 시스템 형(Type)
```

| 입력 종류 | Type 0 | Type 1 | Type 2 |
|---------|--------|--------|--------|
| 계단 입력 (위치) | 1/(1+Kp) | 0 | 0 |
| 경사 입력 (속도) | ∞ | 1/Kv | 0 |
| 가속도 입력 | ∞ | ∞ | 1/Ka |

> **Type이 높을수록 오차가 적음** (하지만 안정성은 나빠질 수 있음)

---

### 오차 상수

```
위치 오차 상수: Kp = lim(s→0) G(s)
속도 오차 상수: Kv = lim(s→0) s·G(s)
가속도 오차 상수: Ka = lim(s→0) s²·G(s)
```

---

### 예제

G(s) = 10 / (s(s+2)), 단위 피드백, 계단 입력 r(t) = 1

```
Type 1 시스템 (분모에 s 하나)

계단 입력에 대한 정상상태 오차:
e_ss = 0  (Type 1 이상이면 계단 입력 오차 = 0)
```

G(s) = 5 / (s+1)(s+2), Type 0, 계단 입력

```
Kp = lim(s→0) G(s) = 5/(1×2) = 2.5

e_ss = 1/(1+Kp) = 1/(1+2.5) = 1/3.5 ≈ 0.286
```

## 시험 포인트

### 루스 판별법

| 항목 | 내용 |
|------|------|
| 안정 조건 | 첫 번째 열 모두 같은 부호 |
| 부호 변화 횟수 | 불안정 근의 수 |
| 필요 조건 | 모든 계수가 양수이고 존재 (충분조건은 아님) |

### 정상상태 오차

| 입력 | Type 0 | Type 1 | Type 2 |
|------|--------|--------|--------|
| 계단 | 유한값 | 0 | 0 |
| 경사 | ∞ | 유한값 | 0 |
| 가속도 | ∞ | ∞ | 유한값 |

- 루스 배열의 **첫 번째 열**만 보면 됨
- 정상상태 오차는 **시스템 형(Type)** 을 먼저 파악하는 것이 핵심
