# 21. 2단자 회로망

## 핵심 개념

> **2단자 회로망이란, 복잡한 회로 내부를 몰라도 두 단자에서 측정되는 전압·전류 특성만으로 회로를 표현하는 방법이다.**

비유: 콘센트를 생각해보자. 벽 안에 전선이 어떻게 연결되어 있는지 몰라도, 우리는 "이 콘센트는 220V, 15A까지 쓸 수 있다"는 특성만 알면 충분하다. 2단자 회로망도 마찬가지다. 두 단자에서 보이는 특성만 알면 내부 구조는 몰라도 된다.

---

## 2단자(포트) 회로망 개념

**2단자 회로망(One-port network)** 은 외부와 딱 2개의 단자(+단자, -단자)로만 연결되는 회로 덩어리다.

```
     ┌──────────────┐
 ●───┤              ├───● 단자 1 (+)
     │   내부 회로   │
 ●───┤              ├───● 단자 2 (-)
     └──────────────┘
         ↑
    여기가 어떻게 생겼는지 몰라도 됨
```

외부에서 전압 V를 걸면 전류 I가 흐르고, 그 비율이 바로 임피던스다.

---

## 구동점 임피던스 Z(s)

**"이 두 단자에 전압을 걸었을 때 얼마나 전류가 흐르는가?"**

```
Z(s) = V(s) / I(s)   [단위: Ω]
```

- V(s): 단자 간 전압 (라플라스 변환)
- I(s): 단자로 흘러들어가는 전류

비유: 수도꼭지에서 물(전류)이 얼마나 잘 나오는지를 수압(전압)과 비교한 값. 임피던스가 크면 같은 전압에서 전류가 조금 흐른다(좁은 파이프), 작으면 전류가 많이 흐른다(넓은 파이프).

**예시: R, L, C 직렬 회로의 구동점 임피던스**

```
Z(s) = R + sL + 1/(sC)
     = (s²LC + sRC + 1) / (sC)
```

---

## 구동점 어드미턴스 Y(s)

**"이 두 단자에 전압을 1V 걸었을 때 얼마나 전류가 흐르는가?"**

```
Y(s) = I(s) / V(s) = 1 / Z(s)   [단위: S (지멘스)]
```

임피던스의 역수. 어드미턴스가 크면 전류가 잘 흐른다(도체에 가까움).

---

## 임피던스의 극점(Pole)과 영점(Zero)

복잡한 Z(s)는 분수 형태로 표현된다:

```
Z(s) = K × (s - z₁)(s - z₂)...
           ─────────────────────
           (s - p₁)(s - p₂)...
```

| 구분 | 정의 | 의미 | 실제 특성 |
|------|------|------|-----------|
| **영점(Zero)** | Z(s) = 0이 되는 s값 (분자 = 0) | 전압이 있어도 전류가 0 | 단락(Short)에 가까운 특성 |
| **극점(Pole)** | Z(s) = ∞가 되는 s값 (분모 = 0) | 전류가 있어도 전압이 ∞ | 개방(Open)에 가까운 특성 |

비유:
- **영점**: 도로가 꽉 막혀서 차가 전혀 못 가는 지점 (Z=0, 단락)
- **극점**: 도로가 사라져서 차가 무한히 쌓이는 지점 (Z=∞, 개방)

**LC 병렬 회로 예시**

```
Z(s) = sL × (1/sC)   =   sL/(s²LC + 1)
       ─────────────
       sL + (1/sC)

영점: s = 0 (DC에서 인덕터가 단락)
극점: s = ±j/√(LC) (공진 주파수에서 병렬공진 → Z → ∞)
```

---

## 포스터 정리 (Foster's Reactance Theorem)

**LC 회로만으로 이루어진 순수 리액턴스 회로망의 성질**

1. 리액턴스 X(ω)는 주파수 ω에 대해 **항상 단조 증가**한다.

```
dX(ω)/dω > 0   (항상 양수)
```

2. 극점과 영점이 **번갈아** 나타난다.

```
ω = 0: 영점 or 극점
↓ 증가
첫 번째: 극점 (또는 영점)
↓ 증가
그 다음: 영점 (또는 극점)
↓ (교대로 반복)
```

3. 가장 낮은 주파수와 가장 높은 주파수(ω→∞)는 영점이거나 극점이다.

비유: 롤러코스터 트랙처럼 전체적으로는 올라가면서 중간에 봉우리(극점)와 골짜기(영점)가 번갈아 나타난다.

---

## 정저항 회로망 (Constant-Resistance Network)

**"주파수가 어떻게 바뀌어도 임피던스가 항상 R로 일정한 회로"**

주파수에 따라 임피던스가 바뀌면 신호가 왜곡될 수 있다. 이를 방지하기 위해 만든 특수 회로다.

### RC 정저항 회로

```
R₁과 Z_a가 직렬, 여기에 R₂와 Z_b가 병렬로 연결된 구조에서:

조건: R₁ × R₂ = k²  (k는 정저항 값)
      Z_a × Z_b = k²
```

대표적인 RC 정저항 조건:

```
Z₁(s) × Z₂(s) = R²

예: Z₁ = R + (1/sC),  Z₂ = 1/(G + sC') 형태에서
    R² = 1/C × C' 등의 소자값 조건 충족
```

실용 조건 기억법:

```
RC 정저항: R₁R₂ = (1/C)² 에 해당하는 소자값 관계
```

---

## 역회로망 (Inverse Network)

**두 회로망 Z₁과 Z₂가 역 관계에 있다는 뜻**

```
Z₁(s) × Z₂(s) = k²   (k: 반복계수, 실수 상수)
```

즉, 한 쪽이 인덕턴스 성분이면 다른 쪽은 캐패시턴스 성분이고, 서로 역수 관계가 된다.

| Z₁ (원 회로) | Z₂ (역 회로) |
|--------------|--------------|
| 직렬 RL | 병렬 RC |
| 병렬 LC | 직렬 LC |
| 저항 R | 컨덕턴스 G = k²/R |

비유: Z₁과 Z₂는 거울에 비친 관계. 한쪽이 크면 다른 쪽은 작아서 곱하면 항상 k²이다.

---

## 예제

**문제**: 다음 회로의 구동점 임피던스를 구하고, 극점과 영점을 찾아라.

```
      R=2Ω    L=1H
  ●───┤├───┬───UUU───●
          │
         ─┼─ C = 0.5F
          │
  ●───────┴──────────●
```

R과 (L 직렬 C)가 병렬로 연결된 회로

**풀이**:

```
Z_LC (직렬) = sL + 1/(sC) = s×1 + 1/(s×0.5) = s + 2/s = (s² + 2)/s

Z_전체 = R ∥ Z_LC = R × Z_LC / (R + Z_LC)

     = 2 × (s² + 2)/s
       ─────────────────
       2 + (s² + 2)/s

     = 2(s² + 2)/s
       ──────────────
       (2s + s² + 2)/s

     = 2(s² + 2)
       ──────────
       s² + 2s + 2
```

영점(분자 = 0):
```
s² + 2 = 0  →  s = ±j√2   (ω = √2 rad/s에서 단락 특성)
```

극점(분모 = 0):
```
s² + 2s + 2 = 0
s = (-2 ± √(4-8))/2 = -1 ± j1
→ 복소수 극점: s = -1+j, s = -1-j
```

## 시험 포인트

| 공식/개념 | 내용 | 기억 팁 |
|-----------|------|---------|
| 구동점 임피던스 | `Z(s) = V(s)/I(s)` | 전압/전류 |
| 구동점 어드미턴스 | `Y(s) = I(s)/V(s) = 1/Z(s)` | 임피던스 역수 |
| 영점(Zero) | Z=0이 되는 s → 단락 특성 | 분자=0 |
| 극점(Pole) | Z=∞이 되는 s → 개방 특성 | 분모=0 |
| 포스터 정리 | LC 리액턴스는 ω에 대해 단조 증가, 극점/영점 교대 | 항상 오르막 |
| 역 회로망 조건 | `Z₁ × Z₂ = k²` | 곱이 상수 |
| 정저항 조건 | 주파수 무관하게 Z = 일정 | 필터 설계 |

**자주 나오는 문제 유형**

1. Z(s) 표현식에서 극점/영점 찾기 (인수분해)
2. R, L, C 회로의 구동점 임피던스 계산
3. 포스터 정리 — 리액턴스 특성 판별 (참/거짓)
4. 역 회로망 조건 확인 (Z₁×Z₂=k² 검증)
