# 07. 테브난 · 노턴 정리

## 왜 필요한가?

복잡한 회로에서 **특정 저항(부하)에만 집중**하고 싶을 때,  
나머지 회로 전체를 간단한 등가 회로로 바꾸는 방법입니다.

---

## 테브난 정리 (Thevenin's Theorem)

### 핵심 개념

> 어떤 복잡한 회로도, 특정 두 단자에서 보면  
> **전압원 하나(Vth) + 직렬 저항 하나(Rth)** 로 표현할 수 있다.

```
복잡한 회로          테브난 등가 회로
                      
[복잡한     ]       +--[Rth]--+
[회로      ] →     [Vth]     [부하]
[      A,B단자]    -----------+
```

### 풀이 순서

1. **부하 저항(RL)을 제거**한다
2. **개방전압(Vth)** 구하기: 부하 제거 후 AB 단자 사이의 전압
3. **테브난 저항(Rth)** 구하기: **모든 전원을 제거**한 후 AB에서 본 합성저항
   - 전압원 → 단락
   - 전류원 → 개방
4. 테브난 등가 회로에 부하 연결 후 원하는 값 계산

### 예제

```
Vth = 10V, Rth = 5Ω, RL = 20Ω 연결 시
I_L = Vth / (Rth + RL) = 10 / (5+20) = 0.4 [A]
```

---

## 노턴 정리 (Norton's Theorem)

### 핵심 개념

> 어떤 복잡한 회로도, 특정 두 단자에서 보면  
> **전류원 하나(IN) + 병렬 저항 하나(RN)** 으로 표현할 수 있다.

```
복잡한 회로          노턴 등가 회로
                      
[복잡한     ]       +----+----+
[회로      ] →    [IN] [RN] [부하]
[      A,B단자]   +----+----+
```

### 노턴 전류(IN) 구하기

AB 단자를 **단락(Short)** 했을 때 흐르는 전류

---

## 테브난 ↔ 노턴 변환

두 정리는 서로 변환할 수 있습니다.

```
Vth = IN × RN
IN = Vth / Rth
Rth = RN  ← 저항은 동일!
```

---

## 한눈에 비교

| 구분 | 테브난 | 노턴 |
|------|------|------|
| 등가 전원 | 전압원 (Vth) | 전류원 (IN) |
| 연결 방식 | 직렬 (Rth) | 병렬 (RN) |
| 구하는 방법 | 개방전압 | 단락전류 |
| 저항 | 동일 (Rth = RN) | 동일 (Rth = RN) |

---

## 시험 포인트

- 시험에서는 **테브난**이 훨씬 더 자주 출제됨
- Rth를 구할 때 **전압원→단락, 전류원→개방** 기억
- 최대 전력 전달 조건: **RL = Rth** 일 때 부하에 최대 전력 전달
  ```
  P_max = Vth² / (4 × Rth)
  ```
- 테브난 등가 회로를 그린 후 나머지는 간단한 직렬 회로 계산
