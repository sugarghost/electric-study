# 15. 라플라스 변환과 전달함수 (제어이론 기초)

## 왜 필요한가?

회로이론 과목에는 **제어공학**이 포함되어 있습니다.  
제어공학의 핵심 도구가 바로 **라플라스 변환**과 **전달함수**입니다.

처음 보면 낯설지만, 시험에서는 패턴이 반복됩니다.

---

## 라플라스 변환이란?

시간 영역의 함수 f(t)를 복소수 영역 F(s)로 바꾸는 것

```
f(t) → [라플라스 변환] → F(s)
```

### 왜 쓰나?

- 시간 영역에서의 **미분방정식** → s 영역에서의 **대수방정식**
- 미분이 s 곱하기로, 적분이 s로 나누기로 변환되어 계산이 쉬워짐

---

## 자주 나오는 라플라스 변환 쌍

| f(t) | F(s) | 이름 |
|------|------|------|
| δ(t) | 1 | 단위 임펄스 |
| u(t) = 1 | 1/s | 단위 계단 |
| t | 1/s² | 램프 |
| e^(-at) | 1/(s+a) | 지수 감쇠 |
| sin(ωt) | ω/(s²+ω²) | 사인 |
| cos(ωt) | s/(s²+ω²) | 코사인 |

> 시험에서는 이 중 **1/s (계단), 1/(s+a) (지수)** 가 가장 자주 나옵니다.

---

## 전달함수 (Transfer Function)

> 시스템의 출력/입력 비율을 s 영역에서 표현한 것

```
G(s) = Y(s) / X(s) = 출력 / 입력
```

### 블록선도 (Block Diagram)

```
X(s) →[  G(s)  ]→ Y(s)

Y(s) = G(s) × X(s)
```

---

## 피드백 제어 시스템

실제 제어 시스템은 출력을 다시 입력으로 돌려보내는 **피드백**을 씁니다.

```
           +      E(s)          Y(s)
R(s) → [Σ] ----→[ G(s) ]------→
        ↑  -                    |
        |                       |
        +-------[ H(s) ]--------+
```

### 폐루프 전달함수

```
C(s)/R(s) = G(s) / (1 + G(s)·H(s))

H(s) = 1 (단위 피드백) 이면:
C(s)/R(s) = G(s) / (1 + G(s))
```

---

## 제어 시스템의 응답

### 1차 시스템

```
G(s) = K / (τs + 1)

- K: 정상상태 이득
- τ: 시정수 (시스템이 63.2% 응답하는 데 걸리는 시간)
```

### 2차 시스템

```
G(s) = ωn² / (s² + 2ζωns + ωn²)

- ωn: 고유 주파수 (Natural frequency)
- ζ (제타): 감쇠비 (Damping ratio)
```

| ζ 범위 | 특성 |
|--------|------|
| ζ > 1 | 과제동 (진동 없이 느리게 수렴) |
| ζ = 1 | 임계제동 (가장 빠르게 진동 없이 수렴) |
| 0 < ζ < 1 | 부족제동 (진동하며 수렴) |
| ζ = 0 | 무제동 (계속 진동) |

---

## 안정도 판별

시스템이 안정하려면 → **특성방정식의 근이 모두 s 평면의 왼쪽 반평면에** 있어야 함

### 루스 판별법 (Routh Criterion)

특성방정식의 계수로 안정도를 판별하는 방법  
(루스 배열을 만들어 첫 열의 부호 변화 횟수 = 불안정 극점의 수)

---

## 시험 포인트

| 항목 | 내용 |
|------|------|
| u(t) → 1/s | 계단 입력의 라플라스 |
| 폐루프 전달함수 | G/(1+GH) |
| 안정 조건 | 극점이 s 왼쪽 반평면 |
| ζ < 1 | 진동 응답 (부족제동) |
| ζ = 1 | 임계제동 |

- 제어이론은 처음엔 어렵게 느껴지지만 **패턴이 정해져 있음**
- 라플라스 변환 쌍 표를 외우면 절반은 해결
- 전달함수 → 폐루프 → 안정도의 흐름으로 문제가 출제됨
- 이해보다 **공식과 패턴 적용** 위주로 접근하는 것이 효율적
