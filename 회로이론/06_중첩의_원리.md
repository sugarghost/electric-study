# 06. 중첩의 원리 (Superposition Theorem)

## 핵심 개념

> 여러 개의 전원이 있는 회로에서,  
> **각 전원이 단독으로 있을 때의 결과를 따로 구한 뒤 더한다.**

---

## 언제 쓰나?

전원(전압원, 전류원)이 **2개 이상**인 회로에서  
특정 저항에 흐르는 전류나 걸리는 전압을 구할 때 사용합니다.

---

## 핵심 규칙: 나머지 전원 처리 방법

한 전원만 남기고 나머지를 **제거**합니다.

| 전원 종류 | 제거 방법 |
|---------|---------|
| 전압원 (배터리) | **단락(Short)** → 0V = 도선으로 대체 (선으로 연결) |
| 전류원 | **개방(Open)** → 0A = 연결 끊기 (선 제거) |

---

## 풀이 순서

1. 전원이 N개면 → N번 계산
2. 매번 하나의 전원만 살리고 나머지 제거
3. 각각의 전류/전압 구하기
4. 모두 더하기 (방향 주의!)

---

## 예제

```
회로: E1=12V, E2=6V, R1=4Ω, R2=2Ω, R3=4Ω
(E1과 E2 사이에 저항들이 있는 회로)
```

### Step 1: E1=12V만 살리기 (E2를 단락 = 도선으로 대체)

```
R2와 R3가 병렬: R23 = (2×4)/(2+4) = 8/6 = 4/3 Ω
전체 저항: R1 + R23 = 4 + 4/3 = 16/3 Ω
전체 전류: I = 12 / (16/3) = 12 × 3/16 = 2.25 A

R3에 흐르는 전류 I' (전류분배):
I' = 2.25 × R2/(R2+R3) = 2.25 × 2/(2+4) = 0.75 [A] (→ 방향)
```

### Step 2: E2=6V만 살리기 (E1을 단락 = 도선으로 대체)

```
R1과 R3가 병렬: R13 = (4×4)/(4+4) = 2 Ω
전체 저항: R2 + R13 = 2 + 2 = 4 Ω
전체 전류: I = 6/4 = 1.5 A

R3에 흐르는 전류 I'' (전류분배):
I'' = 1.5 × R1/(R1+R3) = 1.5 × 4/(4+4) = 0.75 [A] (← 방향, E2 기준)
```

### Step 3: 합산 (방향 확인 후)

```
두 전류가 R3에서 같은 방향이면: I_R3 = I' + I'' = 0.75 + 0.75 = 1.5 [A]
반대 방향이면: I_R3 = I' - I'' = 0 [A]
```

---

## 방향 주의!

각 전원으로 구한 전류가 **같은 방향이면 더하고**,  
**반대 방향이면 빼야** 합니다.

보통 문제에서 기준 방향을 정해주거나, 직접 방향을 가정하고 풀면 됩니다.

---

## 중첩의 원리가 적용되지 않는 것

- **전력(P = I²R)은 중첩이 안 됩니다.**  
  전력은 선형이 아니기 때문 (제곱 관계)
- 전류, 전압만 중첩 가능

---

## 시험 포인트

| 항목 | 내용 |
|------|------|
| 전압원 제거 | 단락 (선으로 연결) |
| 전류원 제거 | 개방 (선 끊기) |
| 전력 계산 | 최종 전류/전압 구한 후 계산 (중첩 X) |
| 방향 | 각 성분의 방향 고려해서 대수합 |

- 계산 실수를 줄이려면 각 Step을 독립된 새 회로로 그려서 풀기
- 전원이 2개면 2번, 3개면 3번 → 성실함이 핵심
