# 10. 인덕터(L)와 캐패시터(C)

## 왜 중요한가?

직류에서는 저항(R)만 있었지만,  
교류에서는 **인덕터(L)와 캐패시터(C)** 가 추가됩니다.  
이 두 소자가 교류 회로를 복잡하게 만드는 주범이자 핵심입니다.

---

## 인덕터 (Inductor, L)

### 물리적 의미

코일(나선형 도선)로 만들어진 소자.  
**전류가 변할 때 자기장을 이용해 변화에 저항**합니다.

### 비유

- 관성이 큰 물체 (갑자기 멈추거나 출발하기 힘든 것)
- 전류를 갑자기 바꾸려 하면 저항함

### 전기적 특성

| 항목 | 내용 |
|------|------|
| 단위 | 헨리 [H] |
| 직류에서 | 그냥 도선 (저항 0) |
| 교류에서 | 주파수가 높을수록 많이 방해함 |

### 리액턴스 (Reactance)

교류에서 인덕터가 전류를 방해하는 정도:

```
XL = ωL = 2πfL   [Ω]
```

- f(주파수) ↑ → XL ↑ (고주파는 더 많이 막음)
- f = 0 (직류) → XL = 0 (막지 않음)

### 위상 특성

> 인덕터에서 **전류는 전압보다 90° 뒤처집니다 (지상)**

```
전압이 먼저 올라가고, 전류는 90° 늦게 따라옴
기억법: "ELI" → E(전압)가 L(인덕터)에서 I(전류)보다 앞섬
```

---

## 캐패시터 (Capacitor, C)

### 물리적 의미

두 금속판 사이에 절연체를 넣은 소자.  
**전하를 저장**합니다. (일종의 전기 창고)

### 비유

- 물탱크 (물을 채웠다가 내보냄)
- 순간적으로 전류를 많이 흘릴 수 있음 (방전)

### 전기적 특성

| 항목 | 내용 |
|------|------|
| 단위 | 패럿 [F] (실제로는 μF, pF 단위 많이 씀) |
| 직류에서 | 전류 안 흐름 (개방) |
| 교류에서 | 주파수가 높을수록 잘 통과시킴 |

### 리액턴스 (Reactance)

교류에서 캐패시터가 전류를 방해하는 정도:

```
XC = 1/(ωC) = 1/(2πfC)   [Ω]
```

- f(주파수) ↑ → XC ↓ (고주파는 잘 통과)
- f = 0 (직류) → XC = ∞ (완전히 막음)

### 위상 특성

> 캐패시터에서 **전류는 전압보다 90° 앞섭니다 (진상)**

```
전류가 먼저 흐르고, 전압이 90° 늦게 올라옴
기억법: "ICE" → I(전류)가 C(캐패시터)에서 E(전압)보다 앞섬
```

---

## R, L, C 비교 요약

| 소자 | 기호 | 단위 | 임피던스 | 위상 | 직류 |
|------|------|------|---------|------|------|
| 저항 R | R | Ω | R | 변화 없음 | 그대로 |
| 인덕터 L | XL | H | jωL | 전류 지상 90° | 단락 |
| 캐패시터 C | XC | F | 1/jωC | 전류 진상 90° | 개방 |

---

## ELI the ICE man (암기법)

```
ELI  →  인덕터(L)에서 전압(E)이 전류(I)보다 앞섬
ICE  →  캐패시터(C)에서 전류(I)가 전압(E)보다 앞섬
```

---

## 시험 포인트

- XL = ωL, XC = 1/ωC 두 공식 반드시 암기
- 주파수 올라가면: XL ↑, XC ↓
- 인덕터는 고주파 차단, 캐패시터는 저주파 차단
- 직류: L은 단락, C는 개방으로 간주
