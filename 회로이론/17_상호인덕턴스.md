# 17. 상호 인덕턴스 (Mutual Inductance)

## 핵심 개념

> 한 코일에 흐르는 전류가 변할 때,  
> **근처에 있는 다른 코일에도 전압이 유도**되는 현상

변압기(Transformer)의 원리가 바로 이것입니다.

---

## 자기 인덕턴스 vs 상호 인덕턴스

| 구분 | 자기 인덕턴스 (L) | 상호 인덕턴스 (M) |
|------|--------------|--------------|
| 의미 | 자기 자신의 전류 변화 → 자신에게 유도 | 한쪽 전류 변화 → 상대방에게 유도 |
| 단위 | 헨리 [H] | 헨리 [H] |

---

## 결합계수 k

두 코일이 얼마나 잘 결합되어 있는지를 나타냄

```
k = M / √(L1 × L2)

0 ≤ k ≤ 1
```

| k 값 | 의미 |
|------|------|
| k = 1 | 완전결합 (이상적인 변압기) |
| k = 0 | 결합 없음 |
| 0 < k < 1 | 실제적인 결합 |

```
M = k × √(L1 × L2)   ← 시험 자주 출제
```

---

## 직렬 연결 시 합성 인덕턴스

### 가동 (같은 방향으로 결합)

```
L_합 = L1 + L2 + 2M
```

### 차동 (반대 방향으로 결합)

```
L_합 = L1 + L2 - 2M
```

> 가동은 자속이 서로 더해지고, 차동은 서로 상쇄됩니다.  
> 점(●) 표시로 결합 방향을 나타냅니다.

---

## 변압기 (Transformer)

### 이상 변압기의 관계식

```
V1/V2 = N1/N2 = a   (권수비)

V1 × I1 = V2 × I2   (전력 보존)
I1/I2 = N2/N1 = 1/a
```

- N1: 1차 코일 권수, N2: 2차 코일 권수
- a: 권수비

| 권수비 a > 1 | 승압 (전압 올라감, 전류 줄어듦) |
|------------|--------------------------|
| 권수비 a < 1 | 강압 (전압 내려감, 전류 늘어남) |

### 임피던스 변환

변압기로 임피던스를 변환할 수 있습니다.

```
Z1 = a² × Z2

(1차에서 본 2차 임피던스 = a² × 실제 2차 임피던스)
```

---

## 예제

**N1=100, N2=500, 1차 전압 220V 인가 시**

```
권수비: a = N1/N2 = 100/500 = 1/5

2차 전압: V2 = V1 × (N2/N1) = 220 × 5 = 1100 [V]

1차에 2A 흐르면:
I2 = I1 × (N1/N2) = 2 × (1/5) = 0.4 [A]
```

---

## 시험 포인트

| 공식 | 내용 |
|------|------|
| M = k√(L1L2) | 상호 인덕턴스 |
| 가동: L1+L2+2M | 같은 방향 직렬 |
| 차동: L1+L2-2M | 반대 방향 직렬 |
| V1/V2 = N1/N2 | 변압기 전압비 |
| I1/I2 = N2/N1 | 변압기 전류비 (역비) |
| Z1 = a²Z2 | 임피던스 변환 |

- **가동(+2M)과 차동(-2M)** 혼동 주의
- 변압기 전류는 권수비의 **역비** (전압과 반대)
- 이상 변압기는 전력 손실이 없음 (P1 = P2)
