# 23. 분포정수회로

## 핵심 개념

> **분포정수회로란, 선로(전선)의 길이가 신호의 파장에 비해 무시할 수 없을 만큼 길어질 때, 선로 자체의 저항·인덕턴스·캐패시턴스·컨덕턴스 성분을 위치마다 분산해서 고려하는 해석 방법이다.**

비유: 수도관 비유를 생각해보자.
- **짧은 수도관(집중정수)**: 수도꼭지를 틀면 거의 동시에 물이 나온다. 관 전체가 똑같이 동작한다고 봐도 된다.
- **긴 수도관(분포정수)**: 서울에서 부산까지 물을 보낸다면, 서울에서 보낸 물이 부산에 도착할 때까지 시간이 걸린다. 게다가 가다가 관이 좁아지거나 새는 구간도 있다. 이런 경우는 위치마다 다른 특성을 고려해야 한다.

---

## 집중정수 vs 분포정수

| 구분 | 집중정수 회로 | 분포정수 회로 |
|------|--------------|--------------|
| 선로 길이 | 파장(λ)에 비해 매우 짧음 | 파장(λ)과 비교할 만한 길이 |
| 적용 | 일반 가정용 회로, 저주파 | 장거리 송전선, 동축케이블, 마이크로파 |
| 소자 위치 | 한 점에 집중 | 선로 전체에 분포 |
| 전파 시간 | 무시 가능 | 반드시 고려 |

**언제 분포정수를 써야 하나?**

```
선로 길이 ≥ λ/10 (파장의 1/10 이상)이면 분포정수 효과 고려
```

예: 1GHz 신호의 파장 λ = c/f = 3×10⁸/10⁹ = 0.3m = 30cm
→ 선로 길이가 3cm 이상이면 분포정수 효과가 나타남

---

## 분포정수선로의 기본 파라미터

선로를 매우 짧은 구간 dz로 쪼개면, 각 구간은 아래 소자들을 갖는다:

```
단위 길이당 파라미터:

  R [Ω/m]  : 도체의 저항 (신호 손실)
  L [H/m]  : 도체의 인덕턴스 (자기장 에너지 저장)
  C [F/m]  : 두 도체 사이의 캐패시턴스 (전기장 에너지 저장)
  G [S/m]  : 절연체의 컨덕턴스 (누설 전류)
```

등가 회로:

```
  R·dz    L·dz
●──┤├──────UUU──●
               │   │
           G·dz ╪  ═ C·dz
               │   │
●──────────────●──●
            (구간 dz)
```

비유: 선로를 100m 호스라고 하면, 매 1m마다 호스 자체의 저항, 감긴 정도(인덕턴스), 주변 물질과의 정전 용량, 그리고 호스 벽을 통해 새는 정도(컨덕턴스)를 모두 갖는다.

---

## 전송선로 방정식

선로를 따라 전압 V(z)와 전류 I(z)는 위치 z에 따라 변한다:

$$\frac{dV}{dz} = -(R + j\omega L) I, \quad \frac{dI}{dz} = -(G + j\omega C) V$$

이를 연립하면 파동 방정식이 나온다:

$$\frac{d^2V}{dz^2} = \gamma^2 V, \quad \frac{d^2I}{dz^2} = \gamma^2 I$$

해(Solution):

$$V(z) = V^+ e^{-\gamma z} + V^- e^{+\gamma z}$$

$$I(z) = \frac{V^+}{Z_0} e^{-\gamma z} - \frac{V^-}{Z_0} e^{+\gamma z}$$

- V⁺e^(-γz): 입력에서 출력 방향으로 진행하는 파 (진행파)
- V⁻e^(+γz): 출력에서 입력 방향으로 반사되는 파 (반사파)

---

## 전파정수 γ (감마)

**"신호가 선로를 따라 진행하면서 얼마나 감쇠하고 위상이 변하는지 나타내는 복소수"**

$$\gamma = \sqrt{(R + j\omega L)(G + j\omega C)} = \alpha + j\beta$$

| 기호 | 이름 | 단위 | 의미 |
|------|------|------|------|
| α (알파) | 감쇠정수 | Np/m | 거리에 따라 신호 크기가 줄어드는 비율 |
| β (베타) | 위상정수 | rad/m | 거리에 따라 위상이 변하는 비율 |

비유:
- **α (감쇠정수)**: 멀리 갈수록 핸드폰 신호가 약해지는 정도. α가 클수록 신호가 빨리 죽는다.
- **β (위상정수)**: 파도가 얼마나 촘촘하게 진행하는지. β = 2π/λ (파장과 반비례)

**위상 속도 (전파 속도)**:

$$v_p = \frac{\omega}{\beta} \quad [\text{m/s}]$$

**파장**:

$$\lambda = \frac{2\pi}{\beta} \quad [\text{m}]$$

---

## 특성 임피던스 Z₀

**"선로 자체가 갖는 고유한 임피던스 — 선로 길이와 무관하고, 선로 재질·구조에만 의존한다"**

$$Z_0 = \sqrt{\frac{R + j\omega L}{G + j\omega C}} \quad [\Omega]$$

비유: 수도관에서 물이 흐를 때, 관의 굵기와 재질이 결정하는 고유 저항. 관이 길어도 관의 굵기가 같으면 특성 임피던스는 같다.

**중요**: 부하 임피던스 ZL을 Z₀과 같게 맞춰야 반사 없이 신호가 전달된다(임피던스 정합).

---

## 무손실 선로 (R=0, G=0)

실제로는 손실이 있지만, 고주파 해석에서 R≪ωL, G≪ωC이면 무손실로 근사할 수 있다.

```
조건: R = 0,  G = 0
```

무손실 조건 대입:

$$\gamma = \sqrt{j\omega L \cdot j\omega C} = \sqrt{-\omega^2 LC} = j\omega\sqrt{LC}$$

$$\Rightarrow \alpha = 0, \quad \beta = \omega\sqrt{LC} \quad \text{(감쇠 없음!)}$$

$$Z_0 = \sqrt{\frac{j\omega L}{j\omega C}} = \sqrt{\frac{L}{C}} \quad [\Omega] \quad \text{(순실수!)}$$

$$v_p = \frac{\omega}{\beta} = \frac{\omega}{\omega\sqrt{LC}} = \frac{1}{\sqrt{LC}} \quad [\text{m/s}]$$

**무손실 선로 정리**

| 항목 | 수식 |
|------|------|
| 특성 임피던스 | $Z_0 = \sqrt{\dfrac{L}{C}}$ |
| 전파정수 | $\gamma = j\beta = j\omega\sqrt{LC}$ |
| 감쇠정수 | $\alpha = 0$ (손실 없음) |
| 위상정수 | $\beta = \omega\sqrt{LC}$ |
| 전파 속도 | $v = \dfrac{1}{\sqrt{LC}}$ |

비유: 마찰이 없는 매끄러운 관에서 물이 흐르는 것. 처음 속도와 압력 그대로 끝까지 전달된다.

---

## 무왜형 조건 (Distortionless Condition, Heaviside 조건)

**"신호 파형이 찌그러지지 않고 그대로 전달되려면?"**

일반적인 선로에서는 주파수마다 감쇠율(α)이 달라서 신호 파형이 변형된다. 이를 방지하는 조건이 Heaviside 조건이다:

$$\frac{R}{L} = \frac{G}{C}$$

이 조건이 만족되면:

$$\alpha = R\sqrt{\frac{C}{L}} = G\sqrt{\frac{L}{C}} \quad \text{(주파수에 독립! → 모든 주파수가 같은 비율로 감쇠)}$$

$$\beta = \omega\sqrt{LC}, \quad Z_0 = \sqrt{\frac{L}{C}} \quad \text{(순실수)}$$

비유: 모든 악기 소리(다양한 주파수)가 녹음 스튜디오에서 같은 크기로 줄어들어야 원래 음악이 재생된다. R/L = G/C 조건은 그런 균일한 감쇠를 보장한다.

실용적 의미: 예전 전화선에서 Oliver Heaviside가 이 조건을 제안해 통화 음질을 크게 개선했다. 선로에 인덕터를 추가해 L을 키워 R/L = G/C 조건을 맞췄다.

## 반사계수와 정재파

### 반사가 생기는 이유

선로 끝(부하)의 임피던스 ZL이 특성 임피던스 Z₀과 다르면, 진행하던 파가 부하에서 일부 반사된다.

```
     진행파 →                  반사파 ←
●────────────────────────────────────● ZL
                                    ↑
                              임피던스 불일치!
```

비유: 고속도로가 갑자기 좁아지면 차들이 막혀서 되돌아오는 것처럼, 임피던스가 갑자기 바뀌면 신호가 반사된다.

### 반사계수 Γ (감마)

$$\Gamma = \frac{Z_L - Z_0}{Z_L + Z_0}$$

| 부하 조건 | ZL | 반사계수 Γ | 의미 |
|-----------|-----|------------|------|
| 정합 (임피던스 일치) | ZL = Z₀ | Γ = 0 | 반사 없음 ✓ |
| 개방 | ZL = ∞ | Γ = +1 | 완전 반사 (동위상) |
| 단락 | ZL = 0 | Γ = -1 | 완전 반사 (역위상) |

### 정재파 (Standing Wave)

진행파와 반사파가 겹치면 파형이 고정된 것처럼 보이는 정재파가 생긴다.

```
      진행파 + 반사파 = 정재파

최대값 지점: 마디(Anti-node)   ← 전압이 최대
최소값 지점: 배(Node)          ← 전압이 최소
```

**정재파비 (SWR, Standing Wave Ratio)**:

$$SWR = \frac{1 + |\Gamma|}{1 - |\Gamma|} \quad (\text{항상} \geq 1)$$

- SWR = 1: 반사 없음 (이상적인 정합)
- SWR > 1: 반사 있음 (클수록 반사가 심함)

## 예제

**문제 1**: 특성 임피던스를 구하여라.

단위 길이당 L = 0.25 μH/m, C = 100 pF/m인 무손실 전송 선로의 특성 임피던스를 구하여라.

**풀이**:

무손실 선로이므로:

$$Z_0 = \sqrt{\frac{L}{C}} = \sqrt{\frac{0.25 \times 10^{-6}}{100 \times 10^{-12}}} = \sqrt{\frac{0.25 \times 10^{-6}}{10^{-10}}} = \sqrt{0.25 \times 10^4} = \sqrt{2500} = 50\ \Omega$$

답: **Z₀ = 50Ω** (동축케이블의 표준 임피던스)

---

**문제 2**: 반사계수를 구하여라.

특성 임피던스 Z₀ = 50Ω인 선로에 ZL = 100Ω의 부하를 연결했다. 반사계수와 SWR을 구하여라.

**풀이**:

$$\Gamma = \frac{Z_L - Z_0}{Z_L + Z_0} = \frac{100 - 50}{100 + 50} = \frac{50}{150} = \frac{1}{3} \approx 0.333$$

$$SWR = \frac{1 + |\Gamma|}{1 - |\Gamma|} = \frac{1 + \tfrac{1}{3}}{1 - \tfrac{1}{3}} = \frac{\tfrac{4}{3}}{\tfrac{2}{3}} = 2$$

답: **Γ = 1/3, SWR = 2**

## 시험 포인트

| 공식/개념 | 수식 | 조건 | 기억 팁 |
|-----------|------|------|---------|
| 전파정수 | `γ = √((R+jωL)(G+jωC)) = α+jβ` | 일반 | α=감쇠, β=위상 |
| 특성 임피던스 | `Z₀ = √((R+jωL)/(G+jωC))` | 일반 | 길이 무관 |
| **무손실: Z₀** | `Z₀ = √(L/C)` | R=0, G=0 | 순실수 |
| **무손실: γ** | `γ = jω√(LC)` | R=0, G=0 | 감쇠 없음 |
| **무손실: 속도** | `v = 1/√(LC)` | R=0, G=0 | 자주 출제 |
| 무왜형 조건 | `R/L = G/C` | Heaviside | 파형 왜곡 없음 |
| 반사계수 | `Γ = (ZL-Z₀)/(ZL+Z₀)` | 부하 불일치 | 정합이면 Γ=0 |
| SWR | `SWR = (1+\|Γ\|)/(1-\|Γ\|)` | 항상 ≥1 | 정합이면 SWR=1 |

**자주 나오는 문제 유형**

1. L, C가 주어졌을 때 무손실 선로의 Z₀, 전파 속도 계산
2. ZL과 Z₀가 주어졌을 때 반사계수 Γ, SWR 계산
3. 무왜형 조건 R/L = G/C — 성립 여부 판단
4. 집중정수/분포정수 구분 — 선로 길이와 파장의 비교
5. α = 0이 되는 조건 → 무손실(R=0, G=0) 확인
