# 14. 3상 교류 (Three-Phase AC)

## 왜 3상인가?

가정용은 단상(1상) 220V를 씁니다.  
하지만 공장, 대형 건물, 발전/송전은 **3상**을 씁니다.

### 3상의 장점

- 전력 전달 효율이 높음
- 전동기 회전 방향이 자연스럽게 결정됨
- 같은 전력을 보낼 때 도선이 적게 필요

---

## 3상의 기본 개념

3개의 교류 전압이 **120°씩 위상 차이**를 두고 존재

```
Va = Vm · sin(ωt)
Vb = Vm · sin(ωt - 120°)
Vc = Vm · sin(ωt - 240°) = Vm · sin(ωt + 120°)
```

```
    Va
   /
  / 120°
Vc --- Vb
```

특징: Va + Vb + Vc = 0 (항상 0)

---

## 결선 방식: Y결선 vs Δ결선

### Y결선 (스타결선, 성형결선)

```
      A
      |
      |
  B---N---C   (N: 중성점)
```

중성점(N)에서 각 선이 뻗어 나오는 형태

| 항목 | 관계식 |
|------|------|
| 선간전압 VL | VL = √3 × 상전압 VP |
| 선전류 IL | IL = 상전류 IP |

> **선간전압 = √3 × 상전압**  
> 220V 상전압 → 선간전압 = 220 × 1.732 ≈ 380V

### Δ결선 (델타결선, 삼각결선)

삼각형 형태로 연결

| 항목 | 관계식 |
|------|------|
| 선간전압 VL | VL = 상전압 VP |
| 선전류 IL | IL = √3 × 상전류 IP |

> **선전류 = √3 × 상전류**

---

## Y ↔ Δ 관계 요약

| 구분 | Y결선 | Δ결선 |
|------|------|------|
| 전압 | VL = √3 · Vp | VL = Vp |
| 전류 | IL = Ip | IL = √3 · Ip |

외우는 팁:
- **Y**: 전압에 √3 (선간전압이 더 큼)
- **Δ**: 전류에 √3 (선전류가 더 큼)

---

## 3상 전력

### 3상 전체 전력

```
P = √3 · VL · IL · cosθ   [W]   ← 가장 자주 쓰는 공식
Q = √3 · VL · IL · sinθ   [Var]
S = √3 · VL · IL           [VA]
```

> VL (선간전압), IL (선전류) 기준  
> 상전압/상전류 기준으로도 풀 수 있으나, 위 공식이 보편적

---

## Y-Δ 변환

Y결선과 Δ결선은 서로 변환할 수 있습니다.

### Y → Δ 변환

```
ZΔ = (Za·Zb + Zb·Zc + Zc·Za) / 대각 Z
```

**같은 저항 R로 Y와 Δ 연결 시:**

```
RΔ = 3 · RY
RY = RΔ / 3
```

---

## 예제

3상 Y결선, 상전압 220V, 부하 R=10Ω (각 상)

```
선간전압: VL = √3 × 220 ≈ 380 [V]
상전류:   Ip = Vp/R = 220/10 = 22 [A]
선전류:   IL = Ip = 22 [A]  (Y결선이므로)

전체 전력: P = √3 × 380 × 22 × 1 ≈ 14,484 [W] ≈ 14.5 [kW]
또는:      P = 3 × Ip² × R = 3 × 22² × 10 = 14,520 [W]
```

---

## 시험 포인트

| 공식 | 내용 |
|------|------|
| Y: VL = √3·VP | Y결선 전압 관계 |
| Δ: IL = √3·IP | Δ결선 전류 관계 |
| P = √3·VL·IL·cosθ | 3상 유효전력 |
| RΔ = 3RY | Y↔Δ 저항 변환 |

- √3 ≈ 1.732 기억 (계산에 자주 사용)
- 한국 상용 3상: 선간전압 380V, 상전압 220V (Y결선 기준)
- 3상 전력 공식은 **암기 필수** — 매회 출제
