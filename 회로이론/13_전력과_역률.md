# 13. 전력과 역률

## 교류 전력의 종류

직류는 P = VI 하나였지만,  
교류에서는 위상 차이 때문에 전력이 **3가지**로 나뉩니다.

---

## 3가지 전력

### 1. 유효전력 P (Active Power, 실전력)

> 실제로 **일을 하는** 전력 (열, 빛, 운동 등)

```
P = VI·cosθ = I²R   [W, 와트]
```

- cosθ: 역률
- 저항에서만 소비됨

### 2. 무효전력 Q (Reactive Power)

> 실제 일은 안 하고 **L, C와 전원 사이를 왔다 갔다** 하는 전력

```
Q = VI·sinθ = I²X   [Var, 바르]
```

- L에서 발생: QL = I²XL (지상 무효전력)
- C에서 발생: QC = I²XC (진상 무효전력)

### 3. 피상전력 S (Apparent Power)

> 전원이 공급하는 **전체 전력** (유효+무효를 합한 겉보기 전력)

```
S = VI = I²Z   [VA, 볼트암페어]
```

---

## 전력 삼각형

```
       S (피상전력)
      /|
     / |
    /  | Q (무효전력)
   /θ  |
  /    |
 P (유효전력)
```

```
S² = P² + Q²
S = √(P² + Q²)

P = S·cosθ
Q = S·sinθ
```

---

## 역률 (Power Factor, cosθ)

> **피상전력 중 실제로 쓸모 있는 비율**

```
cosθ = P/S = R/Z
```

| 역률 | 의미 |
|------|------|
| 1.0 | 완벽 (저항만 있는 경우, 공진 상태) |
| 0.8 | 피상전력의 80%만 유효 |
| 0.0 | 유효전력 0 (L 또는 C만 있는 경우) |

### 역률 개선

역률이 낮으면 → 같은 일을 하는데 더 많은 전류 필요 → 손실 ↑

역률 개선 방법:
- **유도성 부하(L)**: 캐패시터(C)를 병렬로 추가
- **용량성 부하(C)**: 인덕터(L)를 병렬로 추가

---

## 복소전력 (Complex Power)

```
S = P + jQ   [VA]
```

- P > 0: 유효전력 소비
- Q > 0: 유도성 (지상)
- Q < 0: 용량성 (진상)

---

## 예제

R=3Ω, XL=4Ω 직렬, V=100V

```
Z = 3 + j4   → |Z| = 5Ω

I = V/Z = 100/5 = 20 [A]

P = I²R = 20² × 3 = 1200 [W]
Q = I²X = 20² × 4 = 1600 [Var]
S = VI = 100 × 20 = 2000 [VA]

cosθ = P/S = 1200/2000 = 0.6
또는 cosθ = R/Z = 3/5 = 0.6
```

---

## 시험 포인트

| 공식 | 내용 |
|------|------|
| P = VIcosθ | 유효전력 [W] |
| Q = VIsinθ | 무효전력 [Var] |
| S = VI | 피상전력 [VA] |
| cosθ = P/S = R/Z | 역률 |
| S² = P² + Q² | 전력 삼각형 |

- 단위 구분: W(유효), Var(무효), VA(피상) 반드시 암기
- 역률 1에 가까울수록 효율적인 회로
- 전력 문제는 거의 매회 출제 → 완벽히 익혀야 함
