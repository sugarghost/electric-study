# 24. 제어계 과도응답 (Transient Response)

## 핵심 개념

> 목표값을 갑자기 바꿨을 때, 출력이 목표에 도달하는 과정을 수치로 표현하는 것

**비유:** 에어컨 설정 온도를 20도 → 26도로 바꾸면, 실제 방 온도가 즉시 26도가 되지 않습니다.  
조금씩 올라가다가 26도를 넘어 27도까지 갔다가, 진동하면서 결국 26도에 안정되는 과정 — 이것이 과도응답입니다.

---

## 계단 응답 성능 지표

계단 입력(Step Input): 입력이 0에서 갑자기 1로 바뀌는 것

```
지연시간 td: 응답이 최종값의 50%에 처음 도달하는 시간
상승시간 tr: 최종값의 10% → 90% 구간을 통과하는 시간
최대 오버슈트 Mp: 최종값을 초과한 최댓값의 비율 [%]
정착시간 ts: 출력이 최종값의 ±2% (또는 ±5%) 이내에 들어와서 유지되는 시간
```

> 시험에는 **Mp (최대 오버슈트), ts (정착시간)** 계산이 특히 많이 출제됩니다.

---

## 2차 시스템 표준형

제어공학에서 가장 많이 사용하는 기본 모델:

```
G(s) = ωn² / (s² + 2ζωn·s + ωn²)
```

| 기호 | 이름 | 역할 |
|------|------|------|
| ωn | 고유 주파수 [rad/s] | 클수록 응답이 빠름 |
| ζ (제타) | 감쇠비 | 진동 특성 결정 |

---

## 감쇠비(ζ)에 따른 응답 분류

```
ζ > 1   : 과제동(Overdamped)  — 느리게 수렴, 진동 없음
ζ = 1   : 임계제동(Critically damped) — 진동 없이 가장 빠르게 수렴
0 < ζ < 1 : 부족제동(Underdamped) — 진동하면서 수렴 (실제로 가장 많이 사용)
ζ = 0   : 무제동(Undamped)    — 계속 진동, 수렴하지 않음
```

**실용적으로는 ζ = 0.4 ~ 0.8 범위를 주로 사용합니다.**

---

## 핵심 공식 (부족제동 0 < ζ < 1)

### 감쇠 고유주파수

```
ωd = ωn × √(1 - ζ²)
```

### 최대 오버슈트 Mp

```
Mp = exp(-π·ζ / √(1-ζ²)) × 100 [%]
```

ζ가 클수록 Mp는 작아집니다 (진동이 줄어듦).

### 정착시간 ts (2% 기준 근사)

```
ts ≈ 4 / (ζ·ωn)
```

### 상승시간 tr (부족제동 근사)

```
tr ≈ π / ωd = π / (ωn × √(1-ζ²))
```

---

## 트레이드오프 관계

| 원하는 것 | 희생하는 것 |
|----------|-----------|
| 빠른 응답 (작은 tr) | 큰 오버슈트 Mp |
| 작은 오버슈트 | 느린 응답 (큰 tr, ts) |
| 빠른 정착 (작은 ts) | 큰 ωn 필요 → 설계 어려움 |

> **빠르고 진동도 없는 응답**은 동시에 얻기 어렵습니다. 설계에서 타협이 필요합니다.

---

## 예제

**조건:** ζ = 0.5, ωn = 10 rad/s 인 2차 시스템

**① 최대 오버슈트 Mp 계산**

```
Mp = exp(-π × 0.5 / √(1 - 0.5²)) × 100

분모: √(1 - 0.25) = √0.75 ≈ 0.866

지수부: -π × 0.5 / 0.866 = -1.5708 / 0.866 ≈ -1.814

Mp = exp(-1.814) × 100 ≈ 0.163 × 100 = 16.3 [%]
```

**② 정착시간 ts 계산 (2% 기준)**

```
ts ≈ 4 / (ζ × ωn) = 4 / (0.5 × 10) = 4 / 5 = 0.8 [초]
```

**③ 감쇠 고유주파수 ωd**

```
ωd = 10 × √(1 - 0.25) = 10 × 0.866 ≈ 8.66 [rad/s]
```

**④ 상승시간 tr**

```
tr ≈ π / ωd = 3.14159 / 8.66 ≈ 0.363 [초]
```

---

## 시험 포인트

| 항목 | 공식 | 비고 |
|------|------|------|
| 최대 오버슈트 Mp | `exp(-πζ/√(1-ζ²)) × 100` | ζ 클수록 작아짐 |
| 정착시간 ts | `4/(ζωn)` | 2% 기준 근사 |
| 상승시간 tr | `π/(ωn√(1-ζ²))` | 부족제동 근사 |
| 감쇠 고유주파수 ωd | `ωn√(1-ζ²)` | 실제 진동 주파수 |
| 임계제동 조건 | `ζ = 1` | 오버슈트 없는 최속 응답 |
| 안정 범위 | `ζ > 0` | ζ < 0이면 발산 |

- ζ = 0.5, ωn = 10 같은 구체적인 수치 대입 계산이 자주 출제됨
- "오버슈트를 줄이려면?" → **ζ를 크게** 하거나 **ωn을 줄임**
- 정착시간이 짧으려면 → **ζ·ωn 곱을 크게** 해야 함
