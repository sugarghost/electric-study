# 18. 비정현파 (Non-Sinusoidal Wave)

## 핵심 개념

> 순수한 사인파가 아닌 **찌그러진 파형**  
> 실제 전력 시스템에서 발생하는 고조파(Harmonic) 문제의 기초

---

## 푸리에 급수 (Fourier Series)

어떤 주기 파형이든 **사인파들의 합**으로 표현할 수 있습니다.

```
f(t) = a₀ + a₁sin(ωt) + a₂sin(2ωt) + a₃sin(3ωt) + ...
       + b₁cos(ωt) + b₂cos(2ωt) + ...
```

| 용어 | 의미 |
|------|------|
| a₀ | 직류 성분 (평균값) |
| 기본파 (1고조파) | ω 성분 (원래 주파수) |
| 2고조파 | 2ω 성분 |
| n고조파 | nω 성분 |

---

## 비정현파의 실효값

각 고조파 실효값의 제곱합의 제곱근

```
V_rms = √(V₀² + V₁² + V₂² + V₃² + ...)

- V₀: 직류 성분
- V₁, V₂, V₃: 각 고조파의 실효값
```

### 예제

직류 5V + 기본파 10V(실효) + 3고조파 5V(실효)인 비정현파의 실효값은?

```
V_rms = √(5² + 10² + 5²) = √(25+100+25) = √150 ≈ 12.25 [V]
```

---

## 비정현파의 전력

```
P = V₀I₀ + V₁I₁cosθ₁ + V₂I₂cosθ₂ + ...
```

> **같은 주파수 성분끼리만** 전력을 주고받습니다.  
> 다른 주파수 성분 사이에서는 전력이 0.

---

## 왜기율 (THD, Total Harmonic Distortion)

기본파에 대한 고조파 성분의 비율

```
THD = √(V₂² + V₃² + ...) / V₁ × 100 [%]
```

- THD가 낮을수록 깨끗한 사인파
- 전력 품질의 지표

---

## 파형의 대칭성 (시험 자주 출제)

| 대칭 종류 | 특징 | 포함 성분 |
|---------|------|---------|
| 반파 대칭 | f(t) = -f(t + T/2) | 홀수 고조파만 존재 |
| 우함수 대칭 | f(t) = f(-t) | 코사인 항만 존재 |
| 기함수 대칭 | f(t) = -f(-t) | 사인 항만 존재 |

> "반파 대칭이면 홀수 고조파만 있다" → 자주 출제되는 패턴

---

## 시험 포인트

| 공식 | 내용 |
|------|------|
| V_rms = √(V₀²+V₁²+V₂²+...) | 비정현파 실효값 |
| 같은 주파수끼리만 전력 발생 | 전력 계산 원칙 |
| 반파 대칭 → 홀수 고조파만 | 파형 대칭성 |

- 실효값 공식에서 **제곱합의 제곱근** 잊지 말 것
- 비정현파 전력: 주파수 다른 성분 사이 전력 = 0
- 고조파 차수가 높을수록 주파수가 높고, 일반적으로 크기가 작아짐
