# 22. 4단자 회로망

## 핵심 개념

> **4단자 회로망이란, 입력 포트와 출력 포트가 각각 한 쌍씩 있는 블랙박스 회로다. 내부가 어떻게 생겼는지 몰라도, 입출력 관계를 나타내는 파라미터(Z, Y, ABCD, h)만 알면 회로를 완전히 분석할 수 있다.**

비유: TV 리모컨(입력 포트)과 TV 화면(출력 포트)이 있다고 하자. 리모컨 신호가 TV 내부 회로를 거쳐 화면으로 나오는 과정을 내부 회로도 없이 "채널 버튼 → 채널 바뀜" 관계만으로 표현하는 것이 4단자 파라미터다.

> **이 단원은 전기기사 시험에서 가장 자주 출제됩니다. ABCD 파라미터는 반드시 암기하세요.**

---

## 4단자 회로망 기본 구조

```
        I₁ →        I₂ →
   ●────────┬──────────────┬────────● 
   +        │              │        +
  V₁      [내부회로]      V₂
   -        │              │        -
   ●────────┴──────────────┴────────●
  포트 1 (입력)           포트 2 (출력)
```

- 포트 1: 입력 단자 (전압 V₁, 전류 I₁)
- 포트 2: 출력 단자 (전압 V₂, 전류 I₂)
- **전류 방향 주의**: I₁은 포트로 들어가는 방향, I₂는 포트에서 나가는 방향이 기준

---

## Z 파라미터 (개방회로 임피던스 파라미터)

**"출력을 개방(연결 안 함)하고 임피던스를 측정하는 방법"**

```
V₁ = Z₁₁I₁ + Z₁₂I₂
V₂ = Z₂₁I₁ + Z₂₂I₂
```

행렬로 표현:

```
[V₁]   [Z₁₁  Z₁₂] [I₁]
[V₂] = [Z₂₁  Z₂₂] [I₂]
```

### 각 파라미터 의미

| 파라미터 | 측정 조건 | 공식 | 의미 |
|----------|-----------|------|------|
| Z₁₁ | I₂ = 0 (출력 개방) | Z₁₁ = V₁/I₁ | 출력 개방 시 입력 임피던스 |
| Z₁₂ | I₁ = 0 (입력 개방) | Z₁₂ = V₁/I₂ | 입력 개방 시 역방향 전달 임피던스 |
| Z₂₁ | I₂ = 0 (출력 개방) | Z₂₁ = V₂/I₁ | 출력 개방 시 순방향 전달 임피던스 |
| Z₂₂ | I₁ = 0 (입력 개방) | Z₂₂ = V₂/I₂ | 입력 개방 시 출력 임피던스 |

**측정 방법 기억법**: Z 파라미터는 반대쪽 전류를 0으로 만들 때(개방), 전압을 전류로 나눈다.

### 상반 회로망 조건

수동 소자(R, L, C)로만 이루어진 회로는 **상반(Reciprocal) 회로망**이다:

```
Z₁₂ = Z₂₁
```

비유: A→B로 신호를 보낼 때와 B→A로 보낼 때 같은 특성을 갖는다. 수동 소자는 방향성이 없기 때문이다.

---

## Y 파라미터 (단락회로 어드미턴스 파라미터)

**"출력을 단락(0V로 연결)하고 어드미턴스를 측정하는 방법"**

```
I₁ = Y₁₁V₁ + Y₁₂V₂
I₂ = Y₂₁V₁ + Y₂₂V₂
```

행렬로 표현:

```
[I₁]   [Y₁₁  Y₁₂] [V₁]
[I₂] = [Y₂₁  Y₂₂] [V₂]
```

### 각 파라미터 의미

| 파라미터 | 측정 조건 | 공식 | 의미 |
|----------|-----------|------|------|
| Y₁₁ | V₂ = 0 (출력 단락) | Y₁₁ = I₁/V₁ | 출력 단락 시 입력 어드미턴스 |
| Y₁₂ | V₁ = 0 (입력 단락) | Y₁₂ = I₁/V₂ | 입력 단락 시 역방향 전달 어드미턴스 |
| Y₂₁ | V₂ = 0 (출력 단락) | Y₂₁ = I₂/V₁ | 출력 단락 시 순방향 전달 어드미턴스 |
| Y₂₂ | V₁ = 0 (입력 단락) | Y₂₂ = I₂/V₂ | 입력 단락 시 출력 어드미턴스 |

**Z와 Y의 관계**:

```
[Y] = [Z]⁻¹   (역행렬 관계)
```

상반 회로망 조건: Y₁₂ = Y₂₁

---

## ABCD 파라미터 (전송 파라미터) ★★★ 가장 자주 출제

**"신호가 입력에서 출력으로 전달되는 특성을 표현하는 방법"**

증폭기, 필터, 전송선로 등 신호가 통과하는 회로 분석에 특히 유용하다.

```
V₁ = AV₂ - BI₂
I₁ = CV₂ - DI₂
```

행렬로 표현:

```
[V₁]   [A  B] [ V₂]
[I₁] = [C  D] [-I₂]
```

주의: I₂ 앞에 마이너스(-)가 붙는다. 출력 전류를 나가는 방향으로 정의하기 때문이다.

### 각 파라미터 의미

| 파라미터 | 측정 조건 | 공식 | 단위 | 의미 |
|----------|-----------|------|------|------|
| A | I₂ = 0 (출력 개방) | A = V₁/V₂ | 무차원 | 전압비 (역방향) |
| B | V₂ = 0 (출력 단락) | B = V₁/(-I₂) | Ω | 전달 임피던스 |
| C | I₂ = 0 (출력 개방) | C = I₁/V₂ | S | 전달 어드미턴스 |
| D | V₂ = 0 (출력 단락) | D = I₁/(-I₂) | 무차원 | 전류비 (역방향) |

### 상반 회로망 조건 (중요!)

```
AD - BC = 1
```

수동 소자로 이루어진 회로는 반드시 AD - BC = 1을 만족한다. 계산 후 이 조건으로 검증할 수 있다.

### 기본 소자의 ABCD 파라미터 ★ 암기 필수

**1. 직렬 임피던스 Z**

```
    I₁ →    Z      I₂ →
●───────[ZZZZ]───────●
+                    +
V₁                  V₂
-                    -
●────────────────────●

[A  B]   [1  Z]
[C  D] = [0  1]
```

**2. 병렬 어드미턴스 Y**

```
    I₁ →             I₂ →
●───────────┬────────────●
+           │            +
V₁         [Y]          V₂
-           │            -
●───────────┴────────────●

[A  B]   [1  0]
[C  D] = [Y  1]
```

**3. 이상 변압기 (권수비 n:1)**

```
[A  B]   [n   0  ]
[C  D] = [0  1/n ]
```

### 종속 접속 (Cascade Connection) ★★ 자주 출제

여러 4단자 회로망이 앞뒤로 연결될 때, 전체 ABCD 파라미터는 각각의 행렬 곱이다.

```
[회로망 1] → [회로망 2] → [회로망 3]

[ABCD]_전체 = [ABCD₁] × [ABCD₂] × [ABCD₃]
```

비유: 번역가 3명이 연속으로 통역할 때(한국어→영어→일어→중국어), 각 통역의 특성을 행렬 곱으로 합칠 수 있다.

**중요**: 행렬 곱은 순서가 중요하다. [ABCD₁][ABCD₂] ≠ [ABCD₂][ABCD₁]이다.

## h 파라미터 (혼합 파라미터)

**"트랜지스터 회로 분석에 주로 사용하는 파라미터"**

V₁과 I₂를 출력으로, I₁과 V₂를 입력으로 섞어서 표현한다:

```
V₁ = h₁₁I₁ + h₁₂V₂
I₂ = h₂₁I₁ + h₂₂V₂
```

### 각 파라미터 의미

| 파라미터 | 측정 조건 | 공식 | 단위 | 의미 |
|----------|-----------|------|------|------|
| h₁₁ | V₂ = 0 (출력 단락) | h₁₁ = V₁/I₁ | Ω | 입력 임피던스 |
| h₁₂ | I₁ = 0 (입력 개방) | h₁₂ = V₁/V₂ | 무차원 | 역방향 전압비 |
| h₂₁ | V₂ = 0 (출력 단락) | h₂₁ = I₂/I₁ | 무차원 | 순방향 전류 이득 (β) |
| h₂₂ | I₁ = 0 (입력 개방) | h₂₂ = I₂/V₂ | S | 출력 어드미턴스 |

트랜지스터에서 h₂₁ = hFE = β (전류 증폭률)로 매우 중요하다.

---

## 파라미터 간 변환

| 변환 | 핵심 공식 |
|------|-----------|
| Z → Y | [Y] = [Z]⁻¹, Y₁₁ = Z₂₂/ΔZ, Y₁₂ = -Z₁₂/ΔZ (ΔZ = Z₁₁Z₂₂ - Z₁₂Z₂₁) |
| Y → Z | [Z] = [Y]⁻¹ |
| ABCD → Z | Z₁₁ = A/C, Z₁₂ = (AD-BC)/C = 1/C, Z₂₁ = 1/C, Z₂₂ = D/C |
| Z → ABCD | A = Z₁₁/Z₂₁, B = ΔZ/Z₂₁, C = 1/Z₂₁, D = Z₂₂/Z₂₁ |

## 예제: T형 회로망의 ABCD 파라미터 계산

**문제**: 아래 T형 회로망의 ABCD 파라미터를 구하여라.

```
    Z₁         Z₂
●──[ZZZZ]──┬──[ZZZZ]──●
           │
          [Z₃]
           │
●──────────┴──────────●
```

Z₁ = Z₂ = Z = 1Ω (직렬), Z₃ = 2Ω (병렬)

**풀이**: T형 회로는 "직렬Z₁ → 병렬Z₃ → 직렬Z₂"의 종속 접속으로 볼 수 있다.

```
[ABCD]_직렬Z₁ = [1  Z₁] = [1  1]
                [0   1]   [0  1]

[ABCD]_병렬Z₃ = [1   0  ] = [1    0  ]
                [1/Z₃ 1 ]   [1/2  1  ]

[ABCD]_직렬Z₂ = [1  Z₂] = [1  1]
                [0   1]   [0  1]
```

전체 ABCD = [직렬Z₁] × [병렬Z₃] × [직렬Z₂]:

```
먼저 [직렬Z₁] × [병렬Z₃]:

[1  1] × [1    0] = [1 + 1/2    1] = [3/2  1]
[0  1]   [1/2  1]   [    1/2    1]   [1/2  1]

다음 결과 × [직렬Z₂]:

[3/2  1] × [1  1] = [3/2 + 1/2   3/2 + 1] = [2   5/2]
[1/2  1]   [0  1]   [1/2 + 0     1/2 + 1]   [1/2  3/2]
```

따라서:

```
A = 2,  B = 5/2 = 2.5Ω,  C = 1/2 = 0.5S,  D = 3/2 = 1.5
```

**검증** (AD - BC = 1이어야 함):

```
AD - BC = 2 × 3/2 - 5/2 × 1/2 = 3 - 5/4 = 7/4 ≠ 1  ???
```

Z₁ = Z₂이고 대칭 회로이므로 다시 계산:

```
T형 회로의 ABCD 공식 (Z₁ = Z₂ = Za, 병렬 = Zb):

A = D = 1 + Za/Zb
B = 2Za + Za²/Zb
C = 1/Zb

Za = 1Ω, Zb = 2Ω 대입:

A = D = 1 + 1/2 = 3/2
B = 2×1 + 1/2 = 5/2
C = 1/2

검증: AD - BC = (3/2)(3/2) - (5/2)(1/2) = 9/4 - 5/4 = 4/4 = 1  ✓
```

## 시험 포인트

| 파라미터 | 방정식 형태 | 측정 조건 | 핵심 조건 |
|----------|-------------|-----------|-----------|
| Z 파라미터 | V = Z × I | 반대편 **개방** | 상반: Z₁₂ = Z₂₁ |
| Y 파라미터 | I = Y × V | 반대편 **단락** | 상반: Y₁₂ = Y₂₁ |
| ABCD 파라미터 | V₁=AV₂-BI₂, I₁=CV₂-DI₂ | 출력 **개방/단락** | 상반: **AD-BC=1** |
| h 파라미터 | 혼합형 | 출력 단락/입력 개방 | 트랜지스터 해석용 |

**직렬·병렬 기본 ABCD 암기표**

| 회로 형태 | A | B | C | D |
|-----------|---|---|---|---|
| 직렬 임피던스 Z | 1 | Z | 0 | 1 |
| 병렬 어드미턴스 Y | 1 | 0 | Y | 1 |

**자주 나오는 문제 유형**

1. T형·π형 회로의 ABCD 파라미터 계산
2. 종속 접속 시 전체 ABCD = [A₁B₁C₁D₁][A₂B₂C₂D₂] 행렬 곱
3. AD - BC = 1 검증 문제
4. Z 파라미터에서 출력 개방 임피던스 계산 (I₂=0 조건 대입)
5. h 파라미터에서 h₂₁ = 전류 이득 해석
