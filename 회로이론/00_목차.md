## 핵심 공식 모음 (빠른 참고용)

### 직류 기본

```
V = IR                    옴의 법칙
P = VI = I²R = V²/R      전력
I = Q/t                   전류 = 전하/시간
R_직렬 = R1 + R2          직렬 합성저항
R_병렬 = R1·R2/(R1+R2)   병렬 합성저항 (2개)
```

### 교류 기본

```
Vrms = Vm / √2           실효값
ω = 2πf                  각주파수
XL = ωL                  유도 리액턴스
XC = 1/(ωC)              용량 리액턴스
Z = R + j(XL - XC)       임피던스
f₀ = 1/(2π√LC)          공진 주파수
```

### 전력

```
P = VIcosθ    유효전력 [W]
Q = VIsinθ    무효전력 [Var]
S = VI        피상전력 [VA]
cosθ = P/S = R/Z   역률
```

### 3상

```
Y결선: VL = √3·VP,  IL = IP
Δ결선: VL = VP,    IL = √3·IP
P = √3·VL·IL·cosθ
```
