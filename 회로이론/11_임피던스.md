# 11. 임피던스 (Impedance)

## 핵심 개념

> 교류 회로에서 **저항의 역할을 하는 것**  
> R, L, C를 모두 포함한 전류의 방해 정도

직류에서 저항(R)만 있었다면,  
교류에서는 R + L + C를 합친 **임피던스(Z)** 를 씁니다.

```
V = Z × I   (교류판 옴의 법칙)
```

---

## 임피던스의 구성

```
Z = R + jX

- R: 저항 성분 (실수부)
- X: 리액턴스 성분 (허수부)
  - X > 0: 유도성 (인덕터 성분이 큼)
  - X < 0: 용량성 (캐패시터 성분이 큼)
```

---

## 각 소자의 임피던스

| 소자 | 임피던스 | 크기 |
|------|---------|------|
| 저항 R | Z = R | \|Z\| = R |
| 인덕터 L | Z = jXL = jωL | \|Z\| = ωL |
| 캐패시터 C | Z = -jXC = -j/ωC | \|Z\| = 1/ωC |

---

## 임피던스의 크기와 위상

```
Z = R + jX

크기: |Z| = √(R² + X²)
위상각: θ = arctan(X/R)
```

### 벡터로 이해하기

```
     허수축 (j)
      |
      |  /Z = R + jX
  jX  | /
      |/θ
------+------→ 실수축 (R)
```

---

## RLC 직렬 임피던스

```
Z = R + j(XL - XC) = R + j(ωL - 1/ωC)

|Z| = √(R² + (XL - XC)²)
θ = arctan((XL - XC)/R)
```

| XL > XC | 유도성 (θ > 0) | 전류가 전압보다 뒤짐 |
|---------|--------------|-----------------|
| XL < XC | 용량성 (θ < 0) | 전류가 전압보다 앞섬 |
| XL = XC | 공진 (θ = 0) | 전류와 전압 동위상 |

---

## 예제

**R=3Ω, L: XL=8Ω, C: XC=4Ω 직렬 연결, V=100V**

```
Z = R + j(XL - XC)
  = 3 + j(8 - 4)
  = 3 + j4  [Ω]

|Z| = √(3² + 4²) = √(9+16) = √25 = 5 [Ω]

I = V/|Z| = 100/5 = 20 [A]

θ = arctan(4/3) ≈ 53.1° (유도성)
```

---

## 어드미턴스 (Admittance, Y)

임피던스의 역수 → 병렬 회로 분석에 편리

```
Y = 1/Z = G + jB

- G (컨덕턴스) = 1/R
- B (서셉턴스) = 1/X
```

---

## 시험 포인트

| 공식 | 내용 |
|------|------|
| Z = R + jX | 임피던스의 기본 형태 |
| \|Z\| = √(R²+X²) | 임피던스의 크기 |
| V = ZI | 교류 옴의 법칙 |
| XL = ωL | 유도 리액턴스 |
| XC = 1/ωC | 용량 리액턴스 |

- 직렬: 임피던스를 더함
- 병렬: 어드미턴스를 더하거나 역수 공식 사용
- 3-4-5, 5-12-13 같은 피타고라스 정수 자주 출제됨
