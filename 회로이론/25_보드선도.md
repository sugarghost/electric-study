# 25. 보드선도 (Bode Plot)

## 핵심 개념

> 주파수 응답(이득과 위상)을 **로그 스케일**로 그린 그래프

**비유:** 음악의 음계(도레미파솔)도 로그 스케일입니다. 주파수가 2배가 될 때마다 1옥타브가 올라가죠.  
보드선도도 마찬가지로, 넓은 주파수 범위를 다루기 위해 로그 스케일을 사용합니다.

---

## 보드선도 구성

**2개의 그래프로 이루어집니다:**

```
크기선도 (Magnitude Plot): 이득 |G(jω)| [dB]  vs  주파수 ω [rad/s]
                           (세로축 dB, 가로축 로그 스케일)

위상선도 (Phase Plot):     위상각 ∠G(jω) [°]   vs  주파수 ω [rad/s]
                           (세로축 도(°), 가로축 로그 스케일)
```

---

## dB(데시벨) 변환

이득을 dB로 표현하는 방법:

$$|G|_{dB} = 20\log_{10}|G(j\omega)|$$

자주 나오는 변환값 암기:

```
0 dB   → 크기 1배 (변화 없음)
20 dB  → 크기 10배
40 dB  → 크기 100배
60 dB  → 크기 1000배
-20 dB → 크기 0.1배 (1/10)
-40 dB → 크기 0.01배 (1/100)
```

> **20 dB = 10배** 를 기준으로 계산하면 편합니다.

---

## 기본 요소별 보드선도 (점근선 근사)

### 1. 이득 K (상수)

```
크기 = 20 log₁₀K  [dB]  (모든 주파수에서 수평선)
위상 = 0°  (K > 0) 또는 -180°  (K < 0)
```

### 2. 적분기 1/s

```
크기: -20 dB/decade  (기울기 일정하게 감소)
위상: -90° (일정)
```

> decade = 주파수가 10배 변하는 구간

### 3. 미분기 s

```
크기: +20 dB/decade  (기울기 일정하게 증가)
위상: +90° (일정)
```

### 4. 1차 지연요소 1/(1+jωτ)

```
절점 주파수(꺾이는 점): ω = 1/τ

ω << 1/τ 일 때: 크기 ≈ 0 dB, 위상 ≈ 0°
ω = 1/τ  일 때: 크기 ≈ -3 dB, 위상 = -45°
ω >> 1/τ 일 때: 크기는 -20 dB/decade로 감소, 위상 → -90°
```

### 5. 2차 요소

```
크기: 절점 주파수(ω = ωn) 이후 -40 dB/decade 감소
위상: 0° → -180° 변화
```

## 안정성 여유 (Stability Margins)

시스템이 얼마나 안정한지를 나타내는 여유 지표입니다.

### 이득여유 (Gain Margin, GM)

위상이 $-180°$가 되는 주파수(위상 교차 주파수 $\omega_{pc}$)에서의 이득의 역수

$$GM = -|G(j\omega_{pc})|_{dB} \quad [\text{dB}]$$

$GM > 0$ (양수) 이면 안정 → 보통 6 dB 이상 필요

**의미:** 이득을 GM만큼 더 높여도 안정을 유지할 수 있는 여유

### 위상여유 (Phase Margin, PM)

이득이 1 (0 dB)이 되는 주파수(이득 교차 주파수 $\omega_{gc}$)에서 실제 위상각과 $-180°$ 사이의 차이

$$PM = \angle G(j\omega_{gc}) - (-180°) = \angle G(j\omega_{gc}) + 180° \quad [°]$$

$PM > 0$ (양수) 이면 안정 → 보통 $30° \sim 60°$ 권장

**의미:** 위상이 PM만큼 더 뒤처져도 안정을 유지할 수 있는 여유

### 안정성 판별 요약

```
GM > 0  AND  PM > 0  →  안정
GM < 0  OR   PM < 0  →  불안정
```

## 예제

$G(s) = \dfrac{10}{s(1 + 0.1s)}$ 의 이득여유와 위상여유 개략 계산

### 풀이: jω 대입

$$G(j\omega) = \frac{10}{j\omega(1 + j0.1\omega)}$$

**① 위상 교차 주파수 찾기 (위상 = -180°)**

$$\angle G(j\omega) = 0° - 90° - \arctan(0.1\omega) = -90° - \arctan(0.1\omega) = -180°$$

따라서: $\arctan(0.1\omega) = 90°\ \Rightarrow\ 0.1\omega \to \infty\ \Rightarrow\ \omega_{pc} \to \infty$

이 시스템은 위상이 $-180°$에 도달하지 않으므로 $GM = \infty$ (이론적으로 매우 안정)

**② 이득 교차 주파수 찾기 (|G(jω)| = 1)**

$$|G(j\omega)| = \frac{10}{\omega \cdot \sqrt{1 + (0.1\omega)^2}} = 1$$

$\omega \approx 10\ \text{rad/s}$ 근방 (수치 계산 필요)

$\omega = 10$ 대입:

$$|G(j10)| = \frac{10}{10 \times \sqrt{1+1}} = \frac{10}{10\sqrt{2}} = \frac{1}{\sqrt{2}} \approx 0.707$$

$\Rightarrow \omega_{gc} \approx 10$보다 조금 작음 (약 9 rad/s 근방)

**③ 위상여유 계산 (ω_gc ≈ 10 근사 사용)**

$$\angle G(j10) = -90° - \arctan(0.1 \times 10) = -90° - \arctan(1) = -90° - 45° = -135°$$

$$PM = -135° - (-180°) = 45°$$

위상여유 $PM \approx 45°$ (안정, 양호한 값)

## 시험 포인트

| 항목 | 내용 |
|------|------|
| dB 변환 | 20 log₁₀|G| (20 dB = 10배 암기) |
| 1차 지연요소 절점 주파수 | ω = 1/τ |
| 1차 요소 기울기 | -20 dB/decade |
| 2차 요소 기울기 | -40 dB/decade |
| 적분기 1/s 위상 | 항상 -90° |
| 이득여유 GM | 위상 = -180°일 때 이득의 역수 (dB) |
| 위상여유 PM | 이득 = 0 dB일 때, 실제위상 + 180° |
| 안정 조건 | GM > 0 dB AND PM > 0° |
| 권장 GM | 6 dB 이상 |
| 권장 PM | 30° ~ 60° |

- "위상여유를 구하라" → 먼저 이득이 0 dB 되는 주파수 찾기
- "이득여유를 구하라" → 먼저 위상이 -180° 되는 주파수 찾기
- 점근선 근사로 손으로 대략 그릴 수 있어야 함
