# 03. 키르히호프 법칙 (Kirchhoff's Laws)

## 왜 필요한가?

옴의 법칙만으로는 **여러 개의 전원과 저항이 복잡하게 연결된 회로**를 풀 수 없습니다.  
키르히호프 법칙은 복잡한 회로 분석의 **핵심 도구**입니다.

법칙은 두 가지입니다:
- **KCL**: 전류에 관한 법칙
- **KVL**: 전압에 관한 법칙

---

## KCL - 전류 법칙 (Kirchhoff's Current Law)

### 핵심 개념

> **어떤 접속점(노드)에 들어오는 전류의 합 = 나가는 전류의 합**

### 비유

교차로에서 차가 들어오는 수와 나가는 수는 같습니다.  
차가 교차로 안에 쌓이거나 사라질 수 없듯이, 전류도 마찬가지입니다.

```
     I1 →
          ●  → I3
     I2 →

KCL: I1 + I2 = I3
```

### 공식

```
Σ(들어오는 전류) = Σ(나가는 전류)
또는
Σ I = 0  (들어오는 것을 +, 나가는 것을 -로 설정)
```

---

## KVL - 전압 법칙 (Kirchhoff's Voltage Law)

### 핵심 개념

> **하나의 폐회로(루프)를 한 바퀴 돌면 전압의 합은 0**

### 비유

산을 올라갔다가 내려오면 결국 원래 높이로 돌아옵니다.  
전압도 한 바퀴를 돌면 올라간 만큼 내려와서 0이 됩니다.

```
    +  V1  -       +  V2  -
+  ┌──[R1]──┬──[R2]──┐  +
   │        │        │
  [E]      [R3]      │
   │        │        │
-  └────────┴────────┘  -
```

전원에서 얻은 전압 = 저항에서 소모한 전압의 합

### 공식

```
Σ(전압 상승) = Σ(전압 강하)
또는
Σ V = 0  (루프를 한 방향으로 돌면서 상승은 +, 강하는 -)
```

---

## 실전 풀이 순서

1. **노드(접속점) 표시** → KCL 적용
2. **루프(폐회로) 설정** → KVL 적용
3. **방정식 세우기**
4. **연립방정식 풀기**

### 예제

```
      10Ω       5Ω
+--[R1=10]--+--[R2=5]--+
|           |           |
[E=20V]    [R3=10Ω]   (-)
|           |           |
+-----+-----+-----------+
      A노드
```

KCL at 노드 A:  
```
I1 = I2 + I3
```

KVL (왼쪽 루프):  
```
20 = 10·I1 + 10·I3
```

KVL (오른쪽 루프):  
```
10·I3 = 5·I2
```

---

## 시험 포인트

| 법칙 | 적용 대상 | 핵심 |
|------|---------|------|
| KCL | 노드(접속점) | 전류의 합 = 0 |
| KVL | 루프(폐회로) | 전압의 합 = 0 |

- 루프 방향은 **시계방향이든 반시계방향이든 상관없음** (일관되게만 유지)
- 전류 방향을 가정한 후 풀었을 때 **음수가 나오면 실제 방향이 반대**라는 뜻 (틀린 게 아님)
- KCL과 KVL은 전기기사 전 과목에서 **기본 중의 기본**
