# 19. 블록선도 등가변환 (Block Diagram Reduction)

## 핵심 개념

> 복잡한 제어 시스템의 블록선도를  
> **하나의 전달함수**로 단순화하는 방법

제어공학 시험에서 가장 많이 출제되는 계산 유형입니다.

---

## 기본 연결 방식 3가지

### 1. 직렬 연결 (Cascade)

```
X → [G1] → [G2] → Y

등가: G = G1 × G2
```

> 직렬 연결 = 전달함수를 곱한다

### 2. 병렬 연결 (Parallel)

```
      +→ [G1] →+
X → [+]        [+] → Y
      +→ [G2] →+

등가: G = G1 + G2
```

> 병렬 연결 = 전달함수를 더한다

### 3. 피드백 연결 (Feedback)

```
        E          Y
R → [Σ] → [G(s)] →
     ↑-              |
     +---[H(s)]------+

등가: G_폐루프 = G(s) / (1 + G(s)H(s))
```

H(s) = 1 (단위 피드백):
```
G_폐루프 = G(s) / (1 + G(s))
```

---

## 블록선도 이동 규칙

복잡한 선도를 단순화할 때 블록의 위치를 옮길 수 있습니다.

### 가산점(Σ) 이동

| 이동 방향 | 규칙 |
|---------|------|
| 블록 앞으로 이동 | 분기 경로에 **G 추가** |
| 블록 뒤로 이동 | 분기 경로에 **1/G 추가** |

### 인출점 이동

| 이동 방향 | 규칙 |
|---------|------|
| 블록 앞으로 이동 | 분기 경로에 **1/G 추가** |
| 블록 뒤로 이동 | 분기 경로에 **G 추가** |

---

## 실전 풀이 순서

1. **가장 안쪽(내부 루프)부터** 등가 변환
2. 직렬/병렬 정리
3. 피드백 공식 적용
4. 반복하여 하나의 G(s)로 만들기

---

## 예제 1: 단순 피드백

```
R(s) → [Σ] → [G=10/(s+1)] → Y(s)
         ↑-                    |
         +------[H=1]----------+
```

풀이:
```
G_폐루프 = G / (1 + GH)
         = [10/(s+1)] / (1 + [10/(s+1)] × 1)
         = [10/(s+1)] / [(s+1+10)/(s+1)]
         = 10 / (s+11)
```

---

## 예제 2: 내부 루프가 있는 경우

```
R → [Σ] → [G1] → [Σ] → [G2] → Y
     ↑-            ↑-            |
     |              +---[H2]-----+
     +----------[H1]-------------+
```

풀이 순서:
```
Step 1: 안쪽 루프 정리
G_inner = G2 / (1 + G2·H2)

Step 2: 전체 등가
G_total = G1 × G_inner = G1·G2 / (1 + G2·H2)

Step 3: 바깥 피드백 적용
최종 = G_total / (1 + G_total·H1)
     = G1·G2 / [(1 + G2·H2) + G1·G2·H1]
```

---

## 시험 포인트

| 연결 방식 | 등가 전달함수 |
|---------|------------|
| 직렬 | G1 × G2 |
| 병렬 | G1 + G2 |
| 피드백 | G / (1 + GH) |
| 단위 피드백 | G / (1 + G) |

- **피드백 공식 G/(1+GH)** 는 반드시 암기
- 내부 루프부터 바깥으로 차례대로 정리
- 분모 계산 시 통분 실수 주의
