# 12. 공진 (Resonance)

## 핵심 개념

> XL = XC 가 되는 특별한 주파수 상태  
> 인덕터와 캐패시터의 효과가 **서로 상쇄**되는 순간

공진 상태에서는:
- 임피던스가 **최소 (직렬공진)** 또는 **최대 (병렬공진)**
- 역률 = 1 (전압과 전류가 동위상)

---

## 직렬 공진 (Series Resonance)

### 회로

```
+--[R]--[L]--[C]--+
|                  |
[V]                |
|                  |
+------------------+
```

### 공진 조건

$$X_L = X_C, \quad \omega L = \frac{1}{\omega C}$$

### 공진 주파수

$$\omega_0 = \frac{1}{\sqrt{LC}} \quad [\text{rad/s}]$$

$$f_0 = \frac{1}{2\pi\sqrt{LC}} \quad [\text{Hz}]$$

### 공진 시 특성

| 항목 | 상태 |
|------|------|
| 임피던스 | Z = R (최솟값) |
| 전류 | I = V/R (최댓값) |
| 역률 | 1 (cos θ = 1) |
| VL, VC | 전원전압보다 Q배 클 수 있음 |

### Q (품질계수, Quality Factor)

공진의 날카로운 정도를 나타내는 값

$$Q = \frac{\omega_0 L}{R} = \frac{1}{\omega_0 CR} = \frac{1}{R}\sqrt{\frac{L}{C}}$$

> Q가 클수록 공진 특성이 날카로움 (선택도 높음)

### 위험성

VL = VC = Q × V 가 될 수 있음  
→ Q가 크면 **전원전압보다 훨씬 높은 전압**이 L, C에 걸릴 수 있음

---

## 병렬 공진 (Parallel Resonance)

### 회로

```
     +--[L]--+
     |       |
[V]--+--[C]--+
     |       |
     +--[R]--+
```

### 공진 시 특성

| 항목 | 상태 |
|------|------|
| 임피던스 | 최댓값 (∞에 가까움) |
| 전류 (선로) | 최솟값 |
| 역률 | 1 |
| L, C 내부 전류 | 매우 클 수 있음 |

직렬공진과 **반대**의 성질을 가집니다.

---

## 직렬 vs 병렬 공진 비교

| 구분 | 직렬 공진 | 병렬 공진 |
|------|---------|---------|
| 임피던스 | 최소 (= R) | 최대 |
| 전류 | 최대 | 최소 |
| 용도 | 원하는 주파수 선택 (수신기) | 원하는 주파수 차단 (필터) |

---

## 공진 주파수 f₀ 암기

$$f_0 = \frac{1}{2\pi\sqrt{LC}}$$

숫자 예:
- L = 1mH, C = 1μF 이면
- f₀ = 1/(2π × √(10⁻³ × 10⁻⁶)) = 1/(2π × 10⁻⁴·⁵) ≈ 5033 Hz

---

## 시험 포인트

| 공식 | 내용 |
|------|------|
| f₀ = 1/(2π√LC) | 공진 주파수 (필수 암기) |
| 공진 시 Z = R | 직렬 공진 임피던스 |
| Q = ω₀L/R | 품질계수 |
| VL = VC = QV | 공진 시 소자 전압 |

- 공진 주파수 공식은 **반드시** 암기
- Q와 대역폭(BW): $BW = \dfrac{f_0}{Q}$ (Q가 크면 대역폭이 좁음)
- 시험에서 "공진 시 전류를 구하라" → Z=R이므로 I=V/R
