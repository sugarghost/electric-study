# 09. 페이저 (Phasor)

## 왜 페이저를 쓰나?

교류 계산을 sin, cos 함수로 하면 **미적분이 필요**해서 매우 복잡합니다.  
페이저를 쓰면 교류를 **복소수**로 바꿔서 대수 계산만으로 풀 수 있습니다.

### 직관적으로 이해하기

교류 전압은 크기와 방향이 계속 변하는 화살표처럼 생각할 수 있습니다.

```
시간에 따라 회전하는 화살표 → 어느 순간의 "스냅샷" = 페이저
```

- 화살표의 **길이** = 전압의 크기 (실효값)
- 화살표의 **각도** = 위상 (다른 신호와의 시간 차이)

두 교류 신호를 더하거나 나누는 것 = 두 화살표를 합치거나 비교하는 것  
→ 이것을 수학적으로 표현한 것이 **복소수**입니다.

> sin 함수 계산 → 복소수 계산으로 단순화

---

## 복소수 기본 (잠깐 복습)

```
복소수: a + jb
- a: 실수부 (Real Part)
- b: 허수부 (Imaginary Part)
- j: 허수단위 (j² = -1)
  ※ 수학에서는 i를 쓰지만 전기에서는 전류(i)와 혼동을 피해 j를 사용
```

### 복소수의 두 가지 표현

```
직각좌표형: a + jb
극좌표형:   V∠θ  또는  V·e^(jθ)
```

### 변환 공식

```
크기: |V| = √(a² + b²)
위상: θ = arctan(b/a)

a = |V| × cosθ
b = |V| × sinθ
```

---

## 페이저 표현

교류 전압 `v(t) = Vm × sin(ωt + θ)` 를 페이저로:

```
V = (Vm/√2) ∠ θ = Vrms ∠ θ
```

> 크기는 실효값, 각도는 위상각

### 예시

```
v(t) = 100√2 · sin(ωt + 30°)  [V]

페이저: V = 100∠30°  [V]
        (크기 = 100√2/√2 = 100V 실효값)
```

---

## 페이저 연산

### 덧셈/뺄셈 → 직각좌표형으로

```
V1 = 3 + j4
V2 = 1 - j2
V1 + V2 = 4 + j2
```

### 곱셈/나눗셈 → 극좌표형으로

```
V1 = 5∠30°
V2 = 2∠60°

V1 × V2 = 10∠90°   (크기는 곱, 각도는 합)
V1 / V2 = 2.5∠-30° (크기는 나누기, 각도는 차)
```

---

## j 연산자의 의미

j를 곱하면 **위상이 90° 앞으로** 이동합니다.

```
j = 1∠90°
j² = 1∠180° = -1
j³ = 1∠270° = -j
j⁴ = 1∠360° = 1
```

> 전기에서 j는 단순한 허수가 아니라 "90° 회전"의 의미

---

## 임피던스에서의 페이저 표현

| 소자 | 임피던스 |
|------|---------|
| 저항 R | R (실수, 위상 변화 없음) |
| 인덕터 L | jωL (허수, 전류가 전압보다 90° 뒤짐) |
| 캐패시터 C | 1/jωC = -j/ωC (허수, 전류가 전압보다 90° 앞섬) |

---

## 시험 포인트

| 항목 | 내용 |
|------|------|
| 페이저 크기 | 실효값 사용 |
| 곱셈 | 크기는 곱, 각도는 합 |
| 나눗셈 | 크기는 나누기, 각도는 차 |
| j의 의미 | 90° 위상 앞으로 이동 |

- 페이저는 **교류 회로 전체**에서 사용되는 핵심 도구
- 복소수 계산이 익숙해질 때까지 계산 연습이 필요
- 극좌표 ↔ 직각좌표 변환을 빠르게 할 수 있어야 함
