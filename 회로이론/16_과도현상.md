# 16. 과도현상 (Transient Response)

## 핵심 개념

> 스위치를 켜거나 끌 때, 회로가 **새로운 정상 상태로 넘어가는 과정**

직류 회로에서 스위치를 닫으면 전류가 순간적으로 최대가 될 것 같지만,  
**L(인덕터)과 C(캐패시터)** 때문에 서서히 변합니다. 이 과도기를 과도현상이라 합니다.

---

## 왜 생기나?

| 소자 | 이유 |
|------|------|
| 인덕터 L | 전류를 갑자기 바꿀 수 없음 (관성) |
| 캐패시터 C | 전압을 갑자기 바꿀 수 없음 (충전 시간 필요) |

---

## RC 회로의 과도현상

### 충전 (스위치 ON)

```
+--[R]--+--[C]--+
|       |       |
[E]     |       |
|       |       |
+-------+-------+
```

스위치를 닫으면 C가 서서히 충전됩니다.

#### 전압 방정식

```
v_C(t) = E × (1 - e^(-t/τ))   [V]
i(t)   = (E/R) × e^(-t/τ)      [A]
```

#### 시정수 τ (타우)

```
τ = RC   [초]
```

| 경과 시간 | 충전 정도 |
|---------|---------|
| t = τ   | 63.2% 충전 |
| t = 2τ  | 86.5% 충전 |
| t = 3τ  | 95.0% 충전 |
| t = 5τ  | 99.3% 충전 → 정상상태 도달 |

> **5τ 이후에는 완전히 충전된 것으로 간주**합니다.

### 방전 (충전 후 스위치 OFF)

```
v_C(t) = E × e^(-t/τ)   [V]
i(t)   = -(E/R) × e^(-t/τ)
```

지수함수로 감소합니다.

---

## RL 회로의 과도현상

### 전류 증가 (스위치 ON)

```
+--[R]--[L]--+
|            |
[E]          |
|            |
+------------+
```

#### 전류 방정식

```
i(t) = (E/R) × (1 - e^(-t/τ))   [A]
v_L(t) = E × e^(-t/τ)            [V]
```

#### 시정수 τ

```
τ = L/R   [초]
```

> RC와 달리 **L/R** 임에 주의

| 경과 시간 | 전류 도달 |
|---------|---------|
| t = τ   | 최대의 63.2% |
| t = 5τ  | 정상상태 (E/R) 도달 |

### 정상상태에서

- L에 걸리는 전압: 0V (저항 역할 사라짐)
- 전류: E/R (최대)

---

## RC vs RL 비교

| 구분 | RC 회로 | RL 회로 |
|------|--------|--------|
| 시정수 | τ = RC | τ = L/R |
| 갑자기 못 바꾸는 것 | 캐패시터 전압 | 인덕터 전류 |
| 충전 완료 후 | C = 개방 (전류 0) | L = 단락 (전압 0) |

---

## 초기값 · 최종값 정리

### 초기값 (t = 0+, 스위치 직후)

- **C의 전압**: 스위치 전과 동일 (못 바뀜)
- **L의 전류**: 스위치 전과 동일 (못 바뀜)

### 최종값 (t → ∞, 정상상태)

- **C**: 개방회로처럼 (전류 = 0)
- **L**: 단락회로처럼 (전압 = 0)

---

## 예제

**R=1kΩ, C=1μF, E=10V, t=0에서 스위치 ON**

```
τ = RC = 1000 × 10⁻⁶ = 10⁻³ = 1 [ms]

t = 1ms 후 전압:
v_C = 10 × (1 - e^(-1)) = 10 × 0.632 = 6.32 [V]

t = 5ms 후 (≈ 정상상태):
v_C ≈ 10 [V]
```

---

## 시험 포인트

| 공식 | 내용 |
|------|------|
| τ = RC | RC 시정수 |
| τ = L/R | RL 시정수 |
| 63.2% | t = τ 일 때 충전/도달률 |
| 5τ | 정상상태 도달 기준 |
| 초기: C전압 불변, L전류 불변 | 스위치 직후 |
| 최종: C=개방, L=단락 | 정상상태 |

- 시정수 τ = RC와 τ = L/R 두 개를 혼동하지 말 것
- 과도현상 문제는 "t=τ일 때 값" 또는 "시정수를 구하라"가 전형적 패턴
