# 08. 교류(AC) 기본 개념

## 직류 vs 교류

| 구분 | 직류 (DC) | 교류 (AC) |
|------|---------|---------|
| 전압 방향 | 항상 일정 | 주기적으로 바뀜 |
| 예 | 건전지, 배터리 | 가정용 콘센트 (220V) |
| 표현 | 단순한 값 | 진폭, 주파수, 위상 필요 |

---

## 교류의 기본 파형: 사인파 (Sine Wave)

```
v(t) = Vm × sin(ωt + φ)
```

| 기호 | 이름 | 의미 |
|------|------|------|
| Vm | 최대값 (진폭) | 파형의 최고점 |
| ω | 각주파수 | 회전 빠르기 [rad/s] |
| t | 시간 | [s] |
| φ (파이) | 초기 위상 | 시작 위치 |

---

## 주파수와 주기

### 주파수 (Frequency, f)

- 1초에 몇 번 반복되는가
- 단위: 헤르츠 [Hz]
- 한국 상용전원: **60Hz** (1초에 60번 진동)

### 주기 (Period, T)

- 한 번 반복하는 데 걸리는 시간
- 단위: 초 [s]

### 관계식

```
f = 1/T        T = 1/f
```

### 각주파수 (Angular Frequency, ω)

```
ω = 2πf = 2π/T   [rad/s]
```

60Hz의 경우: ω = 2π × 60 ≈ 377 rad/s

---

## 교류의 크기 표현 3가지

| 종류 | 기호 | 설명 |
|------|------|------|
| 최대값 | Vm, Im | 파형의 피크값 |
| 실효값 | V, I (또는 Vrms) | 직류와 동등한 효과를 내는 값 |
| 평균값 | Vav | 반주기 평균 |

### 사인파에서의 관계

```
실효값 = 최대값 / √2 ≈ 최대값 × 0.707

Vrms = Vm / √2
Irms = Im / √2
```

> 가정용 220V는 **실효값**입니다.  
> 최대값은 220 × √2 ≈ 311V

### 평균값

```
Vav = (2/π) × Vm ≈ 0.637 × Vm
```

---

## 위상 (Phase)

두 교류 신호의 **시간적인 차이**

```
v1(t) = Vm × sin(ωt)         ← 기준
v2(t) = Vm × sin(ωt + 90°)  ← v1보다 90° 앞섬 (진상)
v3(t) = Vm × sin(ωt - 90°)  ← v1보다 90° 뒤짐 (지상)
```

- **진상 (Leading)**: 앞서는 것
- **지상 (Lagging)**: 뒤처지는 것

---

## 시험 포인트

| 공식 | 내용 |
|------|------|
| ω = 2πf | 각주파수 |
| T = 1/f | 주기 |
| Vrms = Vm/√2 | 실효값 ↔ 최대값 |
| f = 60Hz | 한국 상용주파수 |

- **실효값**이 일반적으로 말하는 교류 전압/전류 (220V, 10A 등)
- 전력 계산은 실효값으로
- 최대값은 절연 설계, 피크 전압 계산에 사용
