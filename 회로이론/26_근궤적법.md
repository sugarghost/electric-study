# 26. 근궤적법 (Root Locus Method)

## 핵심 개념

> 이득 K가 0에서 ∞로 변할 때, 특성방정식의 근(극점)이 s 평면에서 이동하는 궤적

**비유:** 스피커 볼륨(이득 K)을 올릴수록 소리가 커지다가, 어느 순간 하울링(불안정)이 발생합니다.  
근궤적은 K를 키웠을 때 시스템이 언제 불안정해지는지를 시각적으로 보여줍니다.

---

## 기본 개념

### 특성방정식

$$1 + K \cdot G(s) \cdot H(s) = 0$$

K가 변하면 이 방정식의 근(시스템 극점)의 위치가 변합니다.

### 출발점과 도착점

```
K = 0 일 때:   근이 개루프 극점에서 출발
K → ∞ 일 때:  근이 개루프 영점으로 도착 (영점보다 극점이 많으면 나머지는 무한대로)
```

> 극점(Pole): G(s)H(s) 분모를 0으로 만드는 s 값  
> 영점(Zero): G(s)H(s) 분자를 0으로 만드는 s 값

---

## 근궤적 작도 기본 법칙

### 법칙 1: 가지(Branch) 수

```
가지 수 = 개루프 극점의 수 n
```

각 극점에서 하나의 궤적이 출발합니다.

### 법칙 2: 실수축 위의 근궤적

```
어떤 실수축 위의 점에서, 그 점의 오른쪽에 있는
실수 극점과 실수 영점의 합이 홀수 → 근궤적 구간
```

**예:** 극점이 0, -2에 있을 때
- 0과 -2 사이: 오른쪽 극점 수 = 1 (홀수) → 근궤적 있음
- -2보다 왼쪽: 오른쪽 극점 수 = 2 (짝수) → 근궤적 없음

### 법칙 3: 점근선 각도

극점이 영점보다 많을 때, 남은 궤적은 무한대로 뻗어 나갑니다.

$$\theta_k = \frac{(2k+1) \times 180°}{n - m}, \quad k = 0, 1, 2, \ldots$$

- $n$: 개루프 극점의 수
- $m$: 개루프 영점의 수

**예:** 극점 3개, 영점 0개 ($n=3,\ m=0$)

$$\theta_0 = \frac{1 \times 180°}{3} = 60°, \quad \theta_1 = \frac{3 \times 180°}{3} = 180°, \quad \theta_2 = \frac{5 \times 180°}{3} = 300°\ (= -60°)$$

### 법칙 4: 점근선 교점 σ (원점)

$$\sigma_a = \frac{\sum p_i - \sum z_i}{n - m}$$

**예:** 극점이 $0,\ -2,\ -4$이고 영점이 없을 때

$$\sigma_a = \frac{0 + (-2) + (-4) - 0}{3 - 0} = \frac{-6}{3} = -2$$

### 법칙 5: 실수축 이탈점 (Break-away Point)

근궤적이 실수축을 떠나는 점:

$\dfrac{dK}{ds} = 0$ 을 풀어서 $s$ 값을 구함

또는: $\dfrac{d}{ds}[G(s)H(s)] = 0$ 의 근 중 실수축 근궤적 구간에 있는 값

### 법칙 6: 허수축 교차점 (안정 한계)

```
루스 판별법 이용:
특성방정식 계수로 루스 배열 작성
→ 첫 번째 열에 0이 생기는 K값 = 안정 한계 이득
→ 그때 허수축 교차 주파수 계산
```

---

## 안정 범위 판별

```
근궤적이 s 평면의 좌반면(실수부 < 0)에 있는 K 범위 → 안정
근궤적이 우반면으로 넘어가는 순간의 K → 안정 한계 이득 Kc
```

---

## 예제

**G(s) = K / [s(s+2)],  H(s) = 1 (단위 피드백)**

### ① 기본 정보

```
개루프 극점: s = 0, s = -2  (n = 2)
개루프 영점: 없음  (m = 0)
가지 수: 2
```

### ② 출발점

```
K = 0 일 때: s = 0 과 s = -2 에서 출발
```

### ③ 실수축 근궤적 구간

```
-2 < s < 0 구간: 오른쪽 실수 극점 수 = 1 (홀수) → 근궤적 있음
s < -2 구간: 오른쪽 극점 수 = 2 (짝수) → 없음
```

### ④ 점근선

$n - m = 2 - 0 = 2$

$$\theta_0 = \frac{1 \times 180°}{2} = 90°, \quad \theta_1 = \frac{3 \times 180°}{2} = 270°\ (= -90°)$$

점근선 방향: 위, 아래 ($\pm 90°$, 즉 허수축 방향)

### ⑤ 점근선 교점

$$\sigma_a = \frac{0 + (-2) - 0}{2 - 0} = \frac{-2}{2} = -1$$

실수축 $s = -1$ 에서 $\pm 90°$ 방향으로 뻗어남

### ⑥ 실수축 이탈점

특성방정식: $s(s+2) + K = 0\ \Rightarrow\ K = -s(s+2) = -s^2 - 2s$

$$\frac{dK}{ds} = -2s - 2 = 0 \quad \Rightarrow \quad s = -1 \quad \text{(이탈점)}$$

$s = -1$은 $-2 < s < 0$ 범위에 있으므로 유효

### ⑦ 허수축 교차점 (루스 판별법)

```
특성방정식: s² + 2s + K = 0

루스 배열:
s² | 1    K
s¹ | 2    0
s⁰ | K

안정 조건: 첫 번째 열 모두 양수 → K > 0

허수축 교차: s¹ 행 = 0 이 되는 경우 없음
→ 이 시스템은 K > 0 이면 항상 안정

(s¹ 행의 2는 K에 무관하므로, K → ∞ 까지도 안정)
```

### 근궤적 요약

| $K$ | 극점 위치 |
|-----|----------|
| $K = 0$ | $s = 0,\ s = -2$ (실수축) |
| $K$ 증가 | 두 극점이 $s = -1$ 을 향해 이동 |
| $K = 1$ | $s = -1$ (이탈점, 두 극점이 만남) |
| $K > 1$ | $s = -1 \pm j\sqrt{K-1}$ (허수부 생김, 진동 응답) |

## 시험 포인트

| 항목 | 공식/방법 | 비고 |
|------|-----------|------|
| 가지 수 | 극점의 수 n | 극점 = 분모의 근 |
| 출발점 | 개루프 극점 (K=0) | |
| 도착점 | 개루프 영점 (K→∞) | |
| 점근선 각도 | (2k+1)×180°/(n-m) | k = 0,1,2,... |
| 점근선 교점 | (Σ극점 - Σ영점)/(n-m) | |
| 이탈점 | dK/ds = 0 | 실수축 근궤적 구간 내 값 |
| 허수축 교차 | 루스 판별법 적용 | 안정 한계 이득 Kc |
| 안정 조건 | 근궤적이 좌반면에 있는 K 범위 | |

- **"이득 K의 범위를 구하라"** → 루스 판별법으로 안정 범위 찾기
- **"허수축 교차점을 구하라"** → 루스 배열에서 s¹ 행 = 0 으로 K 계산
- 점근선 각도와 교점은 반드시 계산할 수 있어야 함
- 이탈점은 dK/ds = 0 으로 구하되, 반드시 실수축 근궤적 구간 확인
