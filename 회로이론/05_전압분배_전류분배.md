# 05. 전압 분배 · 전류 분배 법칙

## 왜 필요한가?

매번 합성저항 → 전체전류 → 각 저항전압 순서로 풀면 오래 걸립니다.  
분배 법칙을 쓰면 **한 번에 바로** 원하는 값을 구할 수 있습니다.

---

## 전압 분배 법칙 (직렬 회로)

### 상황

직렬 연결에서 **각 저항에 걸리는 전압**을 구할 때

```
+--[R1]--[R2]--+ 전체 전압 V 인가
```

### 공식

```
V1 = V × R1 / (R1 + R2)
V2 = V × R2 / (R1 + R2)
```

> 전체 전압을 저항의 비율대로 나눠 가집니다.

### 직관적 이해

저항이 클수록 더 많은 전압을 가져갑니다.  
(파이프가 좁을수록 그 부분에서 압력 손실이 더 크다)

### 예제

V=100V, R1=30Ω, R2=70Ω 직렬 연결 시 각 전압은?

```
V1 = 100 × 30/(30+70) = 100 × 0.3 = 30 [V]
V2 = 100 × 70/(30+70) = 100 × 0.7 = 70 [V]
```

확인: 30 + 70 = 100V ✓

---

## 전류 분배 법칙 (병렬 회로)

### 상황

병렬 연결에서 **각 저항에 흐르는 전류**를 구할 때

```
      +--[R1]--+
I -->-+        +--> I
      +--[R2]--+
```

### 공식 (2개 병렬)

```
I1 = I × R2 / (R1 + R2)
I2 = I × R1 / (R1 + R2)
```

> 주의: **자기 저항이 아닌 상대방 저항** 비율을 씁니다!

### 직관적 이해

저항이 작은 쪽으로 전류가 더 많이 흐릅니다.  
(좁은 파이프보다 넓은 파이프로 물이 더 많이 흐름)  
그래서 공식에 상대방 저항이 들어가는 것입니다.

### 예제

전체 전류 I=12A, R1=4Ω, R2=8Ω 병렬 시 각 전류는?

```
I1 = 12 × 8/(4+8) = 12 × 8/12 = 8 [A]
I2 = 12 × 4/(4+8) = 12 × 4/12 = 4 [A]
```

확인: 8 + 4 = 12A ✓  
R1이 더 작으므로 더 많은 전류가 흐름 ✓

---

## 한눈에 비교

| 구분 | 전압 분배 (직렬) | 전류 분배 (병렬) |
|------|----------------|----------------|
| 적용 | 직렬 회로 | 병렬 회로 |
| 공식 | V1 = V × **R1**/(R1+R2) | I1 = I × **R2**/(R1+R2) |
| 비율 | **자기 저항** 비율 | **상대방 저항** 비율 |
| 특징 | 저항 클수록 전압 많이 | 저항 작을수록 전류 많이 |

---

## 시험 포인트

- 전류 분배에서 **분자에 상대방 저항**이 오는 것 헷갈리지 말 것
- 계산 후 **합이 전체와 같은지** 검산하는 습관
- 3개 이상 병렬은 각각 V/R로 전류 구하는 것이 더 빠름
