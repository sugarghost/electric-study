# 09. 전력용 반도체와 정류기

## 핵심 개념
> 반도체 소자로 전력의 형태(AC↔DC)와 크기를 자유롭게 변환하는 전력 전자 기술

---

## 1. 주요 전력용 반도체 소자

| 소자 | 영문/별칭 | 턴온 방법 | 턴오프 방법 | 속도 | 용도 |
|------|---------|---------|-----------|------|------|
| 다이오드 | Diode | 순방향 전압 | 역방향 전압 | 빠름 | 정류기 |
| 사이리스터 | SCR (Silicon Controlled Rectifier) | 게이트 신호 | 자연 전류 영점 | 느림 | 위상 제어 |
| GTO | Gate Turn-Off Thyristor | 게이트(+) | 게이트(-) | 중간 | 대용량 인버터 |
| IGBT | Insulated Gate Bipolar Transistor | 게이트 전압 | 게이트 전압 OFF | 빠름 | 인버터, 컨버터 |
| MOSFET | Metal-Oxide-Semiconductor FET | 게이트 전압 | 게이트 전압 OFF | 매우 빠름 | 소용량 스위칭 전원 |

**비교 정리:**
```
제어 능력:
다이오드 < SCR < GTO < IGBT ≤ MOSFET

전력 용량:
MOSFET < IGBT < GTO < SCR

스위칭 속도:
SCR/GTO < IGBT < MOSFET
```

---

## 2. 사이리스터(SCR) 상세

### 구조와 동작

SCR은 PNPN 4층 구조의 반도체로, 게이트 신호로 도통을 제어합니다.

```
동작 조건:
턴온: 1) 순방향 전압 + 2) 게이트 펄스 신호
턴오프: 자연 턴오프 — 애노드 전류가 유지전류(IH) 이하로 감소 시
또는: 역방향 전압 인가

특징:
- 한 번 게이트로 켜면 게이트 신호 없어도 계속 도통 상태 유지
- 스스로 끄는 것은 불가 → AC 회로에서는 전류 영점 통과 시 자동 소호
```

**비유:** 수도꼭지처럼 — 한 번 열면(게이트 신호) 물이 계속 흐르고, 물이 저절로 멈출 때(전류 영점)까지 닫히지 않음

### 사이리스터 정격

```
전압 정격: 최대 역방향 전압 (PIV: Peak Inverse Voltage)
전류 정격: 평균 전류, RMS 전류

직렬 연결: 전압 분담 (고전압 대응)
병렬 연결: 전류 분담 (대전류 대응)
```

## 3. 단상 반파 정류 회로

가장 간단한 정류 회로입니다.

```
다이오드 1개 사용
교류의 양(+)의 반파만 통과, 음(-)의 반파는 차단

직류 평균 전압:
Vdc = Vm / π ≈ 0.318 × Vm

Vm : 교류 피크(최대) 전압
Vm = √2 × Vrms (Vrms: 실효값)

예: 220V AC → Vm = √2 × 220 ≈ 311V
→ Vdc = 311/π ≈ 99V
```

---

## 4. 단상 전파 정류 회로 (브리지)

가장 일반적인 정류 회로입니다.

```
다이오드 4개 브리지 연결
양(+), 음(-)의 반파 모두 같은 방향으로 정류

직류 평균 전압:
Vdc = 2Vm / π ≈ 0.636 × Vm

반파보다 2배 높은 직류 전압 획득
맥동(리플) 주파수: 입력 주파수의 2배 (60Hz → 120Hz)

예: 220V AC → Vm ≈ 311V
→ Vdc = 2×311/π ≈ 198V
```

---

## 5. 3상 전파 정류 회로

산업용에서 가장 많이 쓰이는 회로입니다.

```
다이오드 6개 (또는 사이리스터 6개)
3상 교류를 직류로 변환

직류 평균 전압:
Vdc = (3√3/π) × Vm_phase
    ≈ 2.34 × Vm_phase
    ≈ 1.35 × VL

VL     : 선간 전압 실효값
Vm_phase: 상 전압 피크값

맥동 주파수: 입력의 6배 (60Hz → 360Hz)
→ 단상보다 훨씬 매끄러운 직류 출력

예: 3상 380V (선간) → Vdc ≈ 1.35 × 380 ≈ 513V
```

## 6. SCR 위상 제어 정류 (Phase Control)

사이리스터의 게이트 점호각 α를 조절하여 출력 전압을 제어합니다.

```
단상 반파 SCR 정류 (저항 부하):
Vdc = Vm(1 + cosα) / (2π)

더 일반적으로 쓰이는 단상 전파 SCR:
Vdc = (2Vm / π) × cosα

α (제어각, 점호각):
α = 0°  → 최대 출력 (다이오드 정류와 동일)
α = 90° → 출력 절반
α = 180° → 출력 최소(≈ 0)

제어 범위: 0 ≤ α ≤ 180°
```

**활용:** 직류 전동기의 속도 제어, 조광 장치(빛 밝기 조절), 히터 온도 제어

---

## 7. 전력 변환 장치의 종류

```
컨버터 (Converter):   교류 → 직류 (AC to DC)
인버터 (Inverter):    직류 → 교류 (DC to AC)
초퍼 (Chopper):       직류 → 직류 (DC to DC, 전압 변환)
사이클로컨버터:        교류 → 교류 (주파수 변환)
```

---

## 8. 인버터 (Inverter)

직류를 교류로 바꾸는 장치로, 가변 속도 전동기 구동에 핵심입니다.

```
기본 원리:
DC 전원 → 4개 (또는 6개) 스위치(IGBT 등) → 교번 스위칭 → AC 출력

VVVF 인버터 (Variable Voltage Variable Frequency):
V(전압)와 f(주파수)를 동시에 변화
→ 유도전동기의 속도를 부드럽게 제어
→ 에너지 절약 효과 (EFC: Energy Saving)
```

---

## 9. PWM 제어 (Pulse Width Modulation)

```
원리:
고정된 전압을 스위칭 ON/OFF 비율(듀티비)로 평균 전압 조절

듀티비 D = ton / T

ton : ON 시간
T   : 스위칭 주기

평균 출력 전압:
Vavg = D × Vs

Vs : 직류 공급 전압

예: Vs=100V, D=0.6(60%)
→ Vavg = 0.6 × 100 = 60V
```

**비유:** 선풍기를 빠르게 껐다 켰다 하면 팬이 서서히 도는 것처럼, 빠른 스위칭으로 낮은 평균 전압을 만듭니다.

---

## 예제 (계산)

**문제:** 단상 220V(실효값) 교류를 전파 다이오드 브리지로 정류할 때 직류 평균 전압을 구하시오.

```
최대 전압:
Vm = √2 × 220 ≈ 1.414 × 220 ≈ 311 [V]

전파 정류 직류 전압:
Vdc = 2Vm / π = 2 × 311 / 3.14159 ≈ 198 [V]
```

**답: 약 198V**

---

## 예제 2 (SCR 위상 제어)

**문제:** 단상 전파 SCR 정류에서 Vm=311V, 제어각 α=60°일 때 직류 평균 전압을 구하시오.

```
Vdc = (2Vm / π) × cosα
    = (2 × 311 / π) × cos60°
    = 198 × 0.5
    = 99 [V]
```

**답: 99V (α=0°일 때 198V의 절반)**

## 시험 포인트

| 공식 / 개념 | 내용 |
|------------|------|
| 단상 반파 | `Vdc = Vm/π ≈ 0.318Vm` |
| 단상 전파 | `Vdc = 2Vm/π ≈ 0.636Vm` |
| 3상 전파 | `Vdc ≈ 1.35VL` |
| SCR 위상 제어 | `Vdc = (2Vm/π)cosα` |
| α=0° | 최대 출력, α=90° 절반, α=180° 최소 |
| SCR 턴오프 | 자연 소호(전류 영점), 역방향 전압 |
| IGBT | 고속 스위칭, 전압 제어, 인버터의 핵심 소자 |
| 인버터 | DC→AC 변환, VVVF로 전동기 속도 제어 |
| PWM | 듀티비로 평균 전압 제어, `Vavg = D×Vs` |

### 자주 출제되는 유형

1. 각 정류 회로의 직류 전압 계산
2. SCR 위상 제어 전압 계산 (α 값 주어질 때)
3. 전력 소자 특성 비교 (SCR·IGBT·다이오드 차이)
4. 인버터·컨버터 용도 구분
