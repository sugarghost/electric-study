# 08. 유도기 (Induction Machine)

## 핵심 개념
> 회전 자기장이 회전자에 전류를 유도하고, 그 전류와 자기장의 힘으로 회전하는 전동기

---

## 1. 동작 원리

```
3상 전원 인가
       ↓
고정자(Stator)에 회전 자기장 발생
       ↓
회전자(Rotor)가 자기장보다 느리게 회전
→ 회전자 도체가 자기장을 "가로지름"
       ↓
플레밍의 오른손 법칙으로 유도 전류 발생
       ↓
플레밍의 왼손 법칙으로 전자력(토크) 발생
       ↓
회전자가 자기장 방향으로 회전
```

**비유:** 알루미늄 원판 위에서 자석을 빠르게 돌리면 원판이 따라 도는 현상 — 이것이 유도전동기의 원리입니다.

**핵심 포인트:** 회전자는 동기 속도에 도달할 수 없습니다. 동기 속도와 같아지면 자기장의 변화가 없어 유도 전류 = 0 → 토크 = 0 → 다시 느려짐. 따라서 항상 슬립이 존재합니다.

---

## 2. 동기 속도와 슬립

### 동기 속도

```
Ns = 120f / P   [rpm]

f : 전원 주파수, P : 극수
```

### 슬립 (Slip)

동기 속도와 실제 회전 속도의 차이를 나타내는 무차원 값입니다.

```
s = (Ns - N) / Ns

N  : 실제 회전수 [rpm]
Ns : 동기 속도 [rpm]

특수 경우:
s = 0   → N = Ns (동기 속도, 현실에서 불가)
s = 1   → N = 0  (정지 상태)
s > 1   → 역방향 회전 (역전 제동 시)
s < 0   → 동기 속도 이상 (유도 발전기 동작)

정상 운전: s = 0.02 ~ 0.08 (2% ~ 8%)
```

**실제 회전수 계산:**
```
N = Ns × (1 - s)   [rpm]

예: Ns=1800rpm, s=0.05 → N = 1800×0.95 = 1710 [rpm]
```

---

## 3. 슬립 주파수

회전자에 유도되는 전류의 주파수는 슬립에 비례합니다.

```
f₂ = s × f₁   [Hz]

f₁ : 고정자 전원 주파수 (60Hz)
f₂ : 회전자 전류 주파수

정지 시 (s=1): f₂ = f₁ = 60Hz
정상 운전 (s=0.05): f₂ = 0.05 × 60 = 3Hz
```

---

## 4. 등가회로와 전력 흐름

### 전력 흐름 다이어그램

```
1차 입력 P₁
    ↓ (1차 동손, 철손 제외)
공극 전력 P₂(gap) ← 에어갭을 통해 회전자에 전달되는 전력
    ↓
2차 동손: Pcu2 = s × P₂
    ↓
기계 출력: Pm = (1-s) × P₂
    ↓ (기계손 제외)
축 출력 Pout
```

### 핵심 공식

```
공극 전력 P₂(gap):
P₂ = T × ωs   (ωs = 2πNs/60)

2차 동손:
Pcu2 = s × P₂

기계 출력:
Pm = (1-s) × P₂ = P₂ - Pcu2

토크:
T = P₂ / ωs   [N·m]

세 값의 비율:
P₂ : Pcu2 : Pm = 1 : s : (1-s)
```

**암기 팁:** "공극 전력이 1이면, 슬립은 동손, (1-슬립)은 출력"

---

## 5. 등가회로 (2차 저항 변환)

유도전동기의 2차 회로를 1차 쪽으로 등가 변환할 때 핵심이 있습니다.

```
2차 임피던스:
Z₂ = R₂/s + jX₂

R₂/s = R₂ + R₂(1-s)/s
       ↑         ↑
  2차 동손    기계적 출력에 해당하는 가상 저항

→ 2차 동손 R₂에서 열 발생
→ R₂(1-s)/s 에서 기계 출력 발생
```

---

## 6. 토크-속도 특성 곡선

```
토크
  |         최대 토크
  |            *
  |          *   *
  |        *       *
  |       *          *          기동 토크
  |     *              *      *
  |___*____________________*_______________→ 속도(N)
  0                       Ns
  (정지)                (동기속도)
  s=1                   s=0
  
  임계 슬립 sm에서 최대 토크 발생
  정상 운전 구간: sm < s < 0.08 (우측 내리막 부분)
```

### 주요 토크 구분

| 토크 | 슬립 | 설명 |
|------|------|------|
| 기동 토크 | s = 1 | 처음 기동 시 발생하는 토크 |
| 최대 토크 | s = sm (임계 슬립) | 전동기가 낼 수 있는 최대 힘 |
| 정격 토크 | s = 0.02~0.08 | 정상 운전 시 토크 |

---

## 7. 비례추이 (Proportional Shift)

권선형 유도전동기의 핵심 특성입니다.

```
최대 토크 발생 슬립: sm ∝ R₂ (2차 저항에 비례)

R₂를 n배 증가시키면:
→ 최대 토크 크기는 변하지 않음
→ 최대 토크 발생 슬립 sm이 n배로 이동

활용: R₂를 조절하여 기동 시 최대 토크를 s=1(정지점)에서 발생시킴
→ 기동 토크를 최대 토크와 같게 만들어 강력한 기동 가능
```

**비유:** 자동차 기어를 바꾸면 최고 속도는 변하지 않지만 최고 출력이 나오는 속도가 달라지는 것과 유사합니다.

---

## 8. 속도 제어 방법

| 방법 | 원리 | 특징 |
|------|------|------|
| 주파수 제어 (VVVF) | f 변화 → Ns 변화 | 정밀 제어, 인버터 필요, 가장 많이 사용 |
| 극수 변환 | P 변화 → Ns 변화 | 단계적 속도 변환 (2단계, 4단계) |
| 전압 제어 | V 조절 | 소형·간단하나 토크 변화 큼 |
| 2차 저항 제어 | R₂ 조절 (권선형) | 슬립 변화, 손실 큼 |

---

## 9. 기동 방법

유도전동기는 기동 시 전류가 정격의 5~8배 흐르므로 기동 방법이 중요합니다.

| 방법 | 원리 | 특징 |
|------|------|------|
| 전전압 기동 | 직접 전원 인가 | 간단, 소형만 가능 (기동 전류 큼) |
| Y-Δ 기동 | 기동 Y결선, 운전 Δ결선 | 기동 전류·토크 1/3 감소 |
| 기동 보상기 | 단권 변압기로 전압 낮춰 기동 | 기동 전류 감소, 비용 높음 |
| 리액터 기동 | 직렬 리액터 삽입 | 기동 전류 제한, 토크 감소 |
| 소프트 스타터 | 사이리스터로 전압 서서히 증가 | 충격 없는 기동 |
| 인버터 기동 | 주파수를 0에서 서서히 증가 | 가장 부드러운 기동, 속도 제어 병행 |

## 예제 (계산)

**문제:** 4극, 60Hz 유도전동기가 슬립 0.05로 운전 중일 때 동기 속도, 실제 회전수, 슬립 주파수를 구하시오.

```
동기 속도:
Ns = 120 × 60 / 4 = 1800 [rpm]

실제 회전수:
N = Ns × (1-s) = 1800 × (1-0.05) = 1800 × 0.95 = 1710 [rpm]

슬립 주파수:
f₂ = s × f₁ = 0.05 × 60 = 3 [Hz]
```

**답: Ns=1800rpm, N=1710rpm, f₂=3Hz**

---

## 예제 2 (전력 분배)

**문제:** 공극 전력 P₂=10kW, 슬립 s=0.05일 때 2차 동손과 기계 출력을 구하시오.

```
2차 동손:
Pcu2 = s × P₂ = 0.05 × 10000 = 500 [W]

기계 출력:
Pm = (1-s) × P₂ = 0.95 × 10000 = 9500 [W] = 9.5 [kW]

검산: Pcu2 + Pm = 500 + 9500 = 10000 W = P₂ ✓
```

**답: 동손=500W, 기계 출력=9.5kW**

## 시험 포인트

| 공식 / 개념 | 내용 |
|------------|------|
| 슬립 | `s = (Ns - N)/Ns`, 정상 운전 2~8% |
| 슬립 주파수 | `f₂ = s × f₁` |
| 전력 비율 | `P₂ : Pcu2 : Pm = 1 : s : (1-s)` |
| 2차 동손 | `Pcu2 = s × P₂` |
| 기계 출력 | `Pm = (1-s) × P₂` |
| 비례추이 | R₂ ↑ → sm 비례 이동, 최대 토크 크기 불변 |
| Y-Δ 기동 | 기동 전류·토크 모두 1/3로 감소 |
| VVVF | 인버터로 주파수 변화시켜 속도 제어 (가장 우수) |
