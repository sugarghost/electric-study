# 06. 동기전동기

## 핵심 개념
> 동기 속도로만 회전하며, 여자 전류 조절로 역률을 마음대로 바꿀 수 있는 유일한 전동기

---

## 1. 동기전동기의 특징

동기전동기는 동기발전기와 구조가 거의 같으며, 전기를 받아서 회전력을 냅니다.

| 특징 | 내용 |
|------|------|
| 속도 | 동기 속도로만 회전 (부하가 변해도 속도 불변) |
| 역률 제어 | 계자 전류 조절로 역률 1.0, 진상, 지상 모두 가능 |
| 자기 기동 | 기동 토크 없음 → 별도 기동 방법 필요 |
| 용도 | 정속 구동, 역률 개선용 |

**직관적 이해:**
```
유도전동기: 부하가 무거우면 느려짐 (슬립 증가)
동기전동기: 부하가 아무리 무거워도 속도 불변 (동기 속도 유지)
            → 대신 너무 무거우면 탈조(멈춤)
```

---

## 2. 동기 속도와 회전 원리

```
Ns = 120f / P   [rpm]   (발전기와 동일)

동기 유지 원리:
회전 자기장(동기 속도)에 회전자 자극이 끌려가며 회전
→ 마치 자석이 자석을 잡아당기듯이 동기 속도 유지
```

**비유:** 무용수가 음악 박자(동기 속도)에 맞춰 춤을 추는 것처럼, 회전자는 회전 자기장에 딱 맞춰 따라갑니다.

---

## 3. V 곡선 (V-Curve) — 동기전동기의 핵심

계자 전류(If)에 따라 전기자 전류(Ia)와 역률이 어떻게 변하는지를 나타내는 중요한 그래프입니다.

```
계자 전류 If   →  전기자 전류 Ia   역률
────────────────────────────────────────
  적음 (부족여자)  → Ia 증가     지상(유도성) ← 전류가 전압보다 뒤처짐
  적당 (정격여자)  → Ia 최소     역률 = 1.0  ← 가장 효율적인 상태
  많음 (과여자)   → Ia 증가     진상(용량성) ← 전류가 전압보다 앞섬
```

그래프 모양이 영문자 V처럼 생겨서 V 곡선이라고 합니다.

```
         Ia
         |
         |  *       *      ← 부족여자 / 과여자
         |   *     *
         |    *   *
         |     * *         ← 역률 = 1.0 (Ia 최소점)
         |______*__________→ If
         
           부족여자  과여자
           (지상)   (진상)
```

---

## 4. 역률 제어

동기전동기가 유도전동기와 구별되는 가장 큰 특징입니다.

```
유도전동기:   항상 지상 역률 (전기 계통에서 무효전력 소비)
동기전동기:   여자 전류 조절로 역률 자유롭게 변환 가능

과여자 운전:  진상 역률 → 계통에 용량성 무효전력 공급
              → 다른 유도전동기의 지상 무효전력 상쇄
              → 전체 계통 역률 개선!
```

**비유:** 전기 계통에서 유도전동기가 "빌려 쓴" 무효전력을 동기전동기의 과여자 운전으로 "갚아주는" 역할

---

## 5. 동기 조상기 (Synchronous Condenser)

### 개념

무부하 상태로 운전하는 동기전동기를 전력 계통의 역률 개선에 사용하는 장치입니다.

```
부하 없이 계자 전류만 조정 → 무효전력 조절 장치로 활용

과여자:  진상 무효전력 공급 (전력 콘덴서처럼 동작)
부족여자: 지상 무효전력 흡수 (리액터처럼 동작)
```

| 비교 | 동기 조상기 | 콘덴서(커패시터) |
|------|-----------|--------------|
| 무효전력 방향 | 양방향 조절 가능 | 진상 방향만 가능 |
| 응답 속도 | 연속적, 부드럽게 조절 | 단계적 투입·차단 |
| 손실 | 있음 (회전체) | 거의 없음 |
| 설치 비용 | 높음 | 낮음 |

**용도:** 장거리 송전 선로의 전압 안정화, 대형 변전소 역률 개선

---

## 6. 기동 방법

동기전동기는 기동 토크가 없어 처음에 스스로 기동하지 못합니다.

```
왜 기동 토크가 없나?
정지 상태 회전자 vs 동기 속도로 돌아가는 회전 자기장
→ 너무 빠른 속도 차이로 회전자가 잡아당겨지기도 전에 밀려버림
→ 평균 토크 = 0
```

### 기동 방법

| 방법 | 원리 | 특징 |
|------|------|------|
| 자기 기동법 | 제동권선(Damper Winding)으로 유도전동기처럼 기동 | 가장 일반적 |
| 보조 전동기법 | 소형 유도전동기로 동기 속도까지 올린 후 계통 연결 | 대형기에 사용 |
| 저주파 기동법 | 주파수를 0부터 서서히 높여 기동 | 인버터 필요 |

**자기 기동법 순서:**
```
1. 계자 회로 열어둠 (또는 저항 연결)
2. 제동권선에 유도 전류 발생 → 유도 전동기처럼 기동
3. 동기 속도 부근까지 가속 (약 95%)
4. 계자 전류 인가 → 동기 속도로 인입 (pull-in)
```

---

## 7. 안정도와 탈조

```
안정 운전 조건: 부하각 δ < 90°

δ가 90°를 넘으면:
→ 복원 토크 없음 → 동기 이탈 → 탈조(Step-out)
→ 전동기 정지

탈조 방지:
- 단락비 큰 기기 사용
- 신속한 여자 제어
- 제동권선 설치 (난조 방지)
```

## 예제 (V 곡선 이해)

**문제:** 동기전동기를 과여자로 운전할 때 나타나는 현상 두 가지를 설명하시오.

```
1. 역률이 진상(앞섬, 용량성)이 된다.
   → 전기자 전류 위상이 단자 전압보다 앞섬

2. 전기자 전류 Ia가 증가한다.
   → V 곡선에서 계자 전류가 정격 여자를 넘으면
     Ia가 다시 증가하는 것 확인

부가 효과: 계통에 진상 무효전력을 공급하여 역률 개선에 기여
```
