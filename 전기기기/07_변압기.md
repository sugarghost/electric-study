# 07. 변압기 (Transformer)

## 핵심 개념
> 전자기 유도로 전압·전류를 변환하며, 효율이 99%에 달하는 가장 완성도 높은 전기기기

---

## 1. 동작 원리

```
1차 코일에 교류 전류 흐름
       ↓
자속(φ) 변화 발생
       ↓
2차 코일에 상호 인덕턴스에 의한 기전력 유도
       ↓
2차 코일에 연결된 부하에 전력 공급
```

**비유:** 두 코일이 자기장을 통해 "원격 악수"를 하는 것과 같습니다. 전선이 직접 연결되지 않아도 에너지가 전달됩니다.

**변압기가 직류에서 안 되는 이유:** 직류는 전류가 일정 → 자속 변화 없음 → 기전력 유도 없음

---

## 2. 권수비 (변압비)

변압기의 가장 기본이 되는 관계식입니다.

```
a = N₁/N₂ = V₁/V₂ = I₂/I₁ = √(Z₁/Z₂)

N₁, N₂ : 1차, 2차 권수
V₁, V₂ : 1차, 2차 전압
I₁, I₂  : 1차, 2차 전류
Z₁, Z₂  : 1차, 2차 임피던스

전압은 권수에 비례, 전류는 반비례
(전력 보존: V₁I₁ = V₂I₂)
```

**예시:**
```
N₁ = 1000, N₂ = 100 → a = 10 (강압 변압기)
V₁ = 1000V → V₂ = 100V
I₁ = 1A   → I₂ = 10A  (전류는 10배 증가)
```

---

## 3. 등가회로 (2차를 1차로 환산)

실제 계산을 위해 2차 회로를 1차 쪽으로 변환합니다.

```
2차 저항, 리액턴스를 1차로 환산:
R₂' = a² × R₂
X₂' = a² × X₂

2차 전압, 전류를 1차로 환산:
V₂' = a × V₂
I₂' = I₂ / a

권수비 a가 크면 (승압비 클수록)
→ 2차 임피던스를 1차로 환산하면 a² 배 증가
```

---

## 4. 전압 변동률

부하가 연결될 때 2차 전압이 얼마나 떨어지는지를 나타냅니다.

```
ε = (V₂₀ - V₂n) / V₂n × 100   [%]

V₂₀ : 무부하 2차 전압
V₂n : 정격 부하 시 2차 전압

근사식:
ε ≈ εr × cosφ + εx × sinφ   [지상 부하]
ε ≈ εr × cosφ - εx × sinφ   [진상 부하]

εr = %저항강하 = (I₁n × R₁eq / V₁n) × 100
εx = %리액턴스강하 = (I₁n × X₁eq / V₁n) × 100
```

- 전압 변동률이 작을수록 좋은 변압기 (전압 안정)
- 진상 부하일 때 ε이 음수가 될 수 있음 (전압이 오히려 올라감)

---

## 5. 효율

### 기본 효율

```
η = P₂ / (P₂ + Pc + Pi) × 100   [%]

P₂ : 2차 출력 = V₂ × I₂ × cosφ  [W]
Pc : 동손 (부하손, 전류에 의한 손실)
Pi : 철손 (무부하손, 전압에 의한 손실)
```

### 부하율에 따른 효율

```
m : 부하율 (0~1, 전부하=1)

Pc(m) = m² × Pc_full  (동손은 전류² 비례)
Pi = 일정 (부하에 무관)

η(m) = m × P₂_rated / (m × P₂_rated + m² × Pc + Pi) × 100
```

### 최대 효율 조건

```
최대 효율 조건: Pi = m² × Pc_full

최대 효율 부하율:
m = √(Pi / Pc_full)

예: Pi = 100W, Pc_full = 400W
→ m = √(100/400) = 0.5 → 50% 부하에서 최대 효율
```

### 전일 효율 (All-Day Efficiency)

하루 동안의 평균 효율을 나타냅니다.

```
η_day = 하루 출력 전력량 / 하루 입력 전력량 × 100

= Σ(P₂ × t) / [Σ(P₂ × t) + Pi × 24 + Σ(Pc × t)] × 100

철손은 24시간 계속 발생 (변압기는 항상 전압이 걸려 있으므로)
```

## 6. 변압기 시험

| 시험 종류 | 연결 방법 | 측정 항목 |
|---------|---------|---------|
| 무부하 시험 | 2차 개방, 1차에 정격 전압 인가 | 철손 Pi, 여자 전류 I₀ |
| 단락 시험 | 2차 단락, 1차에 소전압 인가 | 동손 Pc, 임피던스 전압 Vz |

**단락 시험의 임피던스 전압:**
```
%Vz = Vz / V₁n × 100   [%]

이 값이 작을수록 전압 변동률이 작고 병렬 운전이 쉬움
```

---

## 7. 결선 방식

3상 변압기의 1차와 2차를 어떻게 연결하느냐에 따라 특성이 달라집니다.

| 결선 | 위상차 | 특징 | 용도 |
|------|-------|------|------|
| Y-Y | 0° | 중성점 접지 가능 | 초고압 |
| Y-Δ | 30° | 1차 중성점 접지 가능 | 강압 (1차 고압, 2차 저압) |
| Δ-Y | 30° | 승압에 적합, 2차 중성점 접지 | 발전소 승압용 |
| Δ-Δ | 0° | 한 대 고장 시 V결선 운전 가능 | 안정적 |

**Y 결선 vs Δ 결선:**
```
Y결선: 선간 전압 = √3 × 상전압,  선전류 = 상전류
Δ결선: 선간 전압 = 상전압,      선전류 = √3 × 상전류
```

---

## 8. 병렬 운전 조건

두 변압기를 병렬로 연결하여 운전하려면 다음 조건이 필요합니다.

```
필수 조건 (불일치 시 순환전류 → 과부하·소손 위험):
1. 극성 일치
2. 권수비(변압비) 일치
3. %임피던스 전압 일치
4. 저항/리액턴스 비 일치 (완전 조건)

(3상의 경우 추가: 상회전 방향, 위상각 일치)
```

**부하 분담 원칙:**
```
부하 분담 비율 ∝ 정격 용량 / %임피던스

→ %Z가 작은 변압기가 부하를 더 많이 담당
```

## 예제 (계산)

**문제:** 1차 전압 V₁=6600V, 2차 전압 V₂=220V인 단상 변압기에서 2차 정격 전류 I₂n=100A일 때 권수비, 1차 전류, 변압기 용량을 구하시오.

```
권수비:
a = V₁/V₂ = 6600/220 = 30

1차 전류:
I₁ = I₂/a = 100/30 ≈ 3.33 [A]

변압기 용량:
P = V₂ × I₂ = 220 × 100 = 22000 [VA] = 22 [kVA]
```

**답: a=30, I₁≈3.33A, 용량=22kVA**

---

## 예제 2 (효율)

**문제:** 10kVA 변압기에서 철손 Pi=100W, 전부하 동손 Pc=200W, 역률 cosφ=0.8일 때 전부하 효율을 구하시오.

```
P₂ = 10000 × 0.8 = 8000 [W]

η = P₂ / (P₂ + Pi + Pc) × 100
  = 8000 / (8000 + 100 + 200) × 100
  = 8000 / 8300 × 100
  ≈ 96.4 [%]
```

**답: 약 96.4%**
