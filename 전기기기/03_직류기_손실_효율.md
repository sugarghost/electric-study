# 03. 직류기 손실과 효율

## 핵심 개념
> 발전기·전동기에서 에너지 일부는 열로 손실되며, 손실을 최소화할 때 최대 효율이 된다

---

## 1. 손실의 종류

전기기기에서 에너지가 낭비되는 방식은 크게 4가지입니다.

| 손실 종류 | 원인 | 발생 부위 | 부하 의존성 |
|---------|------|---------|-----------|
| 동손 (Copper Loss) | 전류에 의한 줄열 (I²R) | 전기자·계자 권선 | 가변손 (부하에 비례) |
| 철손 (Iron Loss) | 히스테리시스 + 와전류 | 전기자 철심 | 고정손 (부하 무관) |
| 기계손 (Mechanical Loss) | 마찰, 풍손(공기저항) | 베어링, 브러시 | 고정손 (속도 의존) |
| 표유부하손 (Stray Load Loss) | 누설 자속, 와전류 등 | 전체 기기 | 보통 전부하의 1% 가정 |

**비유:**
- 동손 = 수도관에서 마찰로 물 압력이 줄어드는 것 (전류가 많을수록 더 많이 손실)
- 철손 = 자석이 철판을 반복적으로 자화·탈자화할 때 생기는 열

---

## 2. 동손 (Copper Loss)

전류가 저항을 통과할 때 발생하는 열 손실입니다.

```
동손:
전기자 동손: Pca = Ia² × Ra
계자 동손:  Pcf = If² × Rf
총 동손:    Pc = Pca + Pcf

부하가 2배가 되면 동손은 4배 (제곱에 비례)
```

---

## 3. 철손 (Iron Loss)

**철손 = 히스테리시스손 + 와전류손**

### 히스테리시스손 (Hysteresis Loss)

```
Ph = kh × f × Bm^n   [W]

f  : 주파수 [Hz]
Bm : 최대 자속밀도 [T]
n  : 슈타인메츠 지수 (1.6 ~ 2.0)
kh : 재질에 따른 상수
```

**원리:** 자기 재료가 자화되었다가 반대로 자화될 때 분자 마찰로 에너지를 소비합니다. 히스테리시스 곡선의 면적이 클수록 손실이 큽니다.

**감소 방법:** 히스테리시스가 작은 규소강판 사용

### 와전류손 (Eddy Current Loss)

```
Pe = ke × f² × Bm² × t²   [W]

t : 철판의 두께 [m]
→ 두께가 2배 → 와전류손은 4배!
```

**원리:** 변화하는 자기장에 의해 철심 안에 소용돌이 전류(와전류)가 흘러 열을 발생시킵니다.

**감소 방법:** 얇은 철판을 여러 겹으로 쌓은 **적층 철심** 사용 (절연층으로 와전류 경로 차단)

```
적층 효과:
두께 t의 철판 1장 → 와전류손 ∝ t²
두께 t/n의 철판 n장 → 와전류손 ∝ (t/n)² × n = t²/n

→ n장으로 쌓으면 와전류손이 1/n로 감소
```

## 4. 효율 계산

### 기본 효율 공식

```
η = Pout / Pin × 100   [%]
  = Pout / (Pout + P_loss) × 100

P_loss = 전체 손실 합계
```

### 발전기 효율

```
발전기:
입력 = 기계 에너지 (원동기에서 받음)
출력 = 전기 에너지 (단자에서 나가는 전력)

η_gen = V×IL / (V×IL + P_loss) × 100   [%]

IL : 부하 전류
```

### 전동기 효율

```
전동기:
입력 = 전기 에너지 (전원에서 받음)
출력 = 기계 에너지 (축에서 나가는 전력)

η_mot = (V×Ia - P_loss) / (V×Ia) × 100   [%]

또는:
η_mot = (Pin - P_loss) / Pin × 100   [%]
```

---

## 5. 최대 효율 조건

가장 중요한 개념 중 하나입니다!

```
최대 효율 조건:
고정손 = 가변손
철손 + 기계손 = 동손

Pi = Pc
(철손 = 동손일 때 효율 최대)
```

**왜 그럴까?**

효율 η = P_out / (P_out + Pi + Pc)

Pi는 고정이고, Pc = k × IL² 이므로 (k는 상수)

전류 IL에 대해 미분하면 Pi = Pc일 때 최대 효율이 됩니다.

**최대 효율 부하율:**

```
부하율 m (최대 효율 발생점) = √(Pi / Pc_full)

Pi     : 철손 (고정손)
Pc_full: 전부하 동손

예: Pi = 100W, Pc_full = 400W이면
m = √(100/400) = √(0.25) = 0.5 → 50% 부하에서 최대 효율
```

---

## 6. 규약 효율 (Conventional Efficiency)

**왜 필요한가?** 대용량 기기는 실제 부하를 걸어 시험하기 어렵습니다 (에너지 낭비, 안전 문제).

따라서 각 손실을 측정하여 계산으로 효율을 구하는 방법을 사용합니다.

```
규약 효율:
η = (정격 출력) / (정격 출력 + 측정된 총 손실) × 100   [%]

실제로 부하를 연결하지 않고
→ 무부하 시험으로 철손 측정
→ 단락 시험(저항 측정)으로 동손 계산
→ 두 값을 더해 총 손실 산출
```

---

## 예제 (계산)

**문제:** 직류 발전기의 정격 출력 10kW, 철손 200W, 전부하 동손 300W, 기계손 100W일 때 전부하 효율을 구하시오.

```
총 손실 = 철손 + 동손 + 기계손
       = 200 + 300 + 100
       = 600 [W]

η = Pout / (Pout + P_loss) × 100
  = 10000 / (10000 + 600) × 100
  = 10000 / 10600 × 100
  ≈ 94.3 [%]
```

**답: 약 94.3%**

---

## 예제 2 (최대 효율 조건)

**문제:** 위 발전기에서 최대 효율이 발생하는 부하율을 구하시오.

```
고정손 Pi = 철손 + 기계손 = 200 + 100 = 300 [W]
전부하 동손 Pc = 300 [W]

최대 효율 부하율 m = √(Pi / Pc)
                  = √(300 / 300)
                  = √1
                  = 1.0 → 100% 부하에서 최대 효율
```

**답: 100% 부하 (고정손 = 가변손이므로 전부하에서 최대 효율)**

## 시험 포인트

| 공식 / 개념 | 내용 |
|------------|------|
| 동손 | `Pc = I²R` — 부하(전류)의 제곱에 비례하는 가변손 |
| 철손 | `Ph + Pe` — 히스테리시스손 + 와전류손, 고정손 |
| 와전류손 감소법 | 적층 철심 사용 (두께 얇게 여러 겹) |
| 발전기 효율 | 출력/(출력+손실) × 100 |
| 전동기 효율 | (입력-손실)/입력 × 100 |
| 최대 효율 조건 | **고정손 = 가변손** (철손 = 동손) |
| 최대 효율 부하율 | `m = √(고정손 / 전부하 동손)` |
| 규약 효율 | 손실 측정으로 계산하는 간접 효율 측정법 |
