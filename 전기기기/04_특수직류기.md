# 04. 특수 직류기

## 핵심 개념
> 특수 목적이나 정밀 제어를 위해 개발된 직류 전동기의 변형·파생 종류

---

## 1. 만능 전동기 (Universal Motor)

### 원리와 특징

직권 전동기를 교류에서도 사용할 수 있도록 개선한 전동기입니다.

**직권 전동기가 교류에서 동작하는 이유:**
```
직권 전동기:
- 계자권선과 전기자권선이 직렬 연결
- 교류 전원을 넣으면 계자 전류와 전기자 전류가 동시에 방향 바뀜
- T ∝ φ × Ia → 두 값이 동시에 반전되어 토크 방향은 유지
→ 교류·직류 모두 같은 방향으로 회전 가능
```

**특징:**

| 항목 | 내용 |
|------|------|
| 전원 | AC/DC 모두 사용 가능 |
| 속도 | 무부하 시 매우 빠름 (과속 주의) |
| 토크 | 기동 토크 큼 |
| 단점 | 소음, 브러시 마모, 수명 짧음 |

**용도:** 전동 드릴, 믹서, 진공청소기, 전동 공구 (손에 들고 쓰는 가전 대부분)

---

## 2. 서보 전동기 (Servo Motor)

### 개념

**서보(Servo)** = 라틴어로 "노예"라는 뜻 → 명령에 충실히 따르는 전동기

위치·속도·토크를 정밀하게 제어하기 위한 전동기로, 피드백 제어 시스템과 함께 사용합니다.

```
피드백 제어 구조:
지령값(목표 위치) → 제어기 → 서보 전동기 → 실제 위치
                         ↑_________________|
                         (인코더로 위치 검출)
```

### 특징

| 항목 | 내용 |
|------|------|
| 응답 속도 | 매우 빠름 (고응답성) |
| 제어 정밀도 | 매우 높음 |
| 구조 | 전동기 + 엔코더 + 드라이버 세트 |
| 회전 방향 | 정·역 양방향 정밀 제어 |

**용도:** CNC 공작기계, 산업용 로봇, 반도체 제조 장비, 의료 기기

---

## 3. 스테핑 모터 (Stepping Motor / Stepper Motor)

### 원리

펄스(디지털 신호) 1개가 입력될 때마다 일정한 각도(스텝각)만큼 회전합니다.

```
스텝각 예시:
- 1.8° 스텝 → 200 펄스로 1바퀴 (360°/1.8° = 200)
- 0.9° 스텝 → 400 펄스로 1바퀴 (더 정밀)

총 회전 각도 = 스텝각 × 입력 펄스 수
회전 속도 = 스텝각 × 초당 펄스 수
```

### 특징

| 항목 | 내용 |
|------|------|
| 제어 방법 | 디지털 펄스로 직접 제어 (별도 엔코더 불필요) |
| 정밀도 | 누적 오차 없음 (오픈루프 제어 가능) |
| 보유 토크 | 전원 인가 시 정지 상태에서도 큰 토크 발생 |
| 단점 | 고속 시 토크 감소, 공진 현상 발생 가능 |

**비유:** 시계의 초침 — 한 번에 정확히 1초(1스텝)씩 이동

**용도:** 프린터(용지 이송), CNC 이송 장치, 3D 프린터, 카메라 자동 초점

---

## 4. 영구자석 직류전동기 (PMDC: Permanent Magnet DC Motor)

### 특징

계자권선 대신 영구자석으로 자기장을 만드는 전동기입니다.

```
일반 직류전동기:
  계자권선에 전류 → 전자석으로 자기장 생성

PMDC:
  영구자석으로 자기장 고정 → 계자권선·계자 전류 불필요
```

| 항목 | 내용 |
|------|------|
| 구조 | 단순 (계자권선 없음) |
| 효율 | 높음 (계자 동손 없음) |
| 자속 제어 | 불가 (영구자석이므로 고정) |
| 크기·무게 | 소형·경량 |
| 단점 | 고온에서 영구자석 감자 위험 |

**용도:** 소형 가전 제품 모터, 자동차 창문 모터, 와이퍼 모터, 완구

---

## 5. 리니어 모터 (Linear Motor)

### 개념

일반 회전형 전동기를 "펼쳐서" 직선 운동을 얻는 전동기입니다.

```
일반 전동기: 회전 운동 → 기어·스크루로 직선 변환 (간접)
리니어 모터: 처음부터 직선 운동 발생 (직접)
```

**장점:**
- 기어, 스크루 등 기계적 변환 장치 불필요
- 마찰 적음, 응답 빠름, 고속 가능

**용도:** 자기부상열차(마그레브), 반도체 제조 스테이지, 공작기계 직선축

---

## 종류 요약 비교

| 종류 | 핵심 특징 | 주요 용도 |
|------|---------|---------|
| 만능 전동기 | AC/DC 겸용, 직권 구조 | 전동공구, 가전 |
| 서보 전동기 | 고정밀 위치·속도 제어, 피드백 | 로봇, CNC |
| 스테핑 모터 | 펄스 1개 = 스텝각 1회전 | 프린터, 3D프린터 |
| PMDC | 영구자석 계자, 소형 | 자동차 보조장치, 소형 가전 |
| 리니어 모터 | 직선 운동 직접 발생 | 자기부상열차, 정밀 스테이지 |

---

## 예제 (개념 적용)

**문제:** 스테핑 모터의 스텝각이 1.8°이고, 1초에 600 펄스가 입력될 때 분당 회전수(rpm)를 구하시오.

```
초당 회전각 = 스텝각 × 초당 펄스수
           = 1.8° × 600 = 1080°/s

1초에 1080° 회전 → 1초에 1080/360 = 3 바퀴

rpm = 3 × 60 = 180 [rpm]
```

**답: 180 rpm**

## 시험 포인트

| 공식 / 개념 | 내용 |
|------------|------|
| 만능 전동기 | 직권 전동기 구조, AC·DC 겸용, 가전기기 |
| 서보 전동기 | 피드백 제어, 고정밀, 로봇·CNC |
| 스테핑 모터 | 펄스 = 스텝각 회전, 오픈루프 제어 가능 |
| 스텝각 계산 | `회전각 = 스텝각 × 펄스수` |
| PMDC | 계자권선 없음, 영구자석, 소형·고효율 |
| 만능 전동기 토크 | `T ∝ φ × Ia` → AC에서도 두 값 동시 반전이므로 토크 방향 유지 |

### 시험 출제 경향

- 종류별 특징과 용도 구분 (객관식 위주)
- 스테핑 모터 스텝각 계산 (간단한 계산 문제)
- 만능 전동기가 교류에서 동작하는 이유 설명
