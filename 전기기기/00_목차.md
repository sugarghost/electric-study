## 핵심 공식 모음

### 직류기 공식

```
유기 기전력:   E = PZNφ / (60A)       [V]
토크:          T = PZφIa / (2πA)      [N·m]
회전수:        N ∝ (V - IaRa) / φ     [rpm]
전압 변동률:   ε = (V₀ - Vn) / Vn × 100  [%]
```

### 동기기 공식

```
동기 속도:     Ns = 120f / P           [rpm]
유기 기전력:   E = 4.44 × kw × f × N × φ   [V]
전력:          P = (EV / Xs) sinδ      [W/상]
단락비:        Ksc = 단락전류 / 정격전류
```

### 변압기 공식

```
권수비:        a = N₁/N₂ = V₁/V₂ = I₂/I₁
전압 변동률:   ε ≈ εr cosφ + εx sinφ  [%]
효율:          η = P₂ / (P₂ + Pc + Pi) × 100  [%]
최대 효율 조건: 철손(Pi) = 동손(Pc)
```

### 유도기 공식

```
슬립:          s = (Ns - N) / Ns
슬립 주파수:   f₂ = s × f₁
공극 전력:     P₂(gap) = T × ωs
실제 출력:     Pout = P₂(1 - s)
2차 동손:      Pcu2 = s × P₂
```

### 정류 회로 공식

```
단상 반파:     Vdc = Vm/π ≈ 0.318Vm
단상 전파:     Vdc = 2Vm/π ≈ 0.636Vm
3상 전파:      Vdc ≈ 1.35VL
SCR 위상 제어: Vdc = Vm(1 + cosα)/π
```

## 출제 비중 및 학습 전략

### 자주 출제되는 주제 TOP 5

1. **변압기** - 권수비, 효율, 병렬 운전 조건 (거의 매회 출제)
2. **유도전동기** - 슬립, 토크 계산, 비례추이 (매우 중요)
3. **동기발전기** - 동기 속도, 단락비, 전력각 (4문제 이상)
4. **직류전동기** - 속도 제어, 기동법 (2~3문제)
5. **정류 회로** - SCR 위상 제어, 정류 전압 계산 (2문제)

### 학습 순서 추천

```
1단계: 직류기 (01 → 02 → 03) — 기초 개념 이해
2단계: 교류기 (05 → 06) — 동기 속도 개념 확립
3단계: 변압기 (07) — 가장 출제 비중 높음, 집중 학습
4단계: 유도기 (08) — 슬립·토크 반복 연습
5단계: 나머지 (04, 09) — 마무리 정리
```

## 단위 정리

| 기호 | 의미 | 단위 |
|------|------|------|
| E | 기전력 | V (볼트) |
| T | 토크 | N·m (뉴턴미터) |
| N | 회전수 | rpm |
| φ | 자속 | Wb (웨버) |
| P | 극수 또는 전력 | - 또는 W |
| s | 슬립 | 무차원 (0~1) |
| η | 효율 | % |
| ε | 전압 변동률 | % |
