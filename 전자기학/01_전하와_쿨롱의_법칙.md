# 01. 전하와 쿨롱의 법칙

## 전자기학을 시작하기 전에

전자기학은 **눈에 보이지 않는 힘과 장(場, Field)** 을 다루는 과목입니다.  
회로이론이 "전기가 도선을 따라 흐르는 것"이라면,  
전자기학은 "전기와 자기가 공간에서 어떻게 작용하는가"입니다.

처음엔 추상적으로 느껴지는 게 정상입니다. 공식 패턴을 익히는 데 집중하세요.

---

## 전하 (Electric Charge)

- 전기의 근원이 되는 물리량
- 단위: 쿨롱 [C]
- 종류: **양전하(+)** 와 **음전하(-)**

### 전하의 기본 성질

```
같은 부호 → 서로 밀어냄 (반발력)
다른 부호 → 서로 당김 (흡인력)
```

### 전자 1개의 전하량

```
e = 1.602 × 10⁻¹⁹ [C]
```

---

## 쿨롱의 법칙 (Coulomb's Law)

> 두 점전하 사이에 작용하는 힘

### 비유

자석끼리 가까울수록 강하게 붙고, 멀어질수록 약해지는 것처럼  
전하 사이의 힘도 **거리의 제곱에 반비례**합니다.

### 공식

```
F = k × Q1 × Q2 / r²

= (1/4πε) × Q1 × Q2 / r²
```

| 기호 | 의미 | 값 |
|------|------|-----|
| F | 두 전하 사이의 힘 [N] | - |
| Q1, Q2 | 각 전하량 [C] | - |
| r | 두 전하 사이의 거리 [m] | - |
| k | 쿨롱 상수 | 9×10⁹ [N·m²/C²] |
| ε (엡실론) | 유전율 | ε₀ = 8.854×10⁻¹² [F/m] (진공) |

```
k = 1 / (4πε₀) = 9 × 10⁹  [N·m²/C²]
```

---

## 유전율 (Permittivity, ε)

전기장이 매질(공간의 재질)을 얼마나 잘 통과하는지를 나타냄

```
ε = ε₀ × εr

- ε₀: 진공의 유전율 = 8.854 × 10⁻¹² [F/m]
- εr: 비유전율 (재료에 따른 상대값, 진공에서 εr = 1)
```

| 매질 | 비유전율 εr |
|------|-----------|
| 진공/공기 | 1 |
| 종이 | ~3.5 |
| 유리 | ~5~10 |
| 물 | ~80 |

유전율이 크면 → 전기장이 약해짐 (쿨롱 힘도 약해짐)

---

## 예제

**Q1 = 2×10⁻⁶ C, Q2 = 3×10⁻⁶ C, r = 0.3m, 진공 중에서 힘은?**

```
F = 9×10⁹ × (2×10⁻⁶ × 3×10⁻⁶) / (0.3)²
  = 9×10⁹ × 6×10⁻¹² / 0.09
  = 9×10⁹ × 6.67×10⁻¹¹
  = 0.6 [N]

같은 부호이므로 → 반발력 0.6N
```

## 시험 포인트

| 공식 | 내용 |
|------|------|
| F = kQ1Q2/r² | 쿨롱의 법칙 |
| k = 9×10⁹ | 쿨롱 상수 (진공) |
| ε₀ = 8.854×10⁻¹² | 진공의 유전율 |
| ε = ε₀εr | 매질의 유전율 |

- 힘은 **거리의 제곱에 반비례** (거리 2배 → 힘 1/4)
- 힘은 **전하량의 곱에 비례** (전하 2배 → 힘 2배)
- 매질이 바뀌면 ε가 바뀌어 힘이 달라짐
