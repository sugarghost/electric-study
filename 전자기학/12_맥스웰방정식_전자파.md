# 12. 맥스웰 방정식 & 전자파

## 맥스웰 방정식이란?

> 전기와 자기의 모든 현상을 **4개의 방정식**으로 완벽하게 기술

전자기학의 최종 보스. 시험에서는 각 방정식의 의미와 이름을 알면 충분합니다.

---

## 4개의 맥스웰 방정식

### 1. 가우스 법칙 (전기)

```
∇ · D = ρ   (미분형)
∯ D · dS = Q   (적분형)
```

**의미**: 전속밀도의 발산 = 전하 밀도  
→ 전하가 전기장의 원천

---

### 2. 가우스 법칙 (자기)

```
∇ · B = 0   (미분형)
∯ B · dS = 0   (적분형)
```

**의미**: 자속밀도의 발산 = 0  
→ 자기 단극자(단독 N극 또는 S극)는 존재하지 않는다  
→ 자기력선은 항상 폐루프

---

### 3. 패러데이 법칙

```
∇ × E = -∂B/∂t   (미분형)
∮ E · dl = -dΦ/dt   (적분형)
```

**의미**: 변화하는 자기장 → 전기장(기전력) 유도  
→ 발전기, 변압기의 원리

---

### 4. 앙페르-맥스웰 법칙

```
∇ × H = J + ∂D/∂t   (미분형)
∮ H · dl = I + Id   (적분형)
```

**의미**: 전류 + 변화하는 전기장 → 자기장 형성  
- J: 전도 전류 밀도
- ∂D/∂t: 변위 전류 (맥스웰이 추가한 항)

> **변위 전류**: 실제 전류가 없어도 전기장이 변하면 자기장이 생긴다  
> → 전자파 존재를 예측한 핵심 개념

---

## 전자파 (Electromagnetic Wave)

맥스웰 방정식으로 예측되고 헤르츠가 실험으로 확인

### 전자파의 특성

- 전기장(E)과 자기장(B)이 **서로 수직**으로 진동하며 전파
- 진행 방향 = E × B 방향
- **진공에서 빛의 속도**로 전파

```
c = 1/√(μ₀ε₀) = 3 × 10⁸ [m/s]
```

---

## 전파 속도

매질에서의 전파 속도:

```
v = 1/√(με) = c/√(μrεr)   [m/s]
```

---

## 파장과 주파수

```
c = f × λ

λ = c/f   [m]

- f: 주파수 [Hz]
- λ (람다): 파장 [m]
```

| 주파수 | 파장 | 종류 |
|--------|------|------|
| 10⁴~10⁶ Hz | km | 장파, AM 라디오 |
| 10⁸~10⁹ Hz | m | FM 라디오, TV |
| 10⁹~10¹² Hz | cm~mm | 마이크로파 |
| 10¹⁴~10¹⁵ Hz | μm | 가시광선 |

---

## 포인팅 벡터 (Poynting Vector)

전자파가 운반하는 **전력 밀도** (단위 면적당 전력):

```
S = E × H   [W/m²]
```

- 방향: 전자파의 진행 방향
- 크기: 단위 면적을 통과하는 전력

---

## 고유 임피던스 (Intrinsic Impedance)

전자파에서 E와 H의 비율:

```
η = E/H = √(μ/ε)   [Ω]

진공: η₀ = √(μ₀/ε₀) ≈ 377 [Ω]
```

---

## 시험 포인트

| 항목 | 내용 |
|------|------|
| 맥스웰 4방정식 | 이름과 의미 각각 암기 |
| c = 3×10⁸ m/s | 빛의 속도 (진공에서 전자파 속도) |
| c = fλ | 파장-주파수 관계 |
| η₀ ≈ 377Ω | 진공의 고유 임피던스 |
| 변위 전류 | 전기장 변화 → 자기장 생성 (맥스웰 추가) |

- 맥스웰 방정식 전체를 계산하는 문제는 거의 없음
- 각 방정식의 **물리적 의미와 이름**을 연결해서 기억
- 전자파 속도 공식 c = 1/√(μ₀ε₀) 출제 가능
