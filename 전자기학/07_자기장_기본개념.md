# 07. 자기장 기본개념 (Magnetic Field)

## 전기장 vs 자기장

| 구분 | 전기장 | 자기장 |
|------|--------|--------|
| 원인 | 전하 (Q) | 전류 (I) 또는 자석 |
| 세기 | E [V/m] | H [A/m] |
| 밀도 | D = εE [C/m²] | B = μH [T, 테슬라] |
| 힘 | F = QE | F = QvB (운동하는 전하에) |

---

## 자기장 H와 자속밀도 B

```
B = μ × H   [T, 테슬라]

- μ (뮤): 투자율 [H/m]
- H: 자기장의 세기 [A/m]
- B: 자속밀도 [T = Wb/m²]
```

### 투자율 μ (Permeability)

자기장이 매질을 얼마나 잘 통과하는지

```
μ = μ₀ × μr

- μ₀: 진공의 투자율 = 4π × 10⁻⁷ [H/m] ≈ 1.257 × 10⁻⁶
- μr: 비투자율 (재료에 따라 다름)
```

| 재료 | 비투자율 μr |
|------|-----------|
| 진공/공기 | 1 |
| 알루미늄 | ≈1.00002 |
| 철(연철) | 수천~수만 |
| 니켈 | ~600 |

> 철은 자기장을 매우 잘 통과시킴 → 전자석, 변압기 철심에 사용

---

## 자기력선의 성질

전기력선과 비슷하지만 중요한 차이가 있습니다.

| 전기력선 | 자기력선 |
|---------|---------|
| 양전하에서 시작, 음전하에서 끝 | **항상 폐루프** (시작과 끝이 없음) |
| 고립된 전하 존재 | 고립된 자극(단자극) 없음 |

> 자석의 N극과 S극은 항상 함께 존재 (단독으로 존재 불가)

---

## 자속 (Magnetic Flux, Φ)

자기력선의 총량

```
Φ = B × A × cosθ   [Wb, 웨버]

- B: 자속밀도 [T]
- A: 면적 [m²]
- θ: B와 면의 법선 방향 사이 각도
```

수직으로 통과할 때(θ=0): `Φ = B × A`

---

## 자기쌍극자와 자기 모멘트

영구자석이나 전류 루프는 자기 쌍극자처럼 행동합니다.

```
자기 모멘트: m = I × A × n̂   [A·m²]

- I: 전류
- A: 루프 면적
- n̂: 오른손 법칙으로 정해지는 법선 방향
```

---

## 자성체의 종류

| 종류 | μr | 특성 |
|------|-----|------|
| 반자성체 | < 1 | 자기장에 약하게 반발 (금, 은, 구리) |
| 상자성체 | > 1 | 자기장에 약하게 끌림 (알루미늄, 백금) |
| 강자성체 | >> 1 | 자기장에 강하게 끌림 (철, 니켈, 코발트) |

> 시험에서는 **강자성체** (철)가 가장 중요

---

## 히스테리시스 (Hysteresis)

강자성체의 자화 특성 — B-H 곡선이 루프를 형성

```
자화 → 자기장 제거 → 잔류자기(Br) 남음
반대 자기장 → 보자력(Hc)에서 자화 소멸
```

- **잔류자기 Br**: 외부 자기장 없어도 남는 자속밀도
- **보자력 Hc**: 잔류자기를 없애는 데 필요한 반대 자기장
- **히스테리시스 손실**: 자화 반전 시 에너지 손실 → 열 발생

---

## 시험 포인트

| 공식 | 내용 |
|------|------|
| B = μH | 자속밀도와 자기장 관계 |
| μ₀ = 4π×10⁻⁷ | 진공 투자율 |
| Φ = BA | 자속 (수직 통과 시) |

- B와 H의 단위 구분: B[T], H[A/m]
- 자기력선은 항상 폐루프 (N→외부→S→내부→N)
- 강자성체: 큰 μr → 변압기/모터 철심 사용 이유
- 히스테리시스 손실 = 철심 손실의 원인
