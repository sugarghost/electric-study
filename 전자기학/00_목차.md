## 전자기학 vs 회로이론 차이

| 회로이론 | 전자기학 |
|---------|---------|
| 도선 위에서 전기가 흐름 | 공간(진공) 속에서 전기·자기 작용 |
| V, I, R 중심 | E, B, ε, μ 중심 |
| 계산 도구: 옴/KVL/KCL | 계산 도구: 가우스/암페어 법칙 |

## 핵심 공식 모음

### 전기장 (정전기학)

```
F = kQ1Q2/r²         쿨롱의 법칙 (k = 9×10⁹)
E = kQ/r²            점전하의 전기장
E = ρL/(2πεr)        선전하의 전기장
E = ρS/(2ε)          면전하의 전기장
V = kQ/r             점전하의 전위
E = -dV/dr           전기장 = 전위의 기울기
D = εE               전속밀도
ε = ε₀εr             유전율
ε₀ = 8.85×10⁻¹²     진공 유전율
```

### 캐패시턴스

```
C = Q/V              캐패시턴스 정의
C = εA/d             평행판 캐패시터
W = ½CV²             저장 에너지
직렬: 1/C_합 = Σ1/C
병렬: C_합 = ΣC
```

### 자기장 (정자기학)

```
B = μH               자속밀도
μ₀ = 4π×10⁻⁷        진공 투자율
H = I/(2πr)          직선 도선 주위 자기장
H = I/(2a)           원형 코일 중심
H = NI/L = nI        솔레노이드 내부
Φ = BA               자속
```

### 전자기 유도

```
e = -N dΦ/dt         패러데이 법칙
e = BLv sinθ         운동 기전력
F = BIL sinθ         도선에 작용하는 힘
F = μ₀I₁I₂L/(2πr)   평행 도선 사이 힘
L = μN²A/l           솔레노이드 인덕턴스
F = NI               기자력
Rm = l/(μA)          자기저항
```

### 전자파

```
c = 3×10⁸ m/s        빛의 속도 (진공)
c = fλ               파장-주파수 관계
η₀ = 377Ω            진공 고유 임피던스
```

## 전기장-자기장 완전 대응표

| 전기 | 자기 |
|------|------|
| 전하 Q [C] | 전류 I [A] |
| 전기장 E [V/m] | 자기장 H [A/m] |
| 전속밀도 D [C/m²] | 자속밀도 B [T] |
| 유전율 ε [F/m] | 투자율 μ [H/m] |
| D = εE | B = μH |
| 가우스 법칙 | 암페어 법칙 |
| 전위 V [V] | 기자력 F [A] |
| 캐패시턴스 C [F] | 인덕턴스 L [H] |

## 공부 방법

1. **전기장 파트(01~06)** 먼저 완료 → 자기장은 대응 개념이라 수월
2. **공식 대응표** 를 활용 — 전기와 자기는 같은 구조
3. 기출문제에서 숫자 계산보다 **"어느 공식을 쓰는가"** 가 핵심
4. 단위 구분 (E: V/m, D: C/m², B: T, H: A/m) 실수 주의
