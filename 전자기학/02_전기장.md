# 02. 전기장 (Electric Field, E)

## 핵심 개념

> 전하 주위의 공간에 형성되는 **전기적인 힘의 장(Field)**  
> "어떤 곳에 전하를 놓으면 얼마의 힘을 받는가"를 나타냄

### 비유

자석 주위에 철가루를 뿌리면 자기력선이 보이듯,  
전하 주위의 공간에도 보이지 않는 전기력선이 뻗어 있습니다.  
**전기장 = 그 공간에서 단위 전하가 받는 힘**

---

## 전기장의 정의

```
E = F / Q   [V/m 또는 N/C]

- E: 전기장의 세기
- F: 전하 Q가 받는 힘 [N]
- Q: 전하량 [C]
```

> 즉, 1C의 전하를 놓았을 때 받는 힘 [N]이 전기장의 세기

---

## 점전하에 의한 전기장

점전하 Q로부터 거리 r인 지점의 전기장:

```
E = k × Q / r²  = Q / (4πεr²)   [V/m]
```

| 특징 | 내용 |
|------|------|
| 방향 | 양전하에서 나가는 방향, 음전하로 들어오는 방향 |
| 크기 | 거리의 제곱에 반비례 |

---

## 전기력선 (Electric Field Line)

전기장의 방향과 세기를 시각적으로 표현한 선

### 전기력선의 성질

1. 양전하(+)에서 출발 → 음전하(-)로 들어감
2. 서로 교차하지 않음
3. 전기력선이 **촘촘할수록** 전기장이 강함
4. 전기력선의 방향 = 전기장의 방향

```
    +Q          -Q
   →→→        →→→
   →→→  →→→  →→→
   →→→        →→→
```

---

## 전기장의 중첩

여러 전하에 의한 전기장 → **벡터 합**

```
E_합 = E₁ + E₂ + E₃ + ...  (벡터 합산)
```

같은 방향이면 더하고, 반대 방향이면 빼면 됩니다.

### 예제

+Q와 -Q가 각각 점 P에서 동일한 거리에 있을 때:

```
E₁ (양전하에서 멀어지는 방향)
E₂ (음전하로 향하는 방향)
→ 두 벡터가 같은 방향이면 E_합 = E₁ + E₂
```

---

## 무한 직선 전하에 의한 전기장

선전하 밀도 ρ_L [C/m]인 무한 직선에서 거리 r인 지점:

```
E = ρ_L / (2πεr)   [V/m]
```

- 거리에 **1승에 반비례** (점전하의 r²과 다름!)

---

## 무한 평면 전하에 의한 전기장

면전하 밀도 ρ_S [C/m²]인 무한 평면:

```
E = ρ_S / (2ε)   [V/m]
```

- 거리에 **무관** (평면에서 멀어져도 세기 동일)

---

## 정리: 전하 분포 종류별 전기장

| 전하 분포 | 전기장 공식 | 거리 의존성 |
|---------|----------|----------|
| 점전하 | E = Q/(4πεr²) | 1/r² |
| 선전하 | E = ρL/(2πεr) | 1/r |
| 면전하 | E = ρS/(2ε) | 무관 |

---

## 시험 포인트

| 공식 | 내용 |
|------|------|
| E = F/Q | 전기장의 정의 |
| E = kQ/r² | 점전하의 전기장 |
| E = ρL/(2πεr) | 선전하의 전기장 |
| E = ρS/(2ε) | 면전하의 전기장 |

- **점/선/면 전하의 공식 구분**이 자주 출제
- 단위: [V/m] = [N/C]
- 전기장 방향: 양전하에서 바깥쪽, 음전하로 안쪽
