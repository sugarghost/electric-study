# 11. 자기회로 (Magnetic Circuit)

## 핵심 개념

> 전기회로(전류가 흐르는 경로)처럼  
> **자속이 흐르는 경로**를 자기회로라고 합니다.

전기회로의 개념을 자기에 그대로 대응시켜 분석합니다.

---

## 전기회로 ↔ 자기회로 대응 관계

| 전기회로 | 자기회로 |
|---------|---------|
| 기전력 E [V] | 기자력 F = NI [A] |
| 전류 I [A] | 자속 Φ [Wb] |
| 저항 R [Ω] | 자기저항 Rm [A/Wb] |
| 옴의 법칙 V=IR | 자기 옴의 법칙 F=Φ×Rm |

> 전기회로를 이해하면 자기회로는 **같은 방식**으로 풀 수 있습니다.

---

## 기자력 (Magnetomotive Force, MMF)

코일에 전류를 흘려서 자속을 만드는 힘

```
F = N × I   [A 또는 AT (암페어 턴)]

- N: 코일 권수
- I: 전류 [A]
```

---

## 자기저항 (Reluctance, Rm)

자속의 흐름을 방해하는 정도

```
Rm = l / (μ × A)   [A/Wb 또는 H⁻¹]

- l: 자기회로의 길이 [m]
- μ: 투자율 [H/m]
- A: 단면적 [m²]
```

> 전기저항 R = ρl/A 와 형태가 같습니다.  
> μ가 크면(철심) → Rm 작음 → 자속이 잘 흐름

---

## 자기 옴의 법칙

```
Φ = F / Rm = NI / Rm   [Wb]

또는: F = Φ × Rm
```

---

## 자기회로의 직렬/병렬

### 직렬 (공극이 있는 철심)

```
Rm_합 = Rm_철 + Rm_공극
```

공극(Air Gap)의 자기저항:
```
Rm_공극 = l_g / (μ₀ × A)
```

> 공극이 아주 작아도 μ₀가 매우 작으므로 자기저항이 커집니다.  
> 변압기/전동기 설계에서 공극 관리가 중요한 이유입니다.

### 병렬

```
1/Rm_합 = 1/Rm₁ + 1/Rm₂
```

---

## 자기회로 풀이 예제

**N=500턴 코일, 철심 길이 l=0.5m, 단면적 A=4×10⁻⁴m², μr=1000, I=?  
원하는 자속 Φ=8×10⁻⁴ Wb**

```
1. 자기저항 계산:
   Rm = l/(μ₀μrA)
      = 0.5 / (4π×10⁻⁷ × 1000 × 4×10⁻⁴)
      = 0.5 / (5.03×10⁻⁷)
      ≈ 9.95×10⁵ [A/Wb]

2. 필요한 기자력:
   F = Φ × Rm = 8×10⁻⁴ × 9.95×10⁵ ≈ 796 [A]

3. 필요한 전류:
   I = F/N = 796/500 ≈ 1.59 [A]
```

---

## 자속밀도와 자기장 세기의 관계

철심 내부:
```
B = Φ/A   [T]
H = NI/l = F/l   [A/m]
B = μH
```

---

## 시험 포인트

| 전기 대응 | 자기 개념 | 공식 |
|---------|---------|------|
| 기전력 V | 기자력 F | F = NI |
| 전류 I | 자속 Φ | - |
| 저항 R | 자기저항 Rm | Rm = l/(μA) |
| 옴의 법칙 | 자기 옴의 법칙 | F = Φ·Rm |

- 자기저항 공식 **Rm = l/(μA)** 반드시 암기
- 공극이 있으면 자기저항이 급격히 증가
- 철심(강자성체)은 μr이 크므로 자기저항이 매우 작음 → 자속을 잘 안내
