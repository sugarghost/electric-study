# 06. 캐패시턴스 (Capacitance)

## 핵심 개념

> 캐패시터(축전기)가 **전하를 저장하는 능력**

```
C = Q / V   [F, 패럿]

- C: 캐패시턴스
- Q: 저장된 전하량 [C]
- V: 양단에 걸린 전압 [V]
```

C가 클수록 → 같은 전압에서 더 많은 전하를 저장

---

## 평행판 캐패시터 (가장 기본)

```
+극판
|   |
| d |   ← 극판 사이 거리
|   |
-극판
  A  ← 극판 면적
```

### 공식

```
C = ε × A / d   [F]

- ε: 극판 사이 유전율 (ε = ε₀εr)
- A: 극판 면적 [m²]
- d: 극판 간 거리 [m]
```

### 직관적 관계

| 변수 | 변화 | C 변화 |
|------|------|--------|
| 면적 A | ↑ | C ↑ |
| 거리 d | ↑ | C ↓ |
| 유전율 ε (εr) | ↑ | C ↑ |

---

## 원통형(동축) 캐패시터

내경 a, 외경 b, 길이 L인 동축 원통:

```
C = 2πεL / ln(b/a)   [F]
```

---

## 구형 캐패시터

내경 a, 외경 b인 동심구:

```
C = 4πε × ab / (b-a)   [F]
```

---

## 캐패시터의 연결

### 직렬 연결

```
1/C_합 = 1/C1 + 1/C2 + ...

2개: C_합 = C1×C2 / (C1+C2)   (저항 병렬과 같은 공식!)
```

- 전하 Q: 동일 (직렬이므로)
- 전압: 각 C에 반비례해서 분배

### 병렬 연결

```
C_합 = C1 + C2 + ...   (저항 직렬과 같은 공식!)
```

- 전압 V: 동일 (병렬이므로)
- 전하: 각 C에 비례해서 분배

> **캐패시터 직렬 ↔ 저항 병렬 공식이 같고**  
> **캐패시터 병렬 ↔ 저항 직렬 공식이 같습니다.** 헷갈리지 말 것!

---

## 캐패시터에 저장되는 에너지

```
W = ½CV² = ½QV = Q²/(2C)   [J]
```

### 평행판 캐패시터의 에너지 밀도 (단위 부피당 에너지)

```
w = ½εE²   [J/m³]
```

---

## 유전체를 삽입했을 때 변화

평행판 캐패시터에 유전율 εr인 유전체를 채우면:

| 항목 | 변화 |
|------|------|
| 캐패시턴스 C | εr배 증가 |
| 전기장 E | εr배 감소 |
| 전압 V (Q 일정) | εr배 감소 |
| 전압 V (V 일정) | 변화 없음 |

---

## 예제

**ε₀=8.85×10⁻¹²F/m, εr=4, A=100cm², d=1mm 인 평행판 캐패시터의 캐패시턴스와 100V 인가 시 저장 에너지는?**

```
C = ε₀εr × A/d
  = 8.85×10⁻¹² × 4 × (100×10⁻⁴) / (1×10⁻³)
  = 8.85×10⁻¹² × 4 × 10
  = 354×10⁻¹² = 354 [pF]

W = ½CV² = ½ × 354×10⁻¹² × 100²
  = ½ × 354×10⁻¹² × 10⁴
  = 1.77×10⁻⁶ = 1.77 [μJ]
```

## 시험 포인트

| 공식 | 내용 |
|------|------|
| C = εA/d | 평행판 캐패시턴스 |
| C = Q/V | 캐패시턴스 정의 |
| W = ½CV² | 저장 에너지 |
| 직렬: 1/C_합 = Σ1/C | 직렬 합성 |
| 병렬: C_합 = ΣC | 병렬 합성 |

- 연결 공식이 저항과 반대임을 항상 확인
- 유전체 삽입 → C 증가, E 감소
- 에너지 3가지 공식 모두 암기 필요
