# 14. 전류와 전류밀도

## 핵심 개념
**전류밀도 J는 전류가 공간에 얼마나 촘촘하게 퍼져 흐르는지를 나타낸다.**
> 같은 100명이 이동해도, 10m 넓이의 복도를 지나면 붐비고, 100m 넓이의 광장을 지나면 여유롭다. 전류도 마찬가지다. 전류 I가 같아도 단면적이 좁으면 전류밀도 J가 크다.

---

## 1. 전류 복습

전류는 단위 시간당 흐르는 전하량이다.

```
I = dQ/dt   [A = C/s]
```

- 양전하가 이동하는 방향 = 전류의 방향 (관례)
- 실제로는 전자(음전하)가 반대 방향으로 이동
- 전류 I는 회로 전체에 대한 "총량" 개념

**한계:** I만으로는 전류가 단면의 어느 부분에 얼마나 집중되어 흐르는지 알 수 없다.  
→ 이 정보까지 담은 것이 **전류밀도 J**

---

## 2. 전류밀도 (Current Density)

전류밀도는 단위 면적당 흐르는 전류다.

```
J = I / A   [A/m²]
```

- A: 전류가 통과하는 단면적 [m²]
- J: 벡터 (전류가 흐르는 방향까지 포함)

**반대로, 전류밀도로부터 전류 계산:**
```
I = ∫∫ J · dS   (면적분)
```
균일한 경우에는:
```
I = J × A
```

**비유:** 수도관에서 물이 흐를 때,  
- I = 초당 흐르는 물의 양 (리터/초)  
- J = 관의 단면 1cm²당 흐르는 물의 양  
→ 관이 좁을수록 같은 물이 더 빠르게, 더 집중되어 흐른다.

---

## 3. 옴의 법칙 미분형: J = σE

회로이론의 V = IR을 공간 분포로 확장하면 더 근본적인 형태가 된다.

```
J = σE
```

- J: 전류밀도 [A/m²]
- σ (시그마): 전도율 (conductivity) [S/m]
- E: 전기장 [V/m]

**물리적 의미:** 전기장(전압이 만드는 힘)이 클수록, 전도율이 클수록 더 많은 전류가 흐른다.

**전도율 σ의 의미:**
- 전도율이 클수록 전류가 잘 흐름 → 도체
- 전도율이 작으면 전류가 잘 안 흐름 → 절연체

| 재료 | 전도율 σ [S/m] |
|------|--------------|
| 은(Ag) | 6.1 × 10⁷ |
| 구리(Cu) | 5.8 × 10⁷ |
| 알루미늄(Al) | 3.5 × 10⁷ |
| 바닷물 | ~4 |
| 순수한 물 | ~10⁻⁴ |
| 유리 | ~10⁻¹² |

---

## 4. 저항률(비저항)과 전도율

저항률(비저항) ρ는 전도율의 역수다.

```
σ = 1/ρ   ,   ρ = 1/σ
```

- ρ (로): 저항률(비저항) [Ω·m]
- 저항률이 클수록 전류가 잘 안 흐름 (전도율이 작은 것과 동일)

**주의:** 여기서 ρ는 전하밀도 ρ_v와 다른 기호다. 문맥으로 구분하자.

**저항 공식과의 연결:**

회로이론에서 배운 저항 공식은 전류밀도와 연결된다:
```
R = ρl/A = l/(σA)   [Ω]
```

- l: 도체의 길이 [m]
- A: 단면적 [m²]

**유도 과정 (이해용):**
```
V = El    (균일 전기장 가정)
I = JA = σEA

R = V/I = El/(σEA) = l/(σA)
```

---

## 5. 연속 방정식 (전하 보존 법칙의 미분형)

전하는 새로 생겨나거나 사라지지 않는다. 이를 수식으로 표현한 것이 연속 방정식이다.

```
∇·J = -∂ρ_v/∂t
```

- ρ_v: 체적 전하밀도 [C/m³]
- 왼쪽: 어느 점에서 전류가 흘러나가는 정도 (발산)
- 오른쪽: 그 점의 전하가 감소하는 속도

**비유:** 물탱크에서 물이 흘러나가면(발산이 양수), 탱크 안의 물이 줄어든다(∂ρ/∂t 가 음수).  
→ 흘러나간 만큼 줄어든다 = 전하 보존!

**정상 전류 (steady current) 의 경우:**  
시간이 변해도 전하분포가 바뀌지 않으면 ∂ρ_v/∂t = 0

```
∇·J = 0   (정상 전류 조건)
```

→ 전류는 새로 만들어지거나 없어지지 않고, 흘러들어온 만큼 흘러나간다.  
→ 회로이론의 키르히호프 전류 법칙(KCL)의 미분형!

---

## 6. 줄 열 (Joule Heating)

전류가 저항을 통과하면 열이 발생한다. 이를 단위 체적당 전력으로 표현:

```
p_v = J · E = J²/σ = σE²   [W/m³]
```

- p_v: 단위 체적당 전력 손실 (체적 전력 밀도)
- 전체 전력: P = ∫∫∫ p_v dV

**회로이론 P = I²R 과의 연결:**
```
P = I²R = (JA)² × (l/σA) = J² × l × A/σ = (J²/σ) × (체적)
```
→ 단위 체적당으로 나누면 p_v = J²/σ 가 된다.

---

## 7. 전류의 종류

| 종류 | 발생 원인 | 설명 | 해당 매질 |
|------|---------|------|----------|
| 전도전류 | 자유전자 이동 | 금속에서 흐르는 일반 전류 | 도체(금속) |
| 변위전류 | 전기장의 시간 변화 | 전하 이동 없이 전기장 변화로 생김 | 유전체, 진공 |
| 대류전류 | 전하를 띤 입자의 이동 | 이온이나 전자빔 등 | 진공, 이온 용액 |

**변위전류 (Displacement Current):**
```
J_d = ∂D/∂t = ε × ∂E/∂t
```
- 콘덴서 내부처럼 실제 전하가 흐르지 않아도 전기장이 변하면 마치 전류처럼 작용
- 맥스웰이 추가한 개념 → 전자기파 이론의 핵심

## 예제

**[예제 1] 전류밀도 계산**  
직경 2mm인 구리선에 5A의 전류가 흐를 때, 전류밀도 J를 구하시오.

```
반지름 r = 1mm = 1×10⁻³ m
단면적 A = πr² = π × (10⁻³)² = π × 10⁻⁶ m²
       A ≈ 3.14 × 10⁻⁶ m²

J = I/A = 5 / (3.14 × 10⁻⁶)
  ≈ 1.59 × 10⁶  A/m²
```

**[예제 2] 옴의 법칙 미분형 적용**  
어떤 도체에서 전기장 E = 0.1 V/m이고, 전도율 σ = 5.8×10⁷ S/m (구리)일 때 전류밀도를 구하시오.

```
J = σE = 5.8×10⁷ × 0.1
  = 5.8×10⁶  A/m²
```

**[예제 3] 저항 계산**  
길이 2m, 단면적 1mm² = 10⁻⁶ m²인 구리선(σ = 5.8×10⁷ S/m)의 저항을 구하시오.

```
R = l/(σA) = 2 / (5.8×10⁷ × 10⁻⁶)
  = 2 / 58
  ≈ 0.0345  Ω  (약 34.5 mΩ)
```

**[예제 4] 줄 열 계산**  
전류밀도 J = 10⁶ A/m², 전도율 σ = 5.8×10⁷ S/m일 때 단위 체적당 전력 손실은?

```
p_v = J²/σ = (10⁶)² / (5.8×10⁷)
    = 10¹² / (5.8×10⁷)
    ≈ 1.72×10⁴  W/m³
    = 17.2  kW/m³
```

## 시험 포인트

| 공식 | 의미 | 단위 |
|------|------|------|
| I = dQ/dt | 전류 = 전하/시간 | A |
| J = I/A | 전류밀도 = 전류/면적 | A/m² |
| J = σE | 옴의 법칙 미분형 | A/m² |
| σ = 1/ρ | 전도율과 저항률은 역수 관계 | S/m |
| R = l/(σA) = ρl/A | 저항 공식 | Ω |
| ∇·J = 0 | 정상 전류 조건 (KCL의 미분형) | - |
| p_v = J²/σ = σE² | 단위 체적당 전력 손실 | W/m³ |

**자주 출제되는 패턴:**
- "전류밀도 J를 구하시오" → 전도율과 전기장이 주어지면 J = σE
- "저항을 구하시오" → 길이, 단면적, 전도율이 주어지면 R = l/(σA)
- "정상 전류 상태에서 전류밀도의 발산은?" → ∇·J = 0
- 전도전류 vs 변위전류 구분 문제 자주 출제
