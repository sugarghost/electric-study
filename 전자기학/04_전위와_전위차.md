# 04. 전위와 전위차 (Electric Potential)

## 핵심 개념

> 전위 = 전기장 안의 어떤 점이 가진 **전기적인 높이(위치에너지)**  
> 전위차 = 두 점 사이의 전위 차이 = 우리가 아는 **전압 V**

---

## 비유: 높이와 위치에너지

| 역학 | 전기 |
|------|------|
| 높이 h | 전위 V |
| 기준면 (h=0) | 무한원점 (V=0) |
| 높이 차이 | 전위차 (전압) |
| 중력 (낮은 곳으로) | 전기력 (낮은 전위로) |

물이 높은 곳에서 낮은 곳으로 흐르듯,  
양전하(+)는 높은 전위에서 낮은 전위로 이동합니다.

---

## 전위의 정의

단위 양전하를 무한원점에서 그 지점까지 옮기는 데 필요한 일

```
V = W / Q   [J/C = V, 볼트]
```

---

## 점전하에 의한 전위

점전하 Q로부터 거리 r인 지점의 전위:

```
V = k × Q / r = Q / (4πεr)   [V]
```

### 전기장(E)과의 비교

```
E = kQ/r²   (1/r²에 비례)
V = kQ/r    (1/r에 비례)
```

> 전위는 전기장보다 거리에 덜 민감합니다.  
> V는 스칼라(방향 없음), E는 벡터(방향 있음)

---

## 전위와 전기장의 관계

```
E = -dV/dr   (1차원)
```

전기장은 전위가 감소하는 방향, 감소하는 비율이 전기장의 세기  
→ **전기장은 전위의 기울기(경사)**

### 등전위면

전위가 같은 점들을 이은 면 → **등전위면(Equipotential Surface)**

- 등전위면과 전기력선은 **항상 수직**
- 등전위면을 따라 이동 시 일 = 0 (전위 변화 없음)

---

## 전위차 (전압)

두 점 A, B 사이의 전위차:

```
VAB = VA - VB   [V]
```

A에서 B로 전하 Q를 이동할 때 한 일:

```
W = Q × VAB = Q × (VA - VB)   [J]
```

---

## 여러 전하에 의한 전위 (중첩)

전위는 **스칼라**이므로 중첩이 매우 간단합니다.

```
V_합 = V₁ + V₂ + V₃ + ...   (부호만 고려, 방향 없음)
```

> 전기장(벡터 합)보다 훨씬 계산이 쉽습니다.

---

## 예제

**Q₁= +4×10⁻⁹ C이 원점에, Q₂= -2×10⁻⁹ C이 x=3m에 있을 때,  
x=1m 지점의 전위는?**

```
r₁ = 1m (Q₁까지 거리)
r₂ = 2m (Q₂까지 거리)

V₁ = 9×10⁹ × 4×10⁻⁹ / 1 = 36 [V]
V₂ = 9×10⁹ × (-2×10⁻⁹) / 2 = -9 [V]

V_합 = 36 + (-9) = 27 [V]
```

---

## 에너지와 전위

전기장 안에서 전하를 이동시키는 데 필요한 에너지:

```
W = Q × V   [J]
```

두 점 사이를 이동할 때:

```
W = Q × (V₁ - V₂)   [J]
```

---

## 시험 포인트

| 공식 | 내용 |
|------|------|
| V = kQ/r | 점전하의 전위 |
| E = -dV/dr | 전기장 = 전위의 기울기 |
| W = QV | 전기적 일·에너지 |
| 등전위면 ⊥ 전기력선 | 기하학적 관계 |

- 전위는 **스칼라** → 합산이 간단 (부호만 신경)
- 전기장은 **벡터** → 방향까지 고려해야 함
- "전위가 높다 = 양전하가 많다 = 전기적으로 높은 곳"
