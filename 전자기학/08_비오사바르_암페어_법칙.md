# 08. 비오-사바르 법칙 & 암페어 법칙

## 핵심 개념

전류가 흐르는 도선 주위에 **자기장이 형성**됩니다.  
이 자기장의 크기와 방향을 구하는 두 가지 방법:

| 방법 | 특징 |
|------|------|
| 비오-사바르 법칙 | 일반적인 경우, 미소 요소 합산 |
| 암페어 법칙 | 대칭성이 있을 때 훨씬 간단 |

---

## 오른손 법칙 (방향 결정)

직선 전류 주위의 자기장 방향:

```
엄지 = 전류 방향
나머지 네 손가락이 감기는 방향 = 자기장 방향 (시계/반시계)
```

전류 루프(코일)의 경우:

```
네 손가락 = 전류가 도는 방향
엄지 = 자기장 방향 (N극 방향)
```

---

## 비오-사바르 법칙 (Biot-Savart Law)

미소 전류 요소 Idl에 의한 미소 자기장:

$$dH = \frac{I \cdot dl \cdot \sin\theta}{4\pi r^2}$$

$$H = \int \frac{I\, dl\, \sin\theta}{4\pi r^2}$$

### 무한 직선 도선

전류 I가 흐르는 무한 직선으로부터 거리 r인 지점:

$$H = \frac{I}{2\pi r} \quad [\text{A/m}]$$

- 자기장은 전선 주위를 원형으로 형성
- 거리에 **반비례** (1/r)

### 원형 코일 중심

반지름 a인 원형 코일에 전류 I:

$$H = \frac{I}{2a} \quad [\text{A/m}] \quad \text{(코일 중심에서)}$$

---

## 암페어 법칙 (Ampere's Law)

> 닫힌 경로를 따라 H를 선적분한 값 = 그 경로를 통과하는 총 전류

$$\oint \mathbf{H} \cdot d\mathbf{l} = I_{enc}$$

대칭적인 경우: $H \times l = I_{enc}$

가우스 법칙(전기)과 쌍을 이루는 자기장 법칙입니다.

---

## 암페어 법칙 실전 적용

### 무한 직선 도선

반지름 r인 원형 경로 설정:

$$H \times 2\pi r = I$$

$$H = \frac{I}{2\pi r}$$

(비오-사바르와 동일)

### 솔레노이드 (Solenoid)

N번 감긴 코일, 길이 L, 전류 I:

$$H = \frac{NI}{L} = nI \quad [\text{A/m}]$$

- $n = N/L$: 단위 길이당 권수

```
B = μH = μ × nI   [T]
```

> 솔레노이드 내부는 균일한 자기장, 외부는 거의 0

### 토로이드 (Toroid)

도넛 형태의 코일:

$$H = \frac{NI}{2\pi r} \quad [\text{A/m}]$$

- $r$: 토로이드 중심에서 해당 지점까지 거리

---

## 자기장 공식 정리

| 형태 | 자기장 H |
|------|---------|
| 무한 직선 | H = I/(2πr) |
| 원형 코일 중심 | H = I/(2a) |
| 솔레노이드 내부 | H = NI/L = nI |
| 토로이드 내부 | H = NI/(2πr) |

---

## 자기장과 전기장 대응 관계

| 전기 | 자기 |
|------|------|
| 전하 Q → 전기장 E | 전류 I → 자기장 H |
| 가우스 법칙 | 암페어 법칙 |
| 쿨롱 상수 k=1/4πε | μ/4π |
| 전속밀도 D=εE | 자속밀도 B=μH |

---

## 시험 포인트

| 공식 | 내용 |
|------|------|
| H = I/(2πr) | 직선 도선 주위 자기장 |
| H = I/(2a) | 원형 코일 중심 자기장 |
| H = NI/L | 솔레노이드 내부 자기장 |
| B = μH | 자속밀도 |

- **솔레노이드 공식 H = nI** 가장 자주 출제
- 방향은 항상 오른손 법칙으로 결정
- 암페어 법칙 = 자기장의 가우스 법칙
