# 10. 전자기 유도 (Electromagnetic Induction)

## 핵심 개념

> 자속(자기력선의 수)이 **변화할 때** 도선에 전압이 유도됩니다.  
> 발전기(Generator)의 원리입니다.

---

## 패러데이 법칙 (Faraday's Law)

```
e = -N × dΦ/dt   [V]

- e: 유도 기전력 [V]
- N: 코일 권수
- Φ: 자속 [Wb]
- dΦ/dt: 자속의 시간 변화율
```

### 핵심

- 자속이 **변하는 속도**가 빠를수록 유도 전압이 큼
- 자속이 일정하면 유도 전압 = 0
- 음의 부호(-): 렌츠의 법칙을 나타냄

---

## 렌츠의 법칙 (Lenz's Law)

> 유도 전류는 **자속의 변화를 방해하는 방향**으로 흐른다

### 직관적 이해

```
자속이 증가 → 유도 전류는 자속을 줄이려는 방향으로 흐름
자속이 감소 → 유도 전류는 자속을 늘리려는 방향으로 흐름
```

자연이 변화에 저항하는 원리 (일종의 "보수적 자연법칙")

### 예시

자석의 N극을 코일에 가까이 가져오면:
- 코일을 통과하는 자속 증가
- 유도 전류는 자석을 밀어내는 방향의 자기장을 만듦 (반발)

---

## 플레밍의 오른손 법칙 (발전기 법칙)

도선이 자기장 안에서 **움직일 때** 유도 전류 방향:

```
오른손을 펴서:
검지 → 자기장(B) 방향
엄지 → 도선의 운동 방향
중지 → 유도 전류(I) 방향
```

| 왼손 법칙 | 오른손 법칙 |
|---------|---------|
| **전동기** (전류 → 힘) | **발전기** (운동 → 전류) |

---

## 운동 기전력 (Motional EMF)

자기장 B 안에서 길이 L인 도선이 속도 v로 움직일 때:

```
e = B × L × v × sinθ   [V]

- θ: v와 B 사이의 각도
- θ = 90° (수직)일 때 최대
```

---

## 자기 인덕턴스와 유도 기전력

코일 자신의 전류 변화로 생기는 유도 기전력:

```
e = -L × di/dt   [V]

- L: 자기 인덕턴스 [H]
- di/dt: 전류의 시간 변화율
```

### 솔레노이드의 인덕턴스

```
L = μ × N² × A / l = μ × n² × A × l   [H]

- N: 총 권수
- n = N/l: 단위 길이당 권수
- A: 단면적
- l: 길이
```

---

## 상호 인덕턴스와 유도 기전력

코일 1의 전류 변화로 코일 2에 생기는 유도 기전력:

```
e₂ = -M × di₁/dt   [V]

- M: 상호 인덕턴스 [H]
```

---

## 와전류 (Eddy Current)

도체 덩어리가 변화하는 자기장 안에 있을 때  
도체 내부에 **소용돌이 모양의 유도 전류**가 흐름

```
와전류 손실 ∝ f² × B²
(주파수, 자속밀도의 제곱에 비례)
```

### 대책

- 철심을 **얇은 판으로 적층** (라미네이션)
- 판 사이를 절연 → 와전류 경로 차단

---

## 전자기 유도 응용

| 응용 | 원리 |
|------|------|
| 발전기 | 코일 회전 → 자속 변화 → 전압 유도 |
| 변압기 | 1차 전류 변화 → 2차에 유도 |
| 유도 전동기 | 회전 자기장 → 회전자에 유도 전류 → 힘 |
| 금속 탐지기 | 와전류 검출 |

---

## 시험 포인트

| 공식 | 내용 |
|------|------|
| e = -N dΦ/dt | 패러데이 법칙 |
| e = BLv sinθ | 운동 기전력 |
| e = -L di/dt | 자기 인덕턴스 유도 기전력 |
| L = μN²A/l | 솔레노이드 인덕턴스 |

- 렌츠의 법칙 = 유도 전류의 **방향** 결정 원리
- 발전기: 오른손, 전동기: 왼손 (절대 혼동 금지)
- 와전류 손실 → 적층 철심으로 감소
