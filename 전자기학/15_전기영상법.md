# 15. 전기영상법

## 핵심 개념
**도체 근처 전하 문제를, 거울에 비친 가상의 "영상 전하"로 간단하게 풀어내는 기법이다.**
> 거울 앞에 양초를 놓으면 거울 뒤에 똑같이 보이는 가상의 불꽃이 생긴다. 전기영상법도 마찬가지다. 접지된 도체 평면 뒤에 반대 부호의 가상 전하를 놓으면, 도체 표면에서 전위가 0이 되는 경계 조건이 자동으로 만족된다.

---

## 1. 왜 전기영상법이 필요한가?

도체 근처에 전하를 놓으면, 그 전하에 의해 도체 표면에 **유도 전하**가 생긴다.  
이 유도 전하가 다시 원래 전하에 힘을 미치기 때문에, 문제가 복잡해진다.

**문제:** 도체 표면의 유도 전하 분포를 직접 계산하는 것은 매우 어렵다.

**해결:** 도체를 제거하고, 그 자리에 가상의 "영상 전하(image charge)"를 놓으면  
경계 조건(도체 표면에서 전위 = 0)을 동일하게 만족시킬 수 있다.

**핵심 조건:** 접지된 도체 → 도체 표면의 전위 = 0

---

## 2. 무한 평면 도체와 점전하

### 기본 설정

```
          +Q (전하)
           |
           | d
           |
-----------+----------- 도체 평면 (접지, V=0)
           |
           | d
           |
          -Q (영상 전하, 가상)
```

- 도체 평면 위 거리 d 위치에 점전하 +Q 존재
- 영상 전하: 도체 평면 아래 거리 d 위치에 -Q 배치 (가상)
- 도체 평면 위(실제 공간)에서의 전기장, 전위 계산은 +Q와 -Q 두 전하가 있는 것처럼 계산

**검증:** 도체 평면 위의 임의의 점에서 +Q와 -Q에 의한 전위를 더하면 0이 된다.  
→ 경계 조건 만족!

### 두 전하 사이의 힘 (전하 받는 힘)

실제로 계산하는 힘은 +Q와 그것을 유인하는 영상 전하 -Q 사이의 힘이다.

두 전하 사이의 거리 = 2d (평면을 기준으로 양쪽에 d씩)

```
F = Q²/(4πε₀(2d)²)  = Q²/(16πε₀d²)   [N]
```

**방향:** +Q는 도체 쪽으로 끌려간다 (흡인력).  
→ 도체에 전하를 가져가면 항상 당기는 힘이 작용한다.

**비유:** 자석을 철판 가까이 가져가면 당겨지는 것과 같다.  
철판에 유도된 자화가 자석을 끌어당기듯, 도체에 유도된 전하가 원전하를 끌어당긴다.

### 도체 표면의 전하밀도

도체 표면에서 전기장 성분으로부터 유도 전하밀도를 계산할 수 있다.

도체 평면에서 전하 +Q로부터 수평 거리 r인 점의 표면 전하밀도:

```
σ_s = -Qd / (2π(r² + d²)^(3/2))   [C/m²]
```

- 부호가 음수: 도체 표면에 유도된 전하는 -Q (원전하 +Q와 반대 부호)
- r = 0 (전하 바로 아래): 전하밀도가 가장 크다
- r → ∞: 전하밀도가 0에 가까워진다

**전체 유도 전하 확인:**  
표면 전체에 걸쳐 σ_s를 적분하면 -Q가 나온다.  
→ 영상 전하의 크기와 일치!

## 3. 접지 구도체와 점전하

### 기본 설정

```
         +Q (외부 전하)
          ○   ← 점 P (전하 위치, 구 중심에서 거리 d)
              d
     ────────────
     /    ○  /
    /   영상 /    ← 접지 구도체 (반지름 a)
    \  전하 \    ← 구 중심에서 a²/d 위치에 영상 전하
     \      /
     ────────────
```

- 반지름 a인 접지 구도체 외부에 점전하 +Q가 구 중심으로부터 거리 d에 위치
- 구도체는 접지 (V = 0)

### 영상 전하

```
영상 전하의 크기:   Q' = -(a/d) × Q
영상 전하의 위치:   구 중심에서 a²/d 거리 (구 내부)
```

**특징:**
- Q' 의 크기: 원전하보다 작다 (a < d 이므로 a/d < 1)
- Q' 의 부호: 음수 (도체 평면의 경우와 동일하게 반대 부호)
- 위치: 구 내부에 위치 (d가 크면 영상 전하는 구 중심 가까이)

**검증:** 구 표면(r = a)의 임의의 점에서 +Q와 Q' 에 의한 전위 합 = 0  
→ 경계 조건 만족!

### 두 전하 사이의 힘

```
F = (1/(4πε₀)) × |Q × Q'| / (d - a²/d)²
  = (1/(4πε₀)) × |Q × (a/d)Q| / ((d² - a²)/d)²
  = Q²a / (4πε₀ × d × (d² - a²)/d²)²  ... (정리 필요)
```

간단히 기억할 포인트:
- 방향: 항상 흡인력 (원전하가 구도체 쪽으로 당겨짐)
- 크기: 쿨롱의 법칙에 Q와 Q', 두 전하 사이 거리 (d - a²/d)를 대입

---

## 4. 전기영상법의 적용 조건

| 조건 | 영상 전하 배치 |
|------|--------------|
| 무한 도체 평면 (접지) | 평면 반대편 같은 거리에 반대 부호 전하 |
| 접지 구도체 | 구 내부 a²/d 위치에 -(a/d)Q 전하 |
| 두 도체 평면 사이 | 다수의 영상 전하 (복잡) |

**전기영상법이 적용되는 조건:**
1. 도체가 접지되어 있어야 함 (V = 0)
2. 또는 도체의 전위가 일정하게 주어진 경우
3. 도체 경계면의 형태가 단순할 때 (평면, 구, 원통)

---

## 5. 전기영상법의 한계

- 복잡한 형태의 도체에는 적용하기 어렵다
- 유전체 경계에서는 다른 영상 전하 공식이 필요하다
- 어디까지나 경계 조건을 만족시키는 수학적 기법 (실제 영상 전하가 존재하는 것이 아님)

## 예제

**[예제 1] 도체 평면 위 점전하에 작용하는 힘**  
진공 중 무한 접지 도체 평면으로부터 d = 0.1m 떨어진 곳에 Q = 2×10⁻⁶ C 의 점전하가 있다. 이 전하에 작용하는 힘을 구하시오. (ε₀ = 8.85×10⁻¹² F/m)

```
영상 전하: -Q = -2×10⁻⁶ C, 도체 반대편 d = 0.1m
두 전하 사이 거리: 2d = 0.2m

F = Q² / (4πε₀(2d)²)
  = (2×10⁻⁶)² / (4π × 8.85×10⁻¹² × (0.2)²)
  = 4×10⁻¹² / (4π × 8.85×10⁻¹² × 0.04)
  = 4×10⁻¹² / (4.45×10⁻¹²)
  ≈ 0.899  N   (약 0.9N, 도체 방향으로 흡인)
```

**[예제 2] 영상 전하 결정 (구도체)**  
반지름 a = 0.1m인 접지 구도체가 있고, 구 중심에서 d = 0.5m 떨어진 곳에 Q = 4×10⁻⁶ C의 점전하가 있다. 영상 전하의 크기와 위치를 구하시오.

```
영상 전하 크기: Q' = -(a/d) × Q
                  = -(0.1/0.5) × 4×10⁻⁶
                  = -0.2 × 4×10⁻⁶
                  = -0.8×10⁻⁶ C   (= -0.8 μC)

영상 전하 위치: 구 중심에서 a²/d = (0.1)²/0.5 = 0.01/0.5 = 0.02m
              (구 내부, 중심에서 2cm)
```

**[예제 3] 도체 표면 전하밀도**  
도체 평면 위 d = 0.5m 위치에 Q = 10⁻⁶ C 의 전하가 있을 때, 전하 바로 아래(r = 0)의 표면 전하밀도를 구하시오.

```
σ_s = -Qd / (2π(r² + d²)^(3/2))

r = 0 이면:
σ_s = -Qd / (2π × d³)
    = -Q / (2πd²)
    = -10⁻⁶ / (2π × (0.5)²)
    = -10⁻⁶ / (2π × 0.25)
    = -10⁻⁶ / (1.571)
    ≈ -6.37×10⁻⁷  C/m²
```

## 시험 포인트

| 상황 | 영상 전하 크기 | 영상 전하 위치 | 힘 공식 |
|------|--------------|--------------|---------|
| 무한 도체 평면 + 전하 Q (거리 d) | -Q | 평면 반대편 거리 d | F = Q²/(16πε₀d²) |
| 접지 구도체(반지름 a) + 전하 Q (거리 d) | -(a/d)Q | 구 중심에서 a²/d | 쿨롱 공식 적용 |

**자주 출제되는 패턴:**
- "도체 평면에서 거리 d에 전하 Q → 힘 계산" → F = Q²/(16πε₀d²) 공식 직접 적용
- "구도체 근방 전하 → 영상 전하 크기와 위치" → Q' = -(a/d)Q, 위치 = a²/d
- "왜 도체 평면 근처 전하에 힘이 작용하는가?" → 유도 전하(영상 전하)에 의한 흡인력
- 영상 전하는 도체 내부(또는 반대편)에 위치 → 실제 공간의 전기장 계산에는 영향을 주지 않음
- 힘의 방향: **항상 도체 쪽으로 흡인**

**기억 팁:**
- 도체 평면: 거울처럼 반대편 동일 거리에 반대 부호
- 구도체: 영상 전하는 구 내부로 들어가고, 크기도 줄어든다 (a/d 비율)
- 두 경우 모두 영상 전하 부호는 원전하와 반대
