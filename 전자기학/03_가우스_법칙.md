# 03. 가우스 법칙 (Gauss's Law)

## 핵심 개념

> 닫힌 면(폐곡면)을 통해 나가는 전기장의 총량  
> = 그 면 안에 있는 전체 전하량 / ε

복잡한 전하 분포에서 전기장을 **훨씬 쉽게** 구할 수 있는 강력한 도구입니다.

---

## 직관적 이해

### 비유: 물이 나오는 수도꼭지

수도꼭지 주위를 어떤 모양의 막으로 감싸도,  
그 막을 통과하는 물의 양은 수도꼭지에서 나오는 물의 양과 같습니다.

마찬가지로, 전하 주위를 어떤 폐곡면으로 감싸도  
그 면을 통과하는 전기장의 총량은 안에 있는 전하에만 의존합니다.

---

## 공식

```
∯ E · dS = Q_enc / ε

- ∯: 폐곡면 전체 적분 (면적분)
- E: 전기장
- dS: 면적 요소 (법선 방향)
- Q_enc: 폐곡면 안의 총 전하량
- ε: 유전율
```

### 간단히 말하면

```
전기장 × 면적 = 내부 전하 / ε

(대칭적인 경우)
E × A = Q / ε
```

---

## 실전 적용: 구 대칭 (점전하)

반지름 r인 구면을 가우스 면으로 설정:

```
E × 4πr² = Q / ε

E = Q / (4πεr²)   ← 쿨롱의 법칙과 동일!
```

> 가우스 법칙으로 쿨롱의 법칙을 유도할 수 있습니다.

---

## 실전 적용: 원통 대칭 (무한 직선 전하)

길이 L, 반지름 r인 원통 가우스 면:

```
E × 2πrL = ρ_L × L / ε

E = ρ_L / (2πεr)   ← 선전하의 전기장 공식과 동일
```

---

## 실전 적용: 무한 평면 전하

평면의 위아래로 나가는 전기장:

```
E × 2A = ρ_S × A / ε

E = ρ_S / (2ε)   ← 면전하의 전기장 공식과 동일
```

---

## 전속(D)과의 관계

전기장 E와 전속밀도 D는 같은 개념의 다른 표현입니다.

```
D = ε × E   [C/m²]

가우스 법칙 (D로 표현):
∯ D · dS = Q_enc
```

| 기호 | 이름 | 단위 |
|------|------|------|
| E | 전기장 세기 | [V/m] |
| D | 전속밀도 | [C/m²] |
| ε | 유전율 | [F/m] |

> D는 매질에 무관, E는 매질에 따라 달라짐

---

## 가우스 면 선택 요령

대칭성을 활용해야 계산이 간단해집니다:

| 전하 형태 | 가우스 면 | 면적 A |
|---------|---------|-------|
| 점전하, 구 | 구면 | 4πr² |
| 무한 직선 | 원통 | 2πrL |
| 무한 평면 | 직육면체(필로우) | 2A |

---

## 시험 포인트

| 항목 | 내용 |
|------|------|
| 가우스 법칙 핵심 | E × A = Q/ε (대칭 경우) |
| D = εE | 전속밀도와 전기장의 관계 |
| 구 가우스 면 | 면적 = 4πr² |
| 원통 가우스 면 | 면적 = 2πrL |

- 가우스 법칙은 복잡한 계산 없이 전기장을 구하는 핵심 도구
- "어떤 폐곡면 안의 전하만 영향을 준다" → 면 밖의 전하는 무관
- D와 E의 구분 — 단위 다름에 주의
