# 05. 도체와 유전체 (Conductor & Dielectric)

## 도체 (Conductor)

### 특징

전자가 **자유롭게 이동**할 수 있는 물질 (금속류)

### 정전기 평형 상태의 도체 성질

전하를 줘도 시간이 지나면 전하가 재분포되어 균형 상태가 됩니다.  
이 상태에서 다음이 성립합니다:

| 성질 | 내용 |
|------|------|
| 내부 전기장 | **E = 0** (도체 내부에 전기장 없음) |
| 내부 전위 | **일정** (등전위체) |
| 전하 분포 | **표면에만** 존재 |
| 표면 전기장 | 표면에 **수직**으로만 존재 |

### 직관적 이해

도체 내부에 전기장이 있으면 전자가 움직이고,  
전자가 움직이면 전기장을 상쇄시켜서 결국 내부 전기장 = 0 이 됩니다.

---

## 도체 표면의 전기장

도체 표면의 전하 면밀도를 σ [C/m²]라 하면:

```
E_표면 = σ / ε   [V/m]
```

- 표면이 뾰족할수록 σ가 크고 전기장이 강함 → **피뢰침 원리**

---

## 정전차폐 (Electrostatic Shielding)

도체 껍질 안은 외부 전기장의 영향을 받지 않음  
→ **패러데이 케이지 (Faraday Cage)** 원리

```
차폐 도체 내부: E = 0   (외부 영향 차단)
```

예: MRI실 전자기파 차폐, 전자기기 금속 케이스

---

## 유전체 (Dielectric)

### 특징

전자가 **자유롭게 이동 못하는** 절연 물질이지만,  
외부 전기장이 걸리면 내부 분자가 **정렬(분극)** 됩니다.

---

## 분극 (Polarization)

유전체에 전기장을 가하면 내부 양전하와 음전하가 미세하게 분리됨

```
외부 E →  [유전체]  → 내부에 작은 쌍극자들이 정렬
```

이 효과로 유전체는 전기장을 **저장하는 능력**이 생깁니다 → 캐패시터 핵심

### 분극의 크기 P (분극벡터)

```
P = ε₀ × χe × E   [C/m²]

- χe: 전기 감수율 (Electric susceptibility)
```

---

## 비유전율과 전속밀도

```
D = ε₀E + P = ε₀εrE = εE

- εr: 비유전율 (재료 고유값)
- ε = ε₀εr: 해당 매질의 유전율
```

비유전율이 클수록:
- 캐패시터 용량 증가
- 전기장이 약해짐 (전기력 감소)

---

## 경계 조건 (두 매질의 경계면)

두 다른 유전체 경계면에서 전기장/전속이 어떻게 바뀌는지:

### 법선(수직) 성분

```
D₁n = D₂n   (전속밀도 법선 성분은 연속)
ε₁E₁n = ε₂E₂n
```

### 접선(수평) 성분

```
E₁t = E₂t   (전기장 접선 성분은 연속)
```

---

## 시험 포인트

| 성질 | 내용 |
|------|------|
| 도체 내부 E | 0 |
| 도체 표면 전하 | 표면에만 분포 |
| 도체 = 등전위체 | 내부 전위 일정 |
| 도체 표면 E | σ/ε, 표면에 수직 |
| D = εE | 전속밀도와 전기장 관계 |
| 경계: D 법선 연속 | ε₁E₁n = ε₂E₂n |
| 경계: E 접선 연속 | E₁t = E₂t |

- 도체의 4가지 성질을 반드시 암기
- 경계 조건은 캐패시터 문제에서 자주 활용
- 피뢰침 원리 = 뾰족한 곳에 전하 집중
