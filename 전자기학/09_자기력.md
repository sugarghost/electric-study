# 09. 자기력 (Magnetic Force)

## 핵심 개념

> 자기장 안에서 **전류가 흐르는 도선** 또는 **운동하는 전하**는 힘을 받습니다.  
> 이것이 전동기(모터)의 원리입니다.

---

## 운동하는 전하에 작용하는 힘 (로렌츠 힘)

자기장 B 안에서 속도 v로 움직이는 전하 q:

```
F = q × v × B × sinθ   [N]

벡터 형태: F = q(v × B)
```

| 기호 | 의미 |
|------|------|
| q | 전하량 [C] |
| v | 전하의 속도 [m/s] |
| B | 자속밀도 [T] |
| θ | v와 B 사이의 각도 |

### 방향: 플레밍의 왼손 법칙

```
왼손을 펴서:
검지 → 자기장(B) 방향
중지 → 전류(I) 또는 전하 이동 방향
엄지 → 힘(F)의 방향
```

> 전류 방향은 양전하의 이동 방향 (= 전자 이동의 반대)

---

## 직선 도선에 작용하는 힘

길이 L인 도선에 전류 I가 흐를 때, 자기장 B 안에서:

```
F = B × I × L × sinθ   [N]

- θ: 전류 방향과 B 방향의 사이 각
- θ = 90°일 때 최대: F = BIL
- θ = 0°일 때 최소: F = 0 (평행)
```

---

## 평행 도선 사이의 힘

두 평행 직선 도선 (거리 r, 전류 I₁, I₂, 길이 L):

```
F = μ₀ × I₁ × I₂ × L / (2πr)   [N]
```

| 전류 방향 | 힘의 방향 |
|---------|---------|
| 같은 방향 | 흡인력 (서로 당김) |
| 반대 방향 | 반발력 (서로 밀어냄) |

> 1암페어(A)의 정의가 이 힘을 기준으로 정해졌습니다.

---

## 자기장 안의 전류 루프 (토크)

직사각형 코일(면적 A)에 전류 I, 자기장 B 안에서:

```
T = N × B × I × A × sinθ   [N·m]

- T: 토크 (회전력)
- N: 코일 권수
- θ: 코일 평면의 법선과 B 사이 각도
```

> 이것이 **전동기(DC Motor)의 작동 원리**

최대 토크: θ = 90° (코일 평면이 B와 평행)  
토크 = 0: θ = 0° (코일 법선이 B와 평행)

---

## 홀 효과 (Hall Effect)

전류가 흐르는 도체에 수직으로 자기장을 가하면  
전하 분리가 생겨 **수직 방향으로 전압(홀 전압)이 발생**

```
VH = (I × B) / (n × e × d)

- n: 전하 캐리어 농도
- e: 전자 전하량
- d: 도체 두께
```

활용: 자기장 측정 센서, 전류 측정

---

## 자기장에서의 전하 운동

균일 자기장 안에서 전하가 운동하면 **원운동** 합니다.

```
원심력 = 자기력
mv²/r = qvB

원운동 반지름: r = mv / (qB)
```

- 질량이 클수록 반지름이 큼
- 자기장이 강할수록 반지름이 작음 (더 강하게 휨)

---

## 시험 포인트

| 공식 | 내용 |
|------|------|
| F = qvBsinθ | 로렌츠 힘 |
| F = BILsinθ | 도선에 작용하는 힘 |
| F = μ₀I₁I₂L/(2πr) | 평행 도선 사이 힘 |
| T = NBIAsinθ | 코일에 작용하는 토크 |

- **플레밍의 왼손 법칙** 방향 판단에 항상 활용
- 같은 방향 전류 → 흡인, 반대 방향 → 반발 (직관과 반대인 것 주의)
- 전동기 원리 = 자기력에 의한 토크
