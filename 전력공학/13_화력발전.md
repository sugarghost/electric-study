# 13. 화력발전

## 핵심 개념
> 연료를 태워 발생한 열로 증기를 만들고, 그 증기로 터빈을 돌려 전기를 만드는 방식을 이해한다.

---

## 화력발전 원리

```
연료 연소
    ↓ (열에너지)
보일러에서 물 가열 → 증기 생성
    ↓ (증기의 열에너지)
터빈 회전 → 기계에너지
    ↓
발전기 회전 → 전기에너지
    ↓
복수기에서 증기 냉각 → 물로 환원 → 순환
```

비유: 주전자에서 물이 끓어 증기가 나오고, 그 증기로 풍차를 돌리는 것을 아주 크게 만든 장치입니다.

---

## 랭킨 사이클 (Rankine Cycle)

화력발전의 기본 열역학 사이클입니다.

```
① 급수펌프: 물(액체)에 압력 가함 (거의 일 없음)
         ↓
② 보일러:  물을 가열 → 증기로 변환 (열 흡수, Q_in)
         ↓
③ 터빈:   고온고압 증기가 팽창하며 일 함 (W_out)
         ↓
④ 복수기: 터빈 통과 후 증기를 냉각수로 식혀 물로 환원 (열 방출, Q_out)
         ↓
         다시 ①로
```

**4개 기기 암기**: 급수펌프 → 보일러 → 터빈 → 복수기

---

## 열효율 (Thermal Efficiency)

투입한 열에너지 중 얼마나 전기로 변환되었는지의 비율입니다.

```
η_th = W_net / Q_in × 100   [%]

W_net: 순 발생 일(출력 전력량)
Q_in: 연료에서 공급한 열량

실제 화력발전소: 약 35~45%
복합화력(CCGT): 약 55~60%

→ 투입 에너지의 60~65%는 복수기 냉각수, 배기가스 등으로 손실!
```

---

## 열소비율 (Heat Rate)

전기 1 kWh를 만드는 데 필요한 열량입니다.

```
열소비율 = Q_in / W_net   [kcal/kWh]

이상적 최솟값: 860 kcal/kWh  (열효율 100%일 때)
∵ 1 kWh = 860 kcal

실제 발전소: 약 2,000~2,500 kcal/kWh
```

**열소비율과 열효율의 관계**
```
열효율 [%] = 860 / 열소비율 × 100

예시: 열소비율 2,000 kcal/kWh
  → 열효율 = 860 / 2,000 × 100 = 43%
```

---

## 보일러 효율

```
η_b = (발생 증기 열량) / (연료 발열량) × 100   [%]

= (증기 엔탈피 - 급수 엔탈피) × 증기량 / (연료량 × 발열량) × 100
```

---

## 효율 향상 기술

### 재열 사이클 (Reheat Cycle)

```
고압 터빈 통과 후 증기를 다시 보일러로 보내 재가열
→ 재가열된 증기를 저압 터빈에 투입
→ 효율 향상 + 습분(물방울) 감소로 터빈 보호
```

```
보일러 → HP 터빈 → [재열기] → LP 터빈 → 복수기
```

### 재생 사이클 (Regenerative Cycle)

```
터빈 중간에서 일부 증기를 추출(bleed)
→ 이 추기 증기로 급수 가열 (급수가열기)
→ 보일러에 들어가는 물의 온도 상승
→ 보일러 열부담 감소 → 효율 향상
```

## 발전소 출력 종류

| 출력 종류 | 설명 |
|---------|------|
| 정격출력 | 장기간 연속 운전 가능한 최대 출력 |
| 첨두출력 | 단시간에만 낼 수 있는 최대 출력 (정격의 110~115%) |
| 소내용 전력 | 발전소 자체 운전에 쓰이는 전력 |
| 송전단 출력 | 정격출력 - 소내용 전력 |

---

## 연료 종류별 특성

| 연료 | 발열량 | 비용 | 환경 | 특징 |
|------|------|------|------|------|
| 석탄 | 중간 | 저렴 | 나쁨 (CO₂, 먼지) | 기저 부하용 |
| 중유(석유) | 높음 | 중간 | 보통 | 점차 감소 추세 |
| LNG (천연가스) | 높음 | 비쌈 | 청정 (CO₂ 적음) | 복합화력에 최적 |

---

## 복합화력 (CCGT: Combined Cycle Gas Turbine)

가스터빈과 증기터빈을 조합한 고효율 발전 방식입니다.

```
LNG → 가스터빈 (1차 발전, ~700°C 고온)
    ↓ 배기가스 (~500°C)
    배열회수보일러(HRSG) → 증기 생성
    ↓
    증기터빈 (2차 발전)

열효율: 55~60% (기존 화력 35~45%보다 우수)
```

---

## 예제

**문제**: 어떤 화력발전소에서 연료 발열량이 6,000 kcal/kg이고, 1kWh 발전에 연료 0.4kg이 소비된다. 열효율은?

```
열소비율 = 0.4 [kg/kWh] × 6,000 [kcal/kg]
         = 2,400 kcal/kWh

열효율 = 860 / 2,400 × 100
       ≈ 35.8%
```
