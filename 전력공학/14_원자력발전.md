# 14. 원자력발전

## 핵심 개념
> 우라늄 원자핵이 분열할 때 나오는 엄청난 열에너지를 이용하여 전기를 만드는 방식을 이해한다.

---

## 원자력발전 원리

```
핵분열 열에너지
    ↓
원자로 내 냉각재 가열
    ↓
증기 발생 (직접 or 증기발생기 통해)
    ↓
터빈 회전 → 발전기 회전 → 전기에너지
```

화력발전과 차이: 연료를 태우는 대신 핵분열 열을 사용한다는 점 외에는 증기 → 터빈 과정이 동일합니다.

---

## 핵분열 (Nuclear Fission)

```
U-235 + n(중성자) → Ba-141 + Kr-92 + 3n + 에너지(약 200 MeV)
```

- U-235(우라늄-235)가 중성자를 흡수하면 불안정해지며 2개의 원소로 분열
- 분열 시 **3개의 새 중성자** 방출 → 다른 U-235와 반응 → 연쇄반응
- 분열 1회당 에너지 ≈ 200 MeV (석탄 연소의 수백만 배!)

비유: 당구공이 쪼개지면서 3개의 공이 튀어나와 다시 다른 공들에 부딪히는 것처럼, 중성자가 계속 분열을 이어갑니다.

---

## 연쇄 반응 (Chain Reaction)

```
1개 핵분열 → 중성자 3개 방출
             ↓
3개 핵분열 → 중성자 9개 방출
             ↓
9개 핵분열 → 중성자 27개 방출
             ↓
제어하지 않으면 폭발적으로 증가! → 제어봉으로 조절
```

**임계 상태**: 중성자 1개가 정확히 1개의 핵분열을 유발 → 안정적 연속 반응

---

## 주요 구성 요소

| 구성 요소 | 역할 | 주요 재료 |
|---------|------|---------|
| 핵연료 | 핵분열 에너지원 | U-235 (농축 우라늄 3~4%) |
| 감속재 | 고속 중성자를 느리게 함 (열중성자로) | 경수, 중수, 흑연 |
| 제어봉 | 중성자 흡수 → 출력 제어 | 붕소(B), 카드뮴(Cd) |
| 냉각재 | 핵분열 열을 외부로 전달 | 경수, 중수, CO₂, 나트륨 |
| 차폐재 | 방사선을 인체로부터 차단 | 콘크리트, 납(Pb) |

### 감속재가 필요한 이유
```
핵분열 직후 중성자: 고속(빠름) → U-235에 잘 흡수 안 됨
감속재 통과 후 중성자: 저속(느림) = 열중성자 → U-235에 잘 흡수됨 → 효율적 연쇄반응
```

### 제어봉 작동 원리
```
출력 높이기: 제어봉을 원자로에서 빼냄 → 중성자 흡수 감소 → 반응 증가
출력 낮추기: 제어봉을 원자로에 삽입 → 중성자 흡수 증가 → 반응 감소
긴급 정지: 제어봉을 완전 삽입 → SCRAM (비상 정지)
```

---

## 원자로 종류

| 종류 | 냉각재 | 감속재 | 핵연료 | 특징 |
|------|------|------|------|------|
| PWR (가압경수로) | 경수 | 경수 | 농축 우라늄(3~4%) | 가장 많이 사용, 한국 주력 |
| BWR (비등경수로) | 경수 | 경수 | 농축 우라늄(3~4%) | 원자로 내 직접 증기 발생 |
| CANDU (중수로) | 중수 | 중수 | 천연 우라늄(0.7%) | 운전 중 연료 교체 가능 |

### PWR (Pressurized Water Reactor) — 가압경수로

한국에서 가장 많이 사용하는 방식입니다.

```
원자로 → [1차 계통, 고압 약 155 atm] → 증기발생기
                                          ↓
                                         [2차 계통] → 증기 → 터빈 → 발전기

핵심: 1차 냉각수는 방사능 오염 → 2차 계통과 분리!
압력이 높아 155℃ 이상에서도 끓지 않음 (고압으로 비등점 상승)
```

### BWR (Boiling Water Reactor) — 비등경수로

```
원자로 내부에서 직접 증기 발생 → 증기가 바로 터빈으로
→ 구조가 단순 (증기발생기 불필요)
→ 단점: 터빈이 방사능에 노출될 수 있음
```

---

## 핵연료 농축도

```
천연 우라늄:  U-235 함량 약 0.7%
             U-238 함량 약 99.3% (핵분열 안 됨)

농축 우라늄 (발전용): U-235 약 3~4%
농축 우라늄 (무기용): U-235 90% 이상

CANDU로는 천연 우라늄(0.7%)도 사용 가능 → 우라늄 자원 절약
```

---

## 원자력발전의 열효율

```
열효율: 약 33% (화력발전 35~45%보다 낮음)

낮은 이유: 
  핵연료 손상 방지를 위해 냉각재 온도를 너무 높이지 못함
  → 증기 온도가 화력보다 낮음
  → 카르노 효율(온도 차이에 비례) 제한
```

---

## 방사성 폐기물 처리

| 구분 | 종류 | 처리 방법 |
|------|------|---------|
| 고준위 | 사용 후 핵연료 | 냉각 후 심층 지하 처분 |
| 중준위·저준위 | 방호복, 공구 등 | 드럼에 밀봉 후 처분 |

---

## 예제

**문제**: 열출력 3,000 MW인 원자력발전소의 열효율이 33%일 때, 전기 출력은?

```
열효율 = 전기 출력 / 열 출력

전기 출력 = 열 출력 × 열효율
          = 3,000 MW × 0.33
          = 990 MW ≈ 1,000 MW (1 GW)

→ 3,000 MW의 열을 넣어 약 1,000 MW의 전기 생산
   나머지 2,000 MW는 냉각수로 방출
```
