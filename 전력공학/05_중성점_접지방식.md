# 05. 중성점 접지방식

## 핵심 개념
> 3상 변압기의 중성점을 어떻게 처리하느냐에 따라 고장 시 동작 방식, 안전성, 보호 계전기 동작이 달라진다.

---

## 중성점이란?

3상 Y결선 변압기의 중앙점을 중성점이라고 합니다.

```
       A상
      /
중성점(N)─── B상
      \
       C상

정상 동작 시: 중성점 전위 = 0V (3상이 균형을 이루므로)
```

---

## 중성점 접지의 목적

1. **이상전압 억제**: 지락 고장 시 건전상 전압이 과도하게 상승하는 것을 억제
2. **보호 계전기 동작 확실**: 지락 전류가 흘러야 계전기가 동작 → 고장 구간 차단
3. **인체 감전 위험 감소**: 지락 전류를 땅으로 흘려 전위 상승 억제

**비유**: 중성점 접지는 마치 건물의 피뢰침과 같습니다. 
- 이상전압이 왔을 때 안전하게 땅으로 흘려보냄

---

## 접지 방식의 종류

### 1. 직접 접지 (Solid Grounding)

```
중성점 N ─────────── 지면 (땅)
          (직접 연결)
```

- **지락전류**: 매우 큼 (직접 연결이므로 저항이 없음)
- **건전상 전압 상승**: 작음 (중성점이 고정되므로)
- **계전기 동작**: 확실 (큰 지락전류 → 즉시 감지)
- **사용처**: 154kV 이상 초고압 계통
- **장점**: 이상전압 억제 우수, 계전기 동작 확실
- **단점**: 큰 지락전류 → 기기 손상 위험, 통신선 유도장해

### 2. 저항 접지 (Resistance Grounding)

```
중성점 N ──/\/\/── 지면
           R 삽입
```

- **지락전류**: 제한됨 (저항이 전류를 제한)
- **건전상 전압 상승**: 중간
- **계전기 동작**: 비교적 확실
- **사용처**: 22.9kV 배전 계통
- **장점**: 지락전류 제한 → 기기 보호, 과도 안정도 우수
- **단점**: 직접접지보다 이상전압 억제 약함

### 3. 소호리액터 접지 (Petersen Coil)

```
중성점 N ──UUUUU── 지면
           L (리액터) 삽입
```

- **원리**: 선로의 정전용량(C)에 의한 지락전류를 리액터(L)로 상쇄
  ```
  Ic (정전용량 전류, 진상) + IL (리액터 전류, 지상) = 0
  → 지락전류 ≈ 0
  ```
- **지락전류**: 최소
- **1선 지락 시**: 계속 운전 가능! (전류가 거의 흐르지 않으므로)
- **계전기 동작**: 불확실 (전류가 너무 작아서)
- **사용처**: 특수 목적, 무정전이 중요한 경우

### 4. 비접지 (Ungrounded)

```
중성점 N ─── (연결 없음, 개방)
```

- **지락전류**: 작음 (정전용량에 의한 충전전류만)
- **건전상 전압 상승**: 큼 (√3배까지 상승 가능)
- **1선 지락 시**: 정전 없이 계속 운전 가능 (단, 전압 상승 주의)
- **사용처**: 저압 소규모 계통

---

## 접지 방식 비교표

| 방식 | 지락전류 | 건전상 전압 상승 | 계전기 동작 | 주 사용처 |
|------|----------|-----------------|-------------|-----------|
| 직접 접지 | 가장 큼 | 가장 작음 (억제) | 가장 확실 | 154kV 이상 |
| 저항 접지 | 중간 | 중간 | 확실 | 22.9kV |
| 소호리액터 접지 | 가장 작음 | 큼 | 불확실 | 특수 경우 |
| 비접지 | 작음 | 가장 큼 (√3배) | 어려움 | 소규모 저압 |

---

## 유도장해 (Induction Hazard)

고압 송전선 근처에 통신선이 있으면, 전기적으로 영향을 받습니다.

### 정전유도 (Capacitive Induction)

```
고압 전선
   |
   C (정전결합)
   |
통신선 → 유도 전압 발생
```

- **원인**: 고압선과 통신선 사이의 정전용량(C)
- **결과**: 통신선에 이상 전압 유도
- **위험**: 정전기적 충격

### 전자유도 (Electromagnetic Induction)

```
고압 전선에 불평형 전류(영상전류) 흐름
         ↓
주변에 자속 발생
         ↓
통신선을 지나는 자속 변화
         ↓
통신선에 유도 기전력 발생
```

전자유도 전압 공식:
```
Em = jωM × I × l   [V]

ω: 각주파수 (= 2πf)
M: 상호 인덕턴스 [H/km]
I: 불평형 전류 (영상전류) [A]
l: 병행 길이 [km]
```

### 유도장해 방지 대책

| 대책 | 방법 | 효과 |
|------|------|------|
| **차폐선** 설치 | 접지된 차폐도선 설치 | 정전유도, 전자유도 감소 |
| **연가 (Transposition)** | 전선의 위치를 주기적으로 바꿈 | 불평형 제거 → 전자유도 감소 |
| 이격 거리 증가 | 통신선과 거리를 멀리 함 | 결합 계수 감소 |
| 케이블화 | 통신선을 지하 케이블로 교체 | 근본적 해결 |
| 중성점 접지 개선 | 소호리액터 접지로 변경 | 영상전류 감소 |

### 연가 (Transposition) 설명

```
구간 1:  A상-위, B상-중, C상-아래
구간 2:  C상-위, A상-중, B상-아래  (위치 교체!)
구간 3:  B상-위, C상-중, A상-아래  (다시 교체!)

→ 전체 평균적으로 각 상의 선로정수가 같아짐
→ 불평형 해소
```

---

## 예제

**문제**: 22.9kV 배전 계통에서 비접지 방식으로 운전 중 1선 지락 고장이 발생했다. 건전상의 최대 전압 상승은?

```
정상 운전 시 상전압: Vp = 22,900/√3 = 13,220 V

비접지 1선 지락 시: 건전상 전압 = √3 × Vp
= √3 × 13,220 ≈ 22,900 V (선간 전압과 같아짐!)

→ 건전상 전압이 정상 상전압의 √3배로 상승
→ 절연 파괴 위험!
```
