# 11. 배전

## 핵심 개념
> 변전소에서 각 가정·공장까지 전기를 나누어 공급하는 방식과, 전압강하·손실을 줄이는 방법을 이해한다.

---

## 배전이란?

송전선으로 전달된 고압의 전기를 수용가(가정, 공장 등)가 사용할 수 있는 전압으로 낮추어 공급하는 과정입니다.

```
발전소 → 송전(154~345kV) → 1차 변전소 → 배전(22.9kV) → 주상변압기 → 수용가(220V/380V)
```

---

## 배전 전압 (한국 표준)

```
배전 전압: 22.9 kV (3상 4선식, 다중 접지 방식)

3상(R, S, T) + 중성선(N) = 4선식
선간전압: 22.9 kV
상전압: 22.9 / √3 ≈ 13.2 kV
```

**다중 접지**: 중성선을 여러 지점에서 접지 → 지락전류 경로 확보, 안전성 향상

---

## 배전 방식 비교

| 방식 | 구성 | 장점 | 단점 | 용도 |
|------|------|------|------|------|
| 방사상식 | 1개 급전점에서 나뭇가지처럼 뻗음 | 간단, 경제적 | 신뢰성 낮음 (1개소 고장 시 정전) | 농촌, 교외 |
| 루프식 (환상식) | 고리(링) 형태로 연결 | 신뢰성 향상 (양방향 공급) | 복잡, 비용 증가 | 도시 |
| 네트워크식 (격자식) | 격자(그물망) 형태 | 최고 신뢰성 | 매우 복잡, 고비용 | 대도시 밀집지역 |

## 배전선 전압강하

전선에 저항과 리액턴스가 있어, 전류가 흐르면 끝단 전압이 낮아집니다.

### 단상 2선식

```
전압강하 e = 2I(R cosφ + X sinφ)   [V]

2배인 이유: 왕복 2선 모두 저항이 있으므로
R: 전선 저항 [Ω]
X: 전선 리액턴스 [Ω]
I: 전류 [A]
cosφ: 역률, sinφ = √(1 - cos²φ)
```

### 3상 3선식

```
전압강하 e = √3 × I(R cosφ + X sinφ)   [V]

√3 배인 이유: 3상 선로의 기하학적 관계
```

### 전압 강하율

```
전압 강하율 ε = e / Vs × 100   [%]

Vs: 송전단 전압
→ 보통 5% 이하로 관리
```

---

## 전력 손실

```
선로 전력 손실 P_L = I² × R   [W]

3상의 경우: P_L = 3 × I² × R
```

손실은 전류의 **제곱에 비례** → 전류를 반으로 줄이면 손실은 1/4로 감소!

---

## 전력 손실 경감 방법

### 1) 역률 개선 (콘덴서 설치)

**원리**: 역률이 낮으면 같은 유효전력을 보내는 데 더 많은 전류 필요
→ 콘덴서로 무효전력 보상 → 전류 감소 → 손실 감소

```
역률 개선 콘덴서 용량:

Qc = P(tanφ₁ - tanφ₂)   [kVar]

P: 부하의 유효전력 [kW]
φ₁: 개선 전 역률각 (cosφ₁이 나쁜 상태)
φ₂: 개선 후 역률각 (cosφ₂가 좋은 상태)
```

비유: 맥주잔에서 거품(무효전력)을 빼고 맥주(유효전력)만 채우는 과정

### 2) 분기점 이동

부하 중심에 급전점을 두면 선로 전류 분포가 최적화되어 손실 감소

### 3) 전압 승압

```
같은 전력 P = √3 × V × I × cosφ에서
V를 2배 올리면 I는 1/2로 감소
전력 손실 P_L = 3I²R → (1/2)² = 1/4로 감소
```

→ 승압이 손실 경감에 매우 효과적!

### 4) 선로 굵기 증가

굵기 증가 → 저항 R 감소 → 손실 감소 (비용 증가 vs 손실 절감 경제성 검토 필요)

## 중성선의 역할

3상 4선식 배전에서 중성선은 두 가지 중요한 역할을 합니다.

1. **불평형 부하 전류의 통로**: 단상 부하가 불균등할 때 전류가 흐르는 귀로
2. **영상전류의 통로**: 지락사고 시 고장전류가 귀환하는 경로

```
3상 평형 시: 중성선 전류 = 0
불평형 시: 중성선 전류 = Ia + Ib + Ic ≠ 0
```

## 예제

**문제**: 단상 2선식 선로에서 부하 전류 50A, 역률 0.8, 선로 저항 0.5Ω, 리액턴스 0.3Ω일 때 전압강하는?

```
주어진 값:
  I = 50A
  cosφ = 0.8  →  sinφ = 0.6 (∵ sin²φ + cos²φ = 1)
  R = 0.5Ω, X = 0.3Ω

전압강하 e = 2I(R cosφ + X sinφ)
           = 2 × 50 × (0.5 × 0.8 + 0.3 × 0.6)
           = 100 × (0.4 + 0.18)
           = 100 × 0.58
           = 58 V
```

**문제**: 역률 0.6인 부하(P=100kW)를 역률 0.9로 개선하는 데 필요한 콘덴서 용량은?

```
cosφ₁ = 0.6  →  tanφ₁ = sinφ₁/cosφ₁ = 0.8/0.6 = 1.333
cosφ₂ = 0.9  →  tanφ₂ = sinφ₂/cosφ₂ = 0.436/0.9 = 0.484

Qc = P(tanφ₁ - tanφ₂)
   = 100 × (1.333 - 0.484)
   = 100 × 0.849
   ≈ 84.9 kVar
```
