# 07. 전력계통 안정도

## 핵심 개념
> 발전기가 전력 계통에 동기(같은 주파수와 위상)를 유지하면서 운전할 수 있는 능력을 안정도라 한다. 상차각 δ가 핵심이다.

---

## 안정도란?

전력 계통에서 발전기는 모두 같은 주파수(60Hz)로 동기 운전해야 합니다.

```
정상 운전: 모든 발전기 60Hz로 동기화 → 안정
고장 발생: 발전기가 갑자기 가속/감속 → 동기에서 벗어날 수 있음
```

안정도 = "이런 상황에서도 동기를 유지할 수 있는가?"

**비유**: 여러 명이 함께 노 젓는 배를 생각하세요.
- 정상: 모두 같은 박자로 노를 저음
- 고장: 갑자기 한 명이 빨리 젓기 시작 → 박자가 틀어짐
- 안정: 다시 같은 박자로 돌아오면 안정
- 불안정: 박자가 더 벌어져서 배가 제어 불능이 되면 불안정

---

## 안정도의 종류

### 1. 정태 안정도 (Steady-State Stability)

```
소규모의 완만한 변동에 대한 안정성
예: 부하가 조금씩 증가하는 경우
```

- **기준**: 소규모 변동 후 원래 동작점으로 돌아오는가
- **판별**: 전력-상차각 곡선의 기울기 (dP/dδ > 0이면 정태 안정)

### 2. 과도 안정도 (Transient Stability)

```
갑작스럽고 큰 변동에 대한 안정성
예: 단락 고장, 발전기 탈락 등
```

- **기준**: 큰 고장 후에도 동기를 회복하는가
- **판별**: 등면적법 (Equal Area Criterion)

### 3. 동태 안정도 (Dynamic Stability)

```
자동제어 시스템(AVR, 조속기 등)을 포함한 안정성
```

---

## 동기 발전기의 전력 전달 공식

### 기본 공식

```
P = (E × V)/X × sin δ   [W]

E: 발전기 내부 기전력 [V]
V: 수전단 전압 [V]
X: 회선의 리액턴스 [Ω] (저항 R 무시)
δ: 상차각 (E와 V 사이의 위상각) [°]
```

**공식의 의미**:
- P는 sinδ에 비례 → δ가 커지면 전력이 증가하다가 감소
- δ = 90°일 때 P가 최대

### 최대 전달 전력

```
P_max = (E × V)/X   [W]

(sin δ = 1, 즉 δ = 90°일 때)
```

---

## 전력-상차각 (P-δ) 특성 곡선

```
P(전력)
│
│        *
P_max ─ * │ *
│      * │   *
│    *   │     *
│  *     │       *
│ *      │         *
└────────┼────────────── δ(상차각)
        90°          180°
```

```
δ < 90°: 정태 안정 구간 (dP/dδ > 0)
δ = 90°: 최대 전력 전달점 (안정 한계)
δ > 90°: 불안정 구간 (δ가 커질수록 P가 감소)
```

**직관**: δ=90°까지는 더 기울일수록 더 빠르게 달릴 수 있지만, 90°를 넘으면 오히려 넘어집니다.

## 등면적법 (Equal Area Criterion)

과도 안정도를 시각적으로 판별하는 방법입니다.

### 개념

```
가속 에너지 = 감속 에너지 → 안정
가속 에너지 > 감속 에너지 → 불안정
```

**가속 에너지**: 고장 시 기계 입력 > 전기 출력 → 발전기 가속
**감속 에너지**: 고장 제거 후 전기 출력 > 기계 입력 → 발전기 감속

### P-δ 곡선에서 면적으로 판별

```
P(전력)
│
│     ┌─고장 전 곡선────────
Pm ───┼──────────*─────────
│    ↑가속 면적  ↑감속 면적
│    │고장 중 곡선│
Pa ──┼─────────┼───────────── P_기계 입력
│    │          │
└────┼──────────┼──── δ
    δ0         δ1
    (고장 전)   (고장 제거)

가속 면적(색칠) ≤ 감속 가능 면적이면 → 안정
```

## 안정도 향상 방법

### 1. 리액턴스 X 감소 → P_max 증가

```
P_max = EV/X
X 작을수록 P_max 증가 → 더 큰 전력 전달 가능 → 안정도 향상
```

**방법**:
- 복도체 사용 (인덕턴스 감소)
- 병행 선로 (두 회선 병렬 운전)
- 직렬 콘덴서 설치 (용량성 리액턴스로 유도성 상쇄)

### 2. 고속 고장 제거

```
고장이 빨리 제거될수록 → 가속 에너지 적음 → 안정
```

**방법**: 고속 차단기 사용 (3사이클 → 2사이클 이내 차단)

### 3. 자동 재폐로 (Auto Reclosing)

```
가공선 지락의 70~80%는 순시성 (번개, 일시적 단락)
→ 차단 후 잠시 뒤 재투입 → 정상 복귀 가능
```

**장점**: 고장 회선을 빨리 재투입 → 안정도 개선

### 4. 발전기 단락비 증가

```
단락비 Kcc = I_단락/I_정격 = 1/(%Xs)

단락비 클수록 → 동기 임피던스 작음 → 안정도 우수
(단, 발전기가 크고 비쌈)
```

### 5. 신속한 여자 제어 (AVR: Automatic Voltage Regulator)

```
고장 시 발전기 단자 전압 변동 → AVR이 즉시 여자 전류 조정
→ 내부 기전력 E 유지 → 전력 전달 능력 유지
```

## 동요 방정식 (Swing Equation)

발전기의 회전 운동 방정식:

```
M × d²δ/dt² = Pa = Pm - Pe

M: 관성 정수
δ: 상차각
Pa: 가속 전력 = 기계 입력(Pm) - 전기 출력(Pe)
```

- Pa > 0: 발전기 가속 (δ 증가)
- Pa < 0: 발전기 감속 (δ 감소)
- Pa = 0: 안정 평형점

## 예제

**문제**: 발전기 내부 기전력 E = 1.2pu, 수전단 전압 V = 1.0pu, 리액턴스 X = 0.3pu일 때:
(1) 최대 전달 전력을 구하시오.
(2) P = 3.0pu을 전달할 때의 상차각 δ를 구하시오.

```
(1) 최대 전달 전력:
P_max = EV/X = 1.2 × 1.0/0.3 = 4.0 pu

(2) 상차각:
P = (EV/X) × sin δ
3.0 = 4.0 × sin δ
sin δ = 3.0/4.0 = 0.75
δ = arcsin(0.75) ≈ 48.6°

(48.6° < 90° → 정태 안정 구간에서 운전 중)
```

## 시험 포인트

| 공식/개념 | 내용 |
|-----------|------|
| 전달 전력 | `P = (EV/X) × sin δ` |
| 최대 전달 전력 | `P_max = EV/X` (δ=90°) |
| 정태 안정 조건 | `δ < 90°` (dP/dδ > 0) |
| 등면적법 | 가속 면적 ≤ 감속 가능 면적 → 과도 안정 |
| 안정도 향상 | X 감소, 고속 차단, 자동재폐로, AVR |
| 복도체 효과 | 인덕턴스 L↓ → X↓ → P_max↑ → 안정도↑ |
| 직렬 콘덴서 | 용량성으로 선로 유도성 X 상쇄 → P_max↑ |
