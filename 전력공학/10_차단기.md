# 10. 차단기

## 핵심 개념
> 차단기는 고장전류(단락전류)를 안전하게 끊어 전력계통을 보호하는 핵심 기기이며, 소호 매질에 따라 종류가 다양하다.

---

## 차단기란?

전력계통에서 고장이 발생했을 때 **대전류를 차단하는 개폐기기**입니다.

- 일반 스위치: 정상 부하 전류 개폐
- **차단기**: 수십~수백 kA의 단락전류도 차단 가능

비유: 가정용 누전차단기(두꺼비집)의 초고성능 버전이 변전소의 차단기입니다.

---

## 차단기 성능 지표

### 차단 용량 (Breaking Capacity)

차단기가 차단할 수 있는 최대 전력의 크기입니다.

```
차단 용량 = √3 × 정격전압 [kV] × 정격차단전류 [kA]   [MVA]

예시:
  정격전압 = 154 kV, 정격차단전류 = 20 kA
  차단 용량 = √3 × 154 × 20 ≈ 5,335 MVA
```

### 정격 차단 시간

```
전체 차단 시간 = 동작 시간 + 아크 소호 시간

고속 차단기: 2~3 사이클 이내 (약 40~60 ms, 60Hz 기준)
→ 빠를수록 계통 안정도 향상
```

---

## 소호(消弧)란?

차단기가 전류를 끊을 때 접점 사이에 **아크(Arc, 전기 불꽃)**가 발생합니다.
이 아크를 빠르게 꺼야 차단이 완료됩니다.
아크를 끄는 행위를 **소호**라고 하며, 소호 매질의 종류에 따라 차단기 종류가 결정됩니다.

비유: 용접할 때 불꽃이 튀는 것처럼, 수십만 V의 전류를 끊으면 엄청난 불꽃이 발생합니다.

---

## 주요 차단기 종류

| 종류 | 소호 매질 | 특징 | 현황 |
|------|---------|------|------|
| OCB (유입차단기) | 절연유 | 구형, 화재 위험 | 거의 사용 안 함 |
| ABB (공기차단기) | 압축공기 | 대용량 가능, 소음 큼 | 점차 감소 |
| VCB (진공차단기) | 진공 | 소형·경량, 보수 쉬움 | 중저압(~36kV) 주력 |
| GCB (SF₆차단기) | SF₆ 가스 | 최고 성능 | 고압(66kV 이상) 주력 |

---

## 각 차단기 상세 설명

### OCB (Oil Circuit Breaker, 유입차단기)
- 절연유 속에서 아크를 발생시켜 소호
- 오래된 방식, 기름 누출·화재 위험으로 신규 설치 기피

### VCB (Vacuum Circuit Breaker, 진공차단기)
- 진공 상태에서는 아크가 매우 빨리 소멸
- 작고 가볍고 유지보수가 쉬움
- 22.9kV 배전용 수전설비에서 가장 많이 사용

### GCB (SF₆ Circuit Breaker, 가스차단기)

가장 성능이 우수한 차단기입니다.

```
SF₆ 가스 특성:
  절연 능력 = 공기의 약 2.5~3배
  아크 소호 능력 = 공기의 약 100배
```

- **GIS(Gas Insulated Switchgear)**: SF₆ 가스를 봉입한 완전 밀폐형 변전소 설비
  - 크기가 매우 작아짐 (기존 AIS 대비 1/10 수준)
  - 도심 지하 변전소에 적합
- **단점**: SF₆는 강력한 온실가스 (CO₂의 23,900배), 친환경 문제 대두

---

## 차단기 동작 시간 상세

```
투입(Close) 지령 → [투입 시간] → 접점 닫힘

트립(Trip) 지령 → [동작 시간] → 접점 열림 → [아크 소호 시간] → 차단 완료

전체 차단 시간 = 동작 시간 + 아크 소호 시간
               = 약 1~3 사이클 (16~50 ms, 60Hz)
```

---

## 재폐로 (Auto-Reclosing)

고장 후 차단기가 자동으로 다시 투입되는 기능입니다.

**배경**: 송전선 고장의 약 80%는 낙뢰에 의한 **순간 고장**
→ 차단 후 잠시 기다리면 아크가 사라지고 복구 가능

```
고장 발생 → 차단기 개방 → 무전압 시간 (약 0.5~1.0초)
→ 자동 재투입 → 성공이면 정상 운전 재개
              → 실패(영구 고장)이면 다시 차단 → 잠금
```

- **영구 고장**: 전선 절단, 철탑 도괴 등 → 재폐로 실패 → 잠금
- **3상 재폐로 vs 단상 재폐로**: 3상 동시 or 고장 상만 재폐로
- **효과**: 전력 공급 신뢰성 크게 향상

---

## 예제

**문제**: 정격전압 154kV, 정격차단전류 31.5kA인 차단기의 차단 용량은?

```
차단 용량 = √3 × 154 × 31.5
          = 1.732 × 154 × 31.5
          = 1.732 × 4,851
          ≈ 8,402 MVA
          ≈ 8,400 MVA
```

## 시험 포인트

| 공식/항목 | 내용 |
|---------|------|
| 차단 용량 공식 | √3 × 정격전압 × 정격차단전류 [MVA] |
| 소호 매질별 특징 | OCB(유), ABB(공기), VCB(진공), GCB(SF₆) |
| SF₆ 절연 능력 | 공기의 2.5~3배 |
| VCB 주사용처 | 중저압(~36kV) |
| GCB(SF₆) 주사용처 | 고압(66kV 이상), GIS에 사용 |
| 재폐로 목적 | 순간 고장(낙뢰) 후 자동 재투입 → 신뢰성 향상 |
| 고속 차단 의미 | 2~3사이클 이내 차단 → 계통 안정도 향상 |
