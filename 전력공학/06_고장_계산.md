# 06. 고장 계산

## 핵심 개념
> 전기기사 시험에서 가장 많이 출제되는 파트. 퍼센트 임피던스(%Z)를 이용하여 단락전류와 단락용량을 계산한다.

---

## 왜 퍼센트 임피던스를 쓸까?

실제 임피던스(Ω)로 계산하면 전압 단계마다 값이 달라져 복잡합니다.

```
예: 변압기 1차측 154kV에서 임피던스 100Ω
   → 2차측 22.9kV에서 환산하면 다른 값!

해결책: %Z = 정격 조건 대비 비율로 표현
   → 어느 전압 단계에서나 같은 % 값 사용 가능
```

**비유**: 키가 큰 사람이나 작은 사람이나 "키가 1% 줄었다"고 표현하면 서로 다른 절대값이지만 비율로 비교가 쉽습니다.

---

## 퍼센트 임피던스 (%Z)

### 기본 공식

$$\%Z = \frac{Z \cdot I_n}{V_n} \times 100 \quad [\%]$$

또는

$$\%Z = \frac{Z \cdot P_n}{V_n^2} \times 100 \quad [\%]$$

- $Z$: 임피던스 [Ω]
- $I_n$: 정격전류 [A]
- $V_n$: 정격전압 [V] (선간전압)
- $P_n$: 정격용량 [VA]

**두 공식이 같은 이유**:

$I_n = \dfrac{P_n}{\sqrt{3} \cdot V_n}$ (3상) 이므로 $\dfrac{Z \cdot I_n}{V_n} = \dfrac{Z \cdot P_n}{\sqrt{3} \cdot V_n^2} \times \sqrt{3} = \dfrac{Z \cdot P_n}{V_n^2}$

### 기준 용량 환산

여러 기기가 있을 때 기준 용량을 통일해야 합니다:

$$\%Z' = \%Z \times \frac{P_{\text{base(새)}}}{P_{\text{base(기존)}}}$$

- $P_{\text{base(새)}}$: 새로운 기준 용량 [VA]
- $P_{\text{base(기존)}}$: 원래 기준 용량 [VA]

**예시**: 10MVA에서 %Z=5%인 변압기를 100MVA 기준으로 환산 → $\%Z' = 5 \times \dfrac{100}{10} = 50\%$

---

## 단락 전류 계산

### 기본 공식

$$I_s = I_n \times \frac{100}{\%Z} \quad [A]$$

또는

$$I_s = \frac{P_n}{\sqrt{3} \cdot V_n \cdot \frac{\%Z}{100}} \quad [A]$$

### 직관적 이해

```
정상 시: 전류 = In, 임피던스 = Z_total
단락 시: 전압 = 정격전압, 임피던스 = 매우 작음 → 전류 폭증
```

**비유**: 수도꼭지를 완전히 열면(단락) 파이프 저항만큼만 흐름 제한 → 엄청난 유량 발생

---

## 단락 용량 (고장 용량)

$$P_s = \sqrt{3} \cdot V_n \cdot I_s = P_n \times \frac{100}{\%Z} \quad [VA]$$

단락 용량이 클수록:
- 사고 시 전류가 더 큼
- 기기 파손 위험 증가
- 차단기 차단 용량을 크게 해야 함

---

## 기준 용량을 사용한 계산 절차

### 계산 단계

**1단계**: 기준 용량 Pbase 설정 (보통 가장 큰 변압기 용량으로)

**2단계**: 각 기기의 %Z를 기준 용량으로 환산
```
%Z_환산 = %Z_기존 × (Pbase/P기존)
```

**3단계**: 전원에서 고장점까지 직렬 임피던스 합산
```
%Z_합계 = %Z1 + %Z2 + %Z3 + ...
```

**4단계**: 단락 전류/용량 계산
```
Is = In_base × (100/%Z_합계)
Ps = Pbase × (100/%Z_합계)
```

## 고장의 종류와 특징

| 고장 종류 | 심볼 | 고장 전류 크기 | 발생 빈도 | 해석법 |
|-----------|------|----------------|-----------|--------|
| 3상 단락 | 3LS | 가장 큼 | 드묾 (약 5%) | 정상분만 |
| 2선 지락 | 2LG | 큼 | - | 정상+역상+영상 |
| 선간 단락 | LL | 중간 | - | 정상+역상 |
| **1선 지락** | **1LG** | **중간~큼** | **가장 많음 (약 70%)** | 정상+역상+영상 |

**기억법**: "3상 단락이 전류는 최대지만, 실제 사고는 1선 지락이 제일 많다"

---

## 대칭좌표법 (Symmetrical Components)

### 개념

3상 불평형 전압/전류를 3가지 대칭 성분으로 분해합니다.

```
불평형 3상 = 정상분 + 역상분 + 영상분

정상분(Positive): 정상적인 상순 (A→B→C)
역상분(Negative): 역순 (A→C→B) — 역회전
영상분(Zero):     3상이 모두 같은 방향 — 편향된 성분
```

**비유**: 회전하는 팽이를 생각해보세요.
- 정상분 = 팽이가 바르게 회전
- 역상분 = 팽이가 반대 방향으로 회전
- 영상분 = 팽이가 기울어진 채로 회전

### 연산자 a (120° 위상 회전)

```
a = e^(j120°) = -0.5 + j0.866
a² = e^(j240°) = -0.5 - j0.866
a³ = 1
```

### 대칭분 변환

```
불평형 → 대칭분:
I₀ = (1/3)(Ia + Ib + Ic)         (영상분)
I₁ = (1/3)(Ia + a·Ib + a²·Ic)   (정상분)
I₂ = (1/3)(Ia + a²·Ib + a·Ic)   (역상분)

대칭분 → 불평형:
Ia = I₀ + I₁ + I₂
Ib = I₀ + a²I₁ + aI₂
Ic = I₀ + aI₁ + a²I₂
```

### 고장 유형별 해석

#### 3상 단락 (가장 간단)
```
대칭 고장 → 역상분, 영상분 = 0
정상분만으로 해석

I_단락 = V/(Z₁)  (Z₁: 정상 임피던스)
```

#### 1선 지락 (a상 지락)

경계 조건: $I_b = 0$, $I_c = 0$, $V_a = 0$ (지락점 전압=0)

결과: $I_0 = I_1 = I_2$

$$I_a = 3I_1 = \frac{3V}{Z_0 + Z_1 + Z_2}$$

- $Z_0$: 영상 임피던스, $Z_1$: 정상 임피던스, $Z_2$: 역상 임피던스

#### 선간 단락 (b, c상 단락)
```
경계 조건: Ia = 0, Ib = -Ic, Vb = Vc

결과: I₀ = 0 (영상분 없음)
단락전류: Ib = -Ic = -j√3 × V/(Z₁ + Z₂)
```

## 예제 1: %Z 계산

**문제**: 정격 10MVA, 22.9kV 변압기의 %Z = 5%이다. 이 변압기의 실제 임피던스를 구하시오.

$$\%Z = \frac{Z \cdot P_n}{V_n^2} \times 100$$

$$5 = \frac{Z \times 10 \times 10^6}{22{,}900^2} \times 100$$

$$Z = \frac{5 \times 22{,}900^2}{100 \times 10 \times 10^6} \approx 2.62 \; [\Omega]$$

---

## 예제 2: 단락 전류 계산

**문제**: 기준 용량 100MVA, 고장점에서 합산 %Z = 25%일 때 단락용량과 단락전류를 구하시오. (기준전압: 154kV)

단락용량:
$$P_s = P_{\text{base}} \times \frac{100}{\%Z} = 100 \times \frac{100}{25} = 400 \; [MVA]$$

기준전류:
$$I_n = \frac{P_{\text{base}}}{\sqrt{3} \cdot V_n} = \frac{100 \times 10^6}{\sqrt{3} \times 154 \times 10^3} \approx 375 \; [A]$$

단락전류:
$$I_s = I_n \times \frac{100}{\%Z} = 375 \times \frac{100}{25} = 1{,}500 \; [A]$$

## 시험 포인트

| 공식 | 내용 |
|------|------|
| %Z 기본 | `%Z = Z × Pn/Vn² × 100` |
| %Z 환산 | `%Z' = %Z × (Pbase_새/Pbase_기존)` |
| 단락전류 | `Is = In × (100/%Z)` |
| 단락용량 | `Ps = Pbase × (100/%Z)` |
| 1선 지락전류 | `Ia = 3V/(Z₀ + Z₁ + Z₂)` |
| 3상 단락 | 정상분만 사용 (가장 간단) |
| 1선 지락 | 영상+역상+정상 분 모두 사용 |
| 영상전류 조건 | `I₀ = (1/3)(Ia + Ib + Ic)` |

> **자주 출제**: %Z 계산, 기준용량 환산, 단락전류/단락용량 계산, 고장 종류별 빈도 (1선 지락이 가장 많음 ~70%), 3상 단락이 가장 큰 전류
