# 12. 수력발전

## 핵심 개념
> 높은 곳에 있는 물이 낮은 곳으로 떨어지는 위치에너지를 이용하여 전기를 만드는 발전 방식을 이해한다.

---

## 수력발전 원리

```
물의 위치에너지
    ↓
수차(터빈) 회전 → 운동에너지
    ↓
발전기 회전 → 전기에너지
```

비유: 물레방아를 생각하면 됩니다. 물이 떨어지는 힘으로 바퀴가 돌고, 그 바퀴에 발전기를 연결한 것이 수력발전입니다.

---

## 수력 출력 공식 (가장 중요!)

```
P = 9.8 × Q × H × η   [kW]

P: 발전 출력 [kW]
Q: 유량 [m³/s] — 초당 흐르는 물의 양
H: 유효낙차 [m] — 물이 실제로 이용하는 높이 차
η: 종합 효율 (보통 0.85~0.92)
9.8: 물의 비중량 (9.8 kN/m³) = 중력가속도(9.8 m/s²) × 물의 밀도(1,000 kg/m³)
```

**단위 확인**
```
P = 9.8 [kN/m³] × Q [m³/s] × H [m] × η
  = kN·m/s × η
  = kW × η      ✓
```

---

## 낙차의 종류

```
총낙차(H_gross) = 상류 수위 - 하류 수위

손실낙차(h_loss) = 수로, 수압관 등에서 마찰에 의한 손실

유효낙차(H) = 총낙차 - 손실낙차
```

비유: 산 꼭대기(댐)에서 산 아래(발전소)까지의 높이가 총낙차, 중간에 파이프 마찰 등으로 잃는 에너지를 빼면 유효낙차입니다.

---

## 수차 종류

| 수차 | 적합 낙차 | 작동 원리 | 대표 특징 |
|------|---------|---------|---------|
| 펠톤(Pelton) | 고낙차 (>300m) | 충격식 (물 분사로 버킷 타격) | 물이 버킷을 때리는 충격력 사용 |
| 프란시스(Francis) | 중낙차 (30~300m) | 반동식 (물의 압력+속도) | 세계에서 가장 많이 사용 |
| 카플란(Kaplan) | 저낙차 (<30m) | 반동식 + 가변 날개 | 대유량 하천·조류에 적합 |

### 펠톤 수차
```
적용: 낙차가 매우 큰 산악 지역
원리: 노즐에서 물을 고속으로 분사 → 버킷(양동이 모양) 타격 → 회전
특징: 물이 수차에 닿기 전 이미 대기압(수압관 내)
```

### 프란시스 수차
```
적용: 일반적인 댐 수력발전
원리: 물이 러너(회전날개) 안에서 안쪽으로 흐르며 반동으로 회전
특징: 효율 최고(90% 이상), 출력 범위 넓음
```

### 카플란 수차
```
적용: 낙차 낮고 유량 많은 하천
원리: 프로펠러(배 스크류)처럼 생긴 날개가 물의 흐름에 반응
특징: 날개 각도를 변환(가변익) → 유량 변화에 적응
```

---

## 비속도 (Specific Speed, Ns)

수차의 크기와 특성을 하나의 수로 나타낸 지수입니다.

```
Ns = N × √P / H^(5/4)

N: 회전수 [rpm]
P: 출력 [kW] (또는 마력 [PS])
H: 유효낙차 [m]
```

| 수차 종류 | 비속도 범위 |
|---------|-----------|
| 펠톤 | 낮음 (5~70) |
| 프란시스 | 중간 (60~400) |
| 카플란 | 높음 (300~900) |

비유: 비속도는 수차의 "DNA"처럼, 주어진 낙차와 유량에 어떤 수차가 맞는지 결정하는 지표입니다.

---

## 양수발전 (Pumped Storage)

전력을 저장하는 방법 중 하나입니다.

```
심야 (전력 남을 때): 잉여 전력으로 펌프 가동 → 하부 저수지의 물을 상부로 올림
                                                 (전기에너지 → 위치에너지 저장)

주간 피크 (전력 부족할 때): 상부 물을 방류하여 발전
                            (위치에너지 → 전기에너지 출력)
```

**효과**: 전력 수요의 부하 평준화 (Load Leveling)
- 야간 저가 전력 흡수 → 주간 고가 전력 공급
- 대규모 배터리 역할

**양수 효율**
```
종합 효율 = 발전 효율 × 양수 효율 × 저장 효율
          ≈ 약 70~80%
```

---

## 예제

**문제**: 유량 50 m³/s, 유효낙차 100m, 종합효율 0.9인 수력발전소의 출력은?

```
P = 9.8 × Q × H × η
  = 9.8 × 50 × 100 × 0.9
  = 9.8 × 4,500
  = 44,100 kW
  = 44.1 MW
```

**문제**: 위 발전소에서 하루 6시간 발전 시 발전량은?

```
발전량 = 출력 × 시간
       = 44,100 kW × 6 h
       = 264,600 kWh
       ≈ 264.6 MWh
```

## 시험 포인트

| 공식/항목 | 내용 |
|---------|------|
| 수력 출력 | P = 9.8 × Q × H × η [kW] |
| 유효낙차 | 총낙차 - 손실낙차 |
| 펠톤 수차 | 고낙차(>300m), 충격식 |
| 프란시스 수차 | 중낙차(30~300m), 가장 많이 사용 |
| 카플란 수차 | 저낙차(<30m), 가변익, 반동식 |
| 비속도 Ns | N√P / H^(5/4), 수차 선정 기준 |
| 양수발전 목적 | 부하 평준화, 심야 잉여전력 저장 |
