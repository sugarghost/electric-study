# 08. 이상전압과 방호대책

## 핵심 개념
> 전력계통에 발생하는 비정상적으로 높은 전압(이상전압)의 종류와, 이를 막기 위한 방어 수단을 이해한다.

---

## 이상전압이란?

전력계통에서 정상적인 운전 전압보다 훨씬 높은 전압이 순간적으로 발생하는 현상입니다.
마치 수도관에 갑자기 엄청난 수압이 걸리는 것처럼, 전선에 과도한 전압이 걸려 기기가 손상될 수 있습니다.

---

## 이상전압의 종류

### 1) 뇌(雷) 이상전압 — 자연에서 오는 충격

| 종류 | 발생 원인 | 특징 |
|------|----------|------|
| 직격뢰 | 낙뢰가 전선에 직접 맞음 | 가장 심각, 수백만 V |
| 유도뢰 | 근처 낙뢰의 전자기 유도 | 직격뢰보다 약하지만 빈번 |

**뇌 서지(Lightning Surge) 파형 — 표준 충격파**

```
파두장: 1.2 μs  (전압이 최대까지 올라가는 시간)
파미장: 50 μs   (최대값의 50%로 내려오는 시간)
표기: 1.2/50 μs

전파 속도 ≈ 3 × 10⁸ m/s  (빛의 속도와 같음)
```

비유: 1.2μs 만에 꼭대기까지 치솟고, 50μs 만에 반쯤 내려오는 아주 날카로운 전압 스파이크입니다.

---

### 2) 개폐 이상전압 — 사람이 만드는 충격

차단기나 스위치를 열고 닫을 때 발생합니다.
마치 수도꼭지를 갑자기 잠글 때 "쾅" 하는 워터해머 현상과 같습니다.

- 크기: 정상 운전 전압의 2~4배 수준
- 화력 발전소나 변전소의 대형 차단기 동작 시 특히 주의 필요

---

### 3) 지속성 이상전압 — 계통 이상으로 생기는 전압 상승

1선 지락(전선 1가닥이 땅에 닿는 사고) 등이 발생하면 다른 건전상의 전압이 상승합니다.

- 비접지 계통: 최대 √3 배까지 상승 가능
- 직접 접지 계통: 상승폭이 작음

---

## 서지 임피던스 (Surge Impedance)

이상전압이 전선을 타고 이동할 때 전선이 갖는 특성 저항값입니다.

```
Z = √(L/C)   [Ω]

L: 단위 길이당 인덕턴스
C: 단위 길이당 커패시턴스
```

| 선로 종류 | 서지 임피던스 |
|---------|------------|
| 가공 송전선 | 약 400~500 Ω |
| 지중 케이블 | 약 30~60 Ω |

케이블의 서지 임피던스가 훨씬 작은 이유: 가공선보다 C(정전용량)가 매우 크기 때문입니다.

---

## 방호 대책

### 1) 가공지선 (Shield Wire) — 머리 위를 지키는 우산

철탑의 꼭대기에 설치하는 접지된 금속선입니다.

- **역할**: 낙뢰가 전선에 직접 맞지 않도록 차폐(가로막음)
- **차폐 각도**: 가공지선과 전선이 이루는 각도가 **30° 이내**일수록 차폐 효율 향상
- 맞더라도 가공지선 → 철탑 → 접지(땅)로 전류가 빠져나감

```
       ★ 가공지선
      /30°\
     /     \
    ○       ○  전선
```

---

### 2) 피뢰기 (Arrester) — 이상전압을 땅으로 빼주는 안전밸브

이상전압이 왔을 때 방전하여 전압을 낮추고, 정상 전압이 되면 자동으로 차단하는 장치입니다.

**동작 원리 (3단계)**
1. 이상전압 내습 → 피뢰기 동작(방전) → 전류를 땅으로 흘림
2. 이상전압 소멸 → 속류(정상 상용주파수 전류) 차단
3. 계통 정상 복귀

**설치 위치**
- 변전소 입구
- 변압기 1차측, 2차측
- 송전선 분기점

**주요 특성값**

| 특성 | 설명 |
|------|------|
| 공칭 방전 전류 | 피뢰기의 정격 방전 전류 (2.5, 5, 10 kA) |
| 제한 전압 | 피뢰기 동작 중 단자에 나타나는 전압 — **낮을수록 좋음** |
| 속류 차단 능력 | 방전 후 상용주파수 전류를 차단하는 능력 |

**방호 협조 (중요!)**
```
피뢰기 제한 전압  <  보호하는 기기의 절연 내력

조건이 만족되어야 기기가 안전하게 보호됨
```

---

### 3) 매설지선 (Counterpoise) — 철탑 발밑의 접지망

철탑 기초 주변에 지중에 매설하는 도체망입니다.

- **목적**: 철탑 접지저항 감소
- **효과**: 역섬락(낙뢰 후 철탑에서 전선으로 역방향 방전) 방지
- 토양 저항률이 높은 암반 지역에 특히 효과적

---

## 예제

**문제**: 가공 송전선의 서지 임피던스가 500Ω이고, 지중 케이블의 서지 임피던스가 50Ω이다. 가공선에서 케이블로 진행하는 서지의 투과율은?

```
투과율 τ = 2Z₂ / (Z₁ + Z₂)

Z₁ = 500Ω (가공선), Z₂ = 50Ω (케이블)

τ = 2 × 50 / (500 + 50)
  = 100 / 550
  ≈ 0.18 (18%)

→ 케이블로 진입 시 서지가 크게 감쇠됨 (82% 감소)
```

## 시험 포인트

| 공식/항목 | 내용 |
|---------|------|
| 표준 충격파 | 1.2/50 μs |
| 서지 임피던스 | Z = √(L/C), 가공선 400~500Ω, 케이블 30~60Ω |
| 가공지선 차폐 각도 | 30° 이내 → 효율적 차폐 |
| 피뢰기 방호협조 | 제한전압 < 기기 절연내력 |
| 직격뢰 vs 유도뢰 | 직격뢰가 더 심각 |
| 매설지선 목적 | 접지저항 감소, 역섬락 방지 |
| 지속성 이상전압 | 1선 지락 시 건전상 전압 √3 배까지 상승 (비접지) |
