# 04. 송전 특성

## 핵심 개념
> 거리에 따라 송전선을 단거리/중거리/장거리로 나누어 다르게 해석하며, 무부하 시 수전단 전압이 오히려 높아지는 페란티 효과를 주의해야 한다.

---

## 송전선 해석의 분류

| 구분 | 거리 기준 | 고려 요소 | 해석 방법 |
|------|-----------|-----------|-----------|
| 단거리 선로 | 60km 미만 (또는 300km 미만) | R, L (C 무시) | 집중 정수 회로 |
| 중거리 선로 | 60~300km 정도 | R, L, C | T형 / π형 등가회로 |
| 장거리 선로 | 300km 이상 | R, L, C, G | 분포 정수 회로 |

**비유**: 
- 단거리 = 자동차 도로 (단순하게 직선으로 가정)
- 중거리 = 기차 노선 (곡선과 구조물 고려)
- 장거리 = 비행 항로 (지구 곡률까지 고려)

---

## 단거리 선로

### 등가 회로

```
        R          jX
 ○─────/\/\/─────UUUUU─────○
Vs                           Vr
         (C 무시)
```

### 전압강하 (근사 공식)

$$V_s \approx V_r + I(R\cos\phi + X\sin\phi) \quad [V]$$

- $V_s$: 송전단 전압, $V_r$: 수전단 전압
- $I$: 선전류, $R$: 선로 저항, $X$: 선로 리액턴스
- $\phi$: 역률각 ($\cos\phi$ = 역률)

**공식 이해**: 
- $R\cos\phi$: 저항으로 인한 전압 손실 (유효분)
- $X\sin\phi$: 리액턴스로 인한 전압 손실 (무효분)

### 전압강하율

$$\varepsilon = \frac{V_s - V_r}{V_r} \times 100 \quad [\%]$$

또는 근사:

$$\varepsilon \approx \frac{IR\cos\phi + IX\sin\phi}{V_r} \times 100 \quad [\%]$$

### 전력 손실 (3상)

$$P_l = 3I^2R \quad [W]$$

전력손실률 $= \dfrac{P_l}{P} \times 100 \; [\%]$ ($P$: 수전전력 [W])

**3상인 이유**: 3개의 전선이 각각 I²R의 손실을 가지므로 3배.

---

## 중거리 선로

정전용량(C)을 무시할 수 없게 되어 등가회로가 복잡해집니다.

### T형 등가회로

```
   Z/2         Z/2
○──/\/\/──┬──/\/\/──○
Vs        │          Vr
          Y
          │ (Y: 전체 어드미턴스 중간에)
          ○
```

- 임피던스(Z)를 양쪽에 **절반씩** 배치
- 어드미턴스(Y = jωC·l)를 **가운데** 배치

### π형 등가회로

```
     Z (전체)
○────/\/\/────○
│              │
Y/2            Y/2
│              │
○─────────────○
Vs              Vr
```

- 임피던스(Z) **가운데** 배치
- 어드미턴스(Y/2)를 **양쪽에 절반씩** 배치

**기억법**: 
- T형 = 임피던스가 T자 모양으로 나뉨
- π형 = 어드미턴스가 π(파이) 모양으로 나뉨

## 장거리 선로 (분포 정수 회로)

### 특성 임피던스 (파동 임피던스)

$$Z_c = \sqrt{\frac{Z}{Y}} \quad [\Omega]$$

- $Z = R + j\omega L$: 단위 길이당 직렬 임피던스
- $Y = G + j\omega C$: 단위 길이당 병렬 어드미턴스

무손실 선로 ($R=0$, $G=0$) 가정 시: $Z_c = \sqrt{L/C}$ [Ω]

### 전파정수

$$\gamma = \sqrt{ZY} = \alpha + j\beta$$

- $\alpha$: 감쇠정수 (신호가 약해지는 비율)
- $\beta$: 위상정수 (신호의 위상 변화)

### 전파 속도

$$v = \frac{\omega}{\beta} = \frac{1}{\sqrt{LC}} \approx 3\times10^8 \; [m/s]$$

무손실 선로에서는 빛의 속도와 같음

## 페란티 효과 (Ferranti Effect)

### 페란티 효과란?

```
일반적으로 예상: 수전단 전압 < 송전단 전압
페란티 효과:    수전단 전압 > 송전단 전압 ← 이상하죠?
```

**무부하 또는 경부하 시 수전단 전압이 송전단보다 높아지는 현상**

### 발생 원인

```
무부하 시 → 선로 충전전류만 흐름
충전전류 = 선로 정전용량(C)으로 인한 진상전류

이 전류가 선로 인덕턴스(L)와 반응 → 전압 상승
```

**비유**: 양수기로 물을 빨아들이는 것과 같습니다. 
- 선로의 정전용량이 "진공 흡입"처럼 작동
- 수전단에서 오히려 전압을 끌어올림

### 페란티 효과의 문제점

```
수전단 전압 과도 상승 → 기기 절연 파괴 위험
```

### 대책: 분로 리액터 (Shunt Reactor) 설치

```
수전단에 인덕터(코일)를 병렬로 연결
→ 진상전류(C에 의한) 를 상쇄하는 지상전류 공급
→ 전압 상승 억제
```

---

## 송전 효율

$$\eta = \frac{P_r}{P_r + P_l} \times 100 = \frac{P_r}{P_s} \times 100 \quad [\%]$$

- $P_r$: 수전전력 [W], $P_l$: 선로 손실 [W], $P_s = P_r + P_l$: 송전전력

---

## 전압 변동률

$$\varepsilon = \frac{V_0 - V_F}{V_F} \times 100 \quad [\%]$$

- $V_0$: 무부하 수전단 전압, $V_F$: 전부하 수전단 전압

---

## 예제 1: 단거리 선로 전압강하

**문제**: 3상 3선식 선로, 역률 0.8(지상), 선전류 100A, 저항 R=10Ω, 리액턴스 X=15Ω, 수전단 전압 Vr=6,600V일 때 송전단 전압을 구하시오.

$\cos\phi = 0.8$이면 $\sin\phi = 0.6$

$$V_s \approx 6{,}600 + 100 \times (10 \times 0.8 + 15 \times 0.6) = 6{,}600 + 1{,}700 = 8{,}300 \; [V]$$

## 예제 2: 전력 손실

**문제**: 3상 선로, 전류 50A, 선로 저항 R=5Ω일 때 전력 손실을 구하시오.

$$P_l = 3I^2R = 3 \times 50^2 \times 5 = 37{,}500 \; [W] = 37.5 \; [kW]$$

## 시험 포인트

| 공식 | 내용 |
|------|------|
| 전압강하 (근사) | `Vs ≈ Vr + I(R cosφ + X sinφ)` |
| 전압강하율 | `ε = (Vs - Vr)/Vr × 100 [%]` |
| 전력손실 (3상) | `Pl = 3I²R [W]` |
| 특성 임피던스 | `Zc = √(Z/Y)` |
| 전파정수 | `γ = √(ZY) = α + jβ` |
| 송전효율 | `η = Pr/(Pr + Pl) × 100 [%]` |
| 페란티 효과 | 무부하 시 수전단 > 송전단 전압 |
| 페란티 대책 | 분로 리액터 설치 |
