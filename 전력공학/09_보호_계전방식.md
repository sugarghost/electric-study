# 09. 보호 계전방식

## 핵심 개념
> 전력계통에서 고장이 발생했을 때 이를 빠르게 검출하고 차단기에 신호를 보내 고장 구간을 분리하는 시스템을 이해한다.

---

## 보호 계전기란?

전력계통의 고장을 감시하는 "경보 + 트리거" 장치입니다.
마치 건물의 화재 감지기처럼, 이상을 감지하면 소화 시스템(차단기)에 신호를 보냅니다.

**동작 흐름**
```
고장 발생 → CT/PT 측정 → 계전기 검출 → 차단기 트립 신호 → 고장 구간 분리
```

- **CT(변류기)**: 전류를 측정하여 계전기에 전달 (수천 A → 5A로 축소)
- **PT(변압기)**: 전압을 측정하여 계전기에 전달 (수만 V → 110V로 축소)

---

## 주요 계전기 종류

| 계전기 | 동작 조건 | 주요 용도 |
|--------|---------|----------|
| 과전류(OCR) | 설정값 이상 전류 흐를 때 | 단락·과부하 보호 |
| 거리(Distance) | 측정 임피던스가 설정 구간 이내일 때 | 송전선 보호 |
| 차동(Differential) | 입출력 전류 차이가 설정값 초과 | 변압기·발전기 보호 |
| 지락(GR) | 영상전류 검출 시 | 지락 고장 보호 |
| 방향(Directional) | 전력(또는 전류) 흐름 방향이 설정 방향일 때 | 루프·병행 계통 보호 |

---

## 과전류 계전기 (OCR) 동작 특성

고장 전류의 크기에 따라 동작 시간이 달라집니다.

### 반한시 (Inverse Time)
```
전류가 클수록 더 빨리 동작
동작 시간 t ∝ 1/I²

→ 심각한 고장일수록 빠르게 차단 (가장 일반적)
```

### 정한시 (Definite Time)
```
전류 크기와 무관하게 설정 시간 후 동작
→ 보호 협조(상·하위 계전기 간 시간 조정)에 활용
```

### 순한시 (Instantaneous)
```
설정값 이상이면 즉시(거의 0초) 동작
→ 큰 단락전류에 대한 즉시 보호
```

**비유**: 반한시는 "불이 크면 빨리 끄고, 작은 불은 좀 기다린다"는 개념입니다.

---

## 거리 계전기 (Distance Relay)

고장 지점까지의 **임피던스(거리)**를 측정하여 동작합니다.

```
측정 임피던스 Z = V측정 / I측정

Z가 설정 구간(원 내부) 이내이면 → 구간 내 고장으로 판단 → 동작
```

### 보호 구간 (Zone) 구성

```
  변전소 A ───────────────────── 변전소 B
      │←── Zone 1(80%)──→│
      │←────── Zone 2(120%) ──────→│
      │←──────────── Zone 3(200%) ──────────→│

Zone 1: 즉시 동작 (자선 보호)
Zone 2: 0.3~0.5초 지연 동작 (후비 보호)
Zone 3: 0.6~1.0초 지연 동작 (원거리 후비 보호)
```

**장점**: 전류 크기만 보는 OCR과 달리, 위치(거리)를 알 수 있어 정확한 보호 가능

---

## 차동 계전기 (Differential Relay) — 변압기·발전기 보호

입력 전류와 출력 전류를 비교하여 차이가 생기면 내부 고장으로 판단합니다.

```
정상 상태: I₁차 = I₂차  → 차동전류 = 0 → 동작 안 함
내부 고장: I₁차 ≠ I₂차  → 차동전류 발생 → 동작!
```

### 비율 차동 계전기 (Percentage Differential Relay)

```
동작 조건:
(I₁ - I₂) / ((I₁ + I₂) / 2) > 설정 비율 (보통 15~30%)

→ 분모(억제 전류)가 클수록 오동작 방지
```

**왜 비율로 계산?**
변압기 여자 돌입전류(처음 투입 시 과도전류)에 의한 오동작을 막기 위함입니다.

---

## 지락 계전기 (GR)

3상 전류의 합인 **영상전류(I₀)**를 검출합니다.

```
정상 상태: Ia + Ib + Ic = 0  →  영상전류 = 0
1선 지락: 합이 0이 아님  →  영상전류 발생 → 동작
```

- **ZCT(영상변류기)**: 3선을 함께 통과시켜 영상전류를 측정

---

## 방향 계전기 (Directional Relay)

전류의 방향을 판별하는 계전기입니다.

- **단순 OCR**: 전류 크기만 보므로, 루프 계통에서 어느 방향 고장인지 모름
- **방향 계전기**: 전압과 전류의 위상 관계로 방향 판별 → 루프·환상 계통 보호에 필수

---

## 보호 협조 (Protection Coordination)

여러 계전기가 서로 역할을 나누어 보호하는 체계입니다.

```
주 보호 (Primary Protection): 고장 구간을 직접 보호하는 계전기
후비 보호 (Backup Protection): 주 보호 실패 시 상위(더 먼) 계전기가 대신 동작
```

**시간 협조 예시**
```
      변전소 A ──────────── B ──────────── C
                  t=0.1s (주보호)  t=0.4s (후비)  t=0.7s (후후비)

고장이 B-C 사이에서 발생 시:
  → B 계전기 0.1s 동작 (주 보호) → 성공하면 종료
  → 실패 시 A 계전기 0.4s 동작 (후비 보호)
```

---

## 예제

**문제**: 거리 계전기 Zone 1에서 선로 임피던스가 총 10Ω일 때, Zone 1 정정값(80%)은?

```
Zone 1 정정 임피던스 = 10 × 0.8 = 8Ω

측정 임피던스 < 8Ω → Zone 1 구간 내 고장 → 즉시 차단
측정 임피던스 8~12Ω → Zone 2 구간 → 0.3~0.5초 후 차단
```

## 시험 포인트

| 공식/항목 | 내용 |
|---------|------|
| OCR 반한시 특성 | 전류 클수록 동작 시간 짧음 (t ∝ 1/I²) |
| 거리 계전기 | 임피던스 측정, Zone 1(80%)·Zone 2(120%)·Zone 3(200%) |
| 차동 계전기 | 입출력 전류 차이 검출, 변압기·발전기 내부 고장 보호 |
| 비율 차동 | (I₁-I₂)/((I₁+I₂)/2) > 설정값 |
| 지락 계전기 | 영상전류(I₀) 검출, ZCT 사용 |
| 방향 계전기 | 전류 방향 판별, 루프 계통에 필수 |
| 보호 협조 | 주 보호 실패 시 후비 보호 동작 (시간 차이로 협조) |
