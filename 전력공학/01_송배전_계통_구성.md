# 01. 송배전 계통 구성

## 핵심 개념
> 전기는 발전소에서 만들어져 변전소에서 전압을 높이고 낮추며 수용가(우리 집)까지 전달된다.

---

## 전력 시스템의 흐름

전기가 만들어져서 우리 집 콘센트까지 오는 과정을 생각해봅시다.

```
발전소 → 승압변전소 → 초고압 송전선 → 1차 변전소 → 2차 변전소 → 배전선 → 수용가
(발전)   (전압 높임)   (장거리 전송)   (전압 낮춤)  (전압 낮춤)  (동네)   (우리 집)
```

**비유**: 수도관과 비슷합니다.
- 발전소 = 정수장 (물을 만드는 곳)
- 송전선 = 큰 수도관 (멀리 보내는 관)
- 배전선 = 골목 수도관 (집 근처 작은 관)
- 수용가 = 각 가정의 수도꼭지

---

## 한국의 전압 단계

| 구분 | 전압 | 설명 |
|------|------|------|
| 발전 전압 | 11kV ~ 25kV | 발전소에서 만들어지는 전압 |
| 초고압 송전 | 765kV, 345kV, 154kV | 멀리 보내기 위해 높인 전압 |
| 1차 배전 | 22.9kV | 동네까지 오는 전압 (3상 4선식) |
| 저압 수용가 | 220V / 380V | 일반 가정용 / 산업용 |

**참고**: 765kV는 현재 신고리-신경남 구간 등 일부에서 사용되는 세계 최고 수준의 송전 전압입니다.

---

## 왜 전압을 높여서 보낼까? (승압의 이유)

핵심 공식:
```
전력 P = V × I   [W]
전력이 같다면 → 전압(V)↑ 이면 전류(I)↓
```

전선에서 열로 사라지는 손실:
```
전력손실 Pl = I² × R   [W]
전류(I)가 절반이 되면 → 손실은 1/4 로 줄어듦!
```

**비유**: 물을 멀리 보낼 때
- 가는 파이프로 세게 흘리면 → 마찰이 크고 압력 손실이 큼
- 넓은 파이프로 천천히 흘리면 → 마찰이 적고 효율적

전기도 마찬가지로, 높은 전압(=낮은 전류)으로 보내면 손실이 훨씬 줄어듭니다.

**예시**: 100MW를 10km 떨어진 곳에 전송할 때
```
154kV로 송전: 전류 I = 100×10⁶ / (√3 × 154×10³) ≈ 375 A
22.9kV로 송전: 전류 I = 100×10⁶ / (√3 × 22.9×10³) ≈ 2,520 A

손실비율: (375/2520)² ≈ 1/45 → 154kV가 약 45배 효율적!
```

---

## 계통 연계의 장점

여러 발전소와 변전소를 하나의 네트워크(계통)로 연결하면:

1. **경제적 운용**: 발전 비용이 낮은 곳에서 우선 발전하여 전체 비용 절감
2. **신뢰성 향상**: 한 발전소가 고장나도 다른 곳에서 공급 가능
3. **예비력 공유**: 각 지역이 따로 예비 발전기를 가질 필요 없음
4. **부하 균등화**: 지역마다 다른 최대 수요를 서로 보완

## 수전 방식 (배전 방식)

### 1. 방사상식 (나뭇가지 모양)
```
변전소 ─── 수용가1
       └── 수용가2
       └── 수용가3
```
- **장점**: 구조 간단, 건설비 저렴
- **단점**: 선로 고장 시 해당 구간 전체 정전
- **사용처**: 농촌, 소규모 배전

### 2. 루프식 (고리 모양)
```
변전소 ─── 수용가1 ─── 수용가2 ─── 변전소
               (고리를 이루는 형태)
```
- **장점**: 한 쪽 선로 고장 시 반대 방향으로 공급 가능 → 정전 범위 축소
- **단점**: 방사상식보다 공사비 증가
- **사용처**: 도시 배전

### 3. 네트워크식 (그물망 모양)
```
변전소1 ─── ─── 변전소2
   │    ×   ×    │
수용가 ─── ─── 수용가
(여러 경로로 연결된 복잡한 망)
```
- **장점**: 신뢰성 최고, 전압 변동 최소
- **단점**: 설비비 매우 비쌈, 보호 계전 복잡
- **사용처**: 대도시 중심부, 중요 시설

| 방식 | 신뢰성 | 비용 | 사용처 |
|------|--------|------|--------|
| 방사상식 | 낮음 | 저렴 | 농촌 |
| 루프식 | 중간 | 중간 | 도시 |
| 네트워크식 | 높음 | 비쌈 | 대도시 |

## 예제

**문제**: 발전소에서 100kW를 1km 떨어진 곳에 공급한다. 송전 전압이 22.9kV일 때 선로 전류를 구하시오. (단, 3상 기준, 역률 1.0)

```
P = √3 × V × I × cosφ
100,000 = √3 × 22,900 × I × 1.0
I = 100,000 / (√3 × 22,900)
I ≈ 100,000 / 39,669
I ≈ 2.52 A
```

**풀이**: 22.9kV의 높은 전압으로 송전하기 때문에 전류가 2.52A로 매우 작습니다.

## 시험 포인트

| 공식/개념 | 내용 |
|-----------|------|
| 전력 공식 | `P = VI` (단상), `P = √3 VI cosφ` (3상) |
| 승압 이유 | 전압↑ → 전류↓ → 손실 `I²R`↓ |
| 전력손실 | `Pl = I²R` → 전류의 제곱에 비례 |
| 수전 방식 순서 | 신뢰성: 네트워크 > 루프 > 방사상 |
| 한국 배전전압 | 22.9kV (3상 4선식) |
| 일반 가정 전압 | 220V (단상 2선식) |

> **자주 출제**: 승압 이유, 수전 방식 특징 비교, 전력손실 계산
