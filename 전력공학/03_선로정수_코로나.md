# 03. 선로정수와 코로나

## 핵심 개념
> 실제 전선은 저항, 인덕턴스, 정전용량, 누설컨덕턴스의 4가지 성질을 동시에 가지며, 전압이 너무 높으면 코로나 방전이 발생한다.

---

## 선로정수 (Line Constants)

전선은 단순한 도선이 아닙니다. 실제 송전선에는 4가지 분포 정수가 있습니다.

```
단위 길이당:
R: 저항 [Ω/km]    → 전력손실
L: 인덕턴스 [H/km] → 자계 에너지 저장
C: 정전용량 [F/km] → 전계 에너지 저장
G: 누설컨덕턴스 [S/km] → 애자 등의 누설전류 (보통 무시)
```

**비유**: 도로를 생각해보면,
- R = 도로의 마찰 (에너지 손실)
- L = 도로의 관성 (변화에 저항)
- C = 도로 옆의 저수지 (에너지를 잠깐 저장)
- G = 도로 옆의 작은 유출구 (소량 누설)

---

## 저항 R

### 저항 기본 공식

```
R = ρl/A   [Ω]

ρ: 고유저항 [Ω·m]
l: 전선 길이 [m]
A: 전선 단면적 [m²]
```

### 온도와 저항

```
R_t = R_0 × (1 + α × t)

R_0: 기준온도(0°C)에서의 저항
α: 저항 온도계수
t: 온도 [°C]
```

**온도 상승 → 저항 증가** (도체에서는 항상)

### 표피 효과 (Skin Effect)

```
직류: 전선 전 단면적에 전류가 균등하게 흐름
교류: 주파수가 높을수록 전류가 전선 표면에 집중

결과: 실질적으로 전류가 흐르는 면적 감소
    → 유효 저항 증가
```

**비유**: 표피 효과는 마치 군중이 건물 테두리에만 서있는 것과 같습니다. 60Hz 상용 주파수에서는 이미 일부 표피 효과가 발생합니다.

## 인덕턴스 L

### 왕복 2선식 선로

```
L = (μ₀/π) × ln(D/r)   [H/m]

μ₀ = 4π × 10⁻⁷ [H/m] (진공 투자율)
D: 두 전선 간의 거리 [m]
r: 전선의 반지름 [m]
```

### D와 r의 영향

```
D 증가 (선간 거리 넓힘) → ln(D/r) 증가 → L 증가
r 증가 (전선 굵게)      → ln(D/r) 감소 → L 감소
```

**직관**: 두 전선이 멀어질수록 루프 면적이 커져서 인덕턴스 증가.

---

## 정전용량 C

### 왕복 2선식 선로

```
C = πε₀/ln(D/r)   [F/m]

ε₀ = 8.854 × 10⁻¹² [F/m] (진공 유전율)
D: 두 전선 간의 거리 [m]
r: 전선의 반지름 [m]
```

### D와 r의 영향

```
D 증가 (선간 거리 넓힘) → ln(D/r) 증가 → C 감소
r 증가 (전선 굵게)      → ln(D/r) 감소 → C 증가
```

**주목**: L과 C는 D, r에 대해 **반대 방향**으로 변합니다!

| 변화 | L 변화 | C 변화 |
|------|--------|--------|
| D 증가 | 증가 | 감소 |
| r 증가 | 감소 | 증가 |

## 코로나 방전 (Corona Discharge)

### 코로나란?

전선 주위의 전기장이 매우 강해지면, 공기 분자가 이온화되어 발광하고 방전이 일어납니다. 이것이 코로나 방전입니다.

```
전선 주위 전기장 E [V/m] > 공기 절연 파괴 전계 Ec
→ 공기 이온화 → 코로나 방전 발생
```

**비유**: 물이 넘치는 것과 비슷합니다. 컵(공기)에 물(전압)을 계속 부으면 결국 넘쳐흐릅니다(코로나 방전).

밤에 고압 송전선 주위에서 보이는 **파란빛 광휘**가 바로 코로나입니다.

### 임계전압 (Peek의 공식)

코로나가 막 발생하기 시작하는 전압:

```
Ec = 30δm₀r × ln(D/r)   [kV (최댓값)]

δ: 상대 공기밀도 (표준 상태에서 δ = 1)
m₀: 전선 표면 상태 계수 (완전한 원형=1, 연선=0.85~0.9)
r: 전선 반지름 [cm]
D: 선간 거리 [cm]
```

**참고**: 실효값으로 표현 시 `Ec(rms) = Ec(max)/√2`

### 코로나 발생에 영향을 주는 요소

```
온도 낮음, 기압 높음 → 공기 밀도 높음(δ↑) → 코로나 발생 어려움
온도 높음, 고도 높음(기압 낮음) → δ↓ → 코로나 발생 쉬움
빗속, 안개 → 코로나 발생 쉬움 (m₀↓)
```

### 코로나 발생 시 문제점

1. **전력 손실**: 코로나 전력 손실 발생
2. **라디오 장해**: 전자파 방사로 통신 방해 (Radio Interference, RI)
3. **소음 발생**: 윙윙 소리 (특히 악천후 시)
4. **오존 발생**: 전선 부식 촉진
5. **전선 손상**: 장기간 운전 시 전선 표면 손상

### 코로나 방지 대책

| 대책 | 원리 | 방법 |
|------|------|------|
| 전선 굵게 | r↑ → 전기장 집중 감소 | 전선 굵은 것 사용 |
| **복도체 사용** | 등가 반지름 증가 | 여러 가닥 묶음 |
| 선간 거리 증가 | 전기장 분산 | 철탑 폭 넓힘 |
| 가선 금구 개선 | 돌출부 제거 | 코로나 링 설치 |

### 복도체 (Bundle Conductor)

```
단도체:   ○  (단일 전선 1가닥)

2복도체:  ○○  (2가닥을 묶음)
           D₁

4복도체:  ○○  (4가닥을 정사각형 배치)
          ○○
```

**복도체 효과**:
- 등가 반지름 증가 → 코로나 방지
- 인덕턴스 감소, 정전용량 증가 → 송전 용량 증가
- 345kV 이상 선로에서 필수 사용

---

## 예제

**문제**: 선간 거리 D = 200cm, 전선 반지름 r = 1cm인 3상 송전선에서 코로나 임계전압을 구하시오. (표준 대기압, 완전 원형 전선 가정: δ = 1, m₀ = 1)

```
Ec = 30 × δ × m₀ × r × ln(D/r)
Ec = 30 × 1 × 1 × 1 × ln(200/1)
Ec = 30 × ln(200)
Ec = 30 × 5.298
Ec ≈ 158.9 kV (최댓값)

실효값: 158.9/√2 ≈ 112.4 kV (선간전압 기준)
```

| 공식 | 내용 |
|------|------|
| 저항 | `R = ρl/A` |
| 인덕턴스 | `L = (μ₀/π) × ln(D/r)` |
| 정전용량 | `C = πε₀/ln(D/r)` |
| 코로나 임계전압 | `Ec = 30δm₀r × ln(D/r)` [kV] |
| 표피 효과 | 주파수↑ → 전류가 표면 집중 → 유효 저항↑ |
| D 증가 효과 | L↑, C↓ (서로 반대!) |
| r 증가 효과 | L↓, C↑ (서로 반대!) |
| 코로나 방지 | 복도체 사용, 전선 굵게, 선간 거리 증가 |
