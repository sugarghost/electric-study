# 02. 가공 송전 선로

## 핵심 개념
> 하늘에 매달린 전선은 무게 때문에 처지는데, 이 처짐 양(이도)을 정확히 계산하는 것이 중요하다.

---

## 전선의 종류

### 주요 전선 비교

| 종류 | 약호 | 특징 | 사용처 |
|------|------|------|--------|
| 경동선 | HD | 구리 전선, 도전성 최고, 무거움 | 저압 배전 |
| 경알루미늄선 | AL | 알루미늄, 가볍지만 강도 약함 | 일부 배전 |
| 강심알루미늄연선 | ACSR | 알루미늄 + 강심선, 가장 많이 사용 | 송전선 |
| 강심알루미늄합금연선 | ACSR/AC | ACSR 개량형 | 장경간 |

### ACSR (Aluminum Conductor Steel Reinforced)이 가장 많이 쓰이는 이유

```
구조: [강심선(Steel)] ← 알루미늄선(Aluminum)으로 감쌈

강심선: 기계적 강도 담당 (끊어지지 않게)
알루미늄: 전류 도통 담당 (전기를 흘림)
```

**비유**: ACSR은 마치 복합 소재 낚싯대와 같습니다.
- 속의 유리섬유(강심) = 강도 담당
- 겉의 탄소섬유(알루미늄) = 기능 담당

**ACSR의 장점**:
- 알루미늄은 구리보다 가벼워서 전주(전봇대) 부담 감소
- 강심선으로 기계적 강도를 보완
- 경제적 (알루미늄이 구리보다 저렴)

## 이도 (Sag, 弛度)

### 이도란?

전선을 두 전주 사이에 매달면 자체 무게 때문에 아래로 처집니다. 이 처지는 양을 **이도(D)**라고 합니다.

```
전주                     전주
 |          D(이도)       |
 |         ↕              |
 |  ~~~전선이 처짐~~~     |
 |________________________|
          S(경간)
```

### 이도 계산 공식

$$D = \frac{WS^2}{8T} \quad [m]$$

- $D$: 이도 (처지는 양) [m]
- $W$: 전선 단위 길이당 합성하중 [N/m]
- $S$: 경간 (지지물 간 거리) [m]
- $T$: 전선의 수평장력 [N]

**공식 이해하기**:
- S(경간)가 클수록 D 크게 증가 (S²에 비례!)
- W(무게)가 무거울수록 D 증가
- T(장력)가 클수록 D 감소 (팽팽하게 당기면 덜 처짐)

**비유**: 빨랫줄을 생각해보세요.
- 간격(S)이 넓을수록 더 많이 처짐
- 무거운 빨래(W)를 걸수록 더 많이 처짐
- 줄을 팽팽하게(T) 당길수록 덜 처짐

### 전선 실제 길이 공식

경간보다 전선이 더 길어야 처질 수 있습니다:

$$L \approx S + \frac{8D^2}{3S} \quad [m]$$

- $S$: 경간 [m], $D$: 이도 [m]

**의미**: 실제 전선 길이 = 경간 + 이도로 인해 추가되는 길이

---

## 전선에 작용하는 하중

전선에는 세 가지 힘이 동시에 작용합니다:

### 1. 자중 (Wc)
- 전선 자체의 무게
- 항상 수직 하향

### 2. 빙설 하중 (Wi)
- 전선에 얼음이나 눈이 붙을 때
- 수직 하향 (무게 증가)
- 겨울철 문제

### 3. 풍압 하중 (Ww)
- 바람이 전선을 옆으로 미는 힘
- 수평 방향

### 합성 하중 계산

$$W = \sqrt{(W_c + W_i)^2 + W_w^2} \quad [N/m]$$

합성하중은 대각선 방향으로 작용

**비유**: 자중+빙설은 아래로 당기고, 풍압은 옆으로 미는 힘입니다. 실제 전선은 이 두 힘의 합력(대각선)을 받습니다.

---

## 온도 변화와 이도

```
온도 상승 → 전선 팽창(늘어남) → 이도 증가 → 지표면에 가까워짐
온도 하강 → 전선 수축(줄어듦) → 이도 감소 → 장력 증가 → 끊어질 위험
```

**중요**: 이도 설계 시 고온과 저온 양쪽을 모두 고려해야 합니다.

## 애자 (Insulator)

### 애자의 역할

전선과 철탑/전주 사이에서 **전기 절연** 역할을 합니다.

```
철탑(접지) ─── 애자 ─── 전선(고전압)
          (절연체)
```

### 현수애자 (Suspension Insulator)

고압 송전선에 주로 사용되며, 여러 개를 직렬로 연결합니다.

```
철탑
 │
[애자1]
 │
[애자2]
 │
[애자3]
 │
전선
```

**직렬 수와 전압의 관계**:
```
애자 개수를 늘릴수록 → 내전압 증가
154kV: 약 10개
345kV: 약 23개
765kV: 약 45개 (대략적인 기준)
```

### 애자 련 (String)

직렬로 연결된 애자 묶음을 **애자련**이라고 합니다.

**문제점**: 전압이 애자에 균등하게 걸리지 않음 (전선에 가까운 애자에 더 많은 전압)
**대책**: 균압링(Grading Ring) 설치

## 지선 (Stay Wire)

### 지선의 역할

전주(전봇대)가 전선의 무게나 장력에 의해 기울어지지 않도록 지지합니다.

```
     전주
    /  │
지선/   │
  /    │
 ●─────● 
 지면   지면
```

**장력이 불균형한 경우에 필요**:
- 전선이 꺾이는 곳 (각도주)
- 선로 끝부분 (인류주)
- 경사지나 강풍 지역

---

## 예제

**문제**: 경간 200m, 전선 단위 하중 20N/m, 수평장력 10,000N일 때 이도를 구하시오.

$$D = \frac{WS^2}{8T} = \frac{20 \times 200^2}{8 \times 10{,}000} = \frac{800{,}000}{80{,}000} = 10 \; [m]$$

**풀이**: 이도는 10m입니다.

**이 때 전선의 실제 길이**:

$$L \approx S + \frac{8D^2}{3S} = 200 + \frac{8 \times 10^2}{3 \times 200} = 200 + \frac{800}{600} \approx 201.33 \; [m]$$

## 시험 포인트

| 공식 | 내용 |
|------|------|
| 이도 | `D = WS²/(8T)` [m] |
| 전선 길이 | `L ≈ S + 8D²/(3S)` [m] |
| 합성 하중 | `W = √((Wc + Wi)² + Ww²)` |
| ACSR | 강심알루미늄연선, 가공 송전선에 가장 많이 사용 |
| 이도와 온도 | 온도↑ → 이도↑, 온도↓ → 이도↓, 장력↑ |
| 현수애자 | 직렬 연결, 개수 많을수록 내전압 증가 |

> **자주 출제**: 이도 계산, ACSR 특징, 합성 하중 계산, 전선 길이 공식
