## 챕터별 핵심 키워드

### 01. 송배전 계통 구성
- 발전소 → 변전소 → 송전선 → 배전선 → 수용가
- 한국 전압 체계: 765kV / 345kV / 154kV / 22.9kV / 220V
- 승압의 이유: `P = VI` → 전압 높이면 전류 줄고 → `I²R` 손실 감소
- 수전 방식: 방사상식 / 루프식 / 네트워크식

### 02. 가공 송전 선로
- 전선 종류: ACSR (강심알루미늄연선) 가장 많이 사용
- 이도 공식: `D = WS²/(8T)`
- 전선 길이: `L ≈ S + 8D²/(3S)`
- 온도 상승 → 이도 증가

### 03. 선로정수와 코로나
- 인덕턴스: `L = (μ₀/π) × ln(D/r)`
- 정전용량: `C = πε₀/ln(D/r)`
- 코로나: 전선 주위 공기가 절연 파괴되는 방전 현상
- 방지: 전선 굵게, 복도체 사용

### 04. 송전 특성
- 단거리: 전압강하 `Vs = Vr + I(R cosφ + X sinφ)`
- 중거리: T형 / π형 등가회로
- 장거리: 분포정수 회로, 특성 임피던스 `Zc = √(Z/Y)`
- 페란티 효과: 무부하 시 수전단 전압 > 송전단 전압

### 05. 중성점 접지방식
- 직접 접지: 154kV 이상, 지락전류 큼
- 저항 접지: 22.9kV 배전, 지락전류 제한
- 소호리액터 접지: 지락전류 최소
- 유도장해: 정전유도 / 전자유도

### 06. 고장 계산 (시험 출제 多)
- `%Z = Z × Pn/Vn² × 100`
- 단락전류: `Is = In × (100/%Z)`
- 단락용량: `Ps = Pn × (100/%Z)`
- 대칭좌표법: 정상분 / 역상분 / 영상분

### 07. 전력계통 안정도
- 전달 전력: `P = (EV/X) × sin δ`
- 최대 전달: δ = 90°일 때
- 등면적법: 가속 면적 = 감속 면적 → 과도 안정
- 안정도 향상: 리액턴스 감소, 고속 차단, 자동 재폐로
