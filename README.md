## 🗺️ 권장 학습 순서

```
회로이론 (기초) → 전자기학 → 전력공학 → 전기기기 → 설비기준(암기)
```

1. **회로이론부터** 시작 — 전 과목의 수학적 기반
2. **전자기학** — 회로이론 개념이 있으면 수월
3. **전력공학** — 실제 전력 시스템 응용
4. **전기기기** — 발전기·변압기·전동기 원리
5. **전기설비기술기준** — 시험 2~3주 전에 집중 암기

---

## 💡 공부 방법

- **1회독**: 이해 안 돼도 읽고 넘어가기
- **2회독**: 공식 따로 정리하며 읽기
- **기출문제**: 어떤 패턴으로 나오는지 파악
- **반복**: 모르는 문제도 답 보고 넘어가기

> 막힌다고 멈추지 말고, 일단 진도부터
